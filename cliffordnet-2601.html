§1 核心贡献
核心贡献

CliffordNet 的核心主张很直接：现代视觉架构（CNN、Transformer、ConvNeXt）依赖「空间混合器 + 通道混合器（FFN）」的启发式堆叠，缺乏数学第一性原理。作者用 Clifford Geometric Product 作为唯一的交互算子，同时完成 token mixing 和 channel mixing，实现 No-FFN 设计。

结果：在 CIFAR-100 上，CliffordNet-Nano（1.4M 参数）匹配 ResNet-18（11.2M），压缩比 8×；CliffordNet-Lite（2.6M）达到 79.05%，创下 tiny 模型新 SOTA。CliffordNet-32（4.8M）甚至超过 ResNet-50（23.7M）和 DenseNet-121（7.0M），且全程不使用 FFN。

「The two systems [Hamilton's and Grassmann's] are not only consistent with one another, but they are actually parts of a larger whole.」

—— William Kingdon Clifford, 1878
§2 背景
背景：Geometry for Attention → Geometry as Computation

CliffordNet 处于三条研究脉络的交汇处：

  • 几何代数神经网络：Clifford Neural Layers (ICLR 2023)、Clifford Group Equivariant NNs (NeurIPS 2023 oral)、GCAN (ICML 2023) 证明 geometric product 可用于构建具有明确几何性质的神经层
  • 线性复杂度视觉模型：Mamba/SSM 通过状态空间模型替代 quadratic attention；VMamba 用 cross-scan 序列化 2D 图像，核心问题是其扫描破坏了 2D 流形连续性
  • 数学第一性原理叙事：KAN 以 Kolmogorov-Arnold 定理为据；RiemannFormer 用 Riemannian geometry 改造 attention；CliffordNet 则将「几何本身」作为计算

作者 Zhongping Ji（杭州电子科技大学）的研究路线很清晰：2025 年 RiemannFormer 用几何改造 attention → 2026 年 CliffordNet 用几何积直接替换 attention + FFN。两篇均为单作者 arXiv 预印本。

核心转向：从「Geometry for Attention」（几何用来修正 attention）到「Geometry as Computation」（几何本身就是 token 交互机制）。现代深度学习重演了 Clifford 所指出的历史分裂：Transformer 重度依赖 dot product（Hamilton 的内积路线），而忽略了 wedge product（Grassmann 的外积路线）。CliffordNet 试图恢复这种代数统一。
§3 核心机制
核心机制：Clifford Geometric Product

CliffordNet 的理论基础是 Clifford Geometric Product：

$$uv = u \cdot v + u \wedge v$$

其中：

  • 内积 $u \cdot v$：捕获特征相干性（标量相似度），类似通道注意力。论文将其解释为 coherence-based diffusion：在特征与上下文一致的区域平滑噪声、稳定学习动态
  • 外积 $u \wedge v$：生成 bivector（双向量），表示 $u$ 和 $v$ 张成的 oriented plane，捕获结构变化和正交性。论文将其解释为 geometric torque / vorticity：当中心特征与局部上下文正交偏离时，产生旋转性的「反应力」，突出边缘和纹理边界

标准的 dot-product attention 只保留标量相似度 $q \cdot k$，丢弃了方向性和结构信息。CliffordNet 使用完整的 geometric product，同时保留 scalar coherence 和 bivector structure，每次局部交互的信息密度远高于标量点积。

代数完备性：标准神经网络只用对称标量分量 $u \cdot v$（丢弃反对称 bivector 分量 $u \wedge v$）。CliffordNet 显式建模两者，称其为 algebraic completeness。如果只用 inner product，会丢失 structural orientation；只用 wedge product，则因 $x \wedge x = 0$ 丢失对角能量。完整 geometric product 同时提供 Energy/Coherence + Structure/Orientation。

连续的 Clifford Interaction Ansatz：

$$F(H, C) = P(H \cdot C \oplus H \wedge C)$$

其中 $P$ 是投影回 $\mathbb{R}^D$ 的线性算子，$\oplus$ 在实现中作为 channel-wise concatenation。$H$ 是特征状态，$C(H)$ 是几何上下文。

Geometric Reaction-Diffusion 解释

如果把 $C \approx \Delta H$ 看作局部 Laplacian，CliffordNet 层可以理解为 geometric reaction-diffusion system。内积 $H \cdot C$ 是 coherence-based diffusion（在一致区域平滑噪声），外积 $H \wedge C$ 是 structural reaction 或 rotational vorticity（在正交偏离处突出边界）。二者一起形成 pattern formation 式的表示演化。这也解释了为什么 Differential Mode（$C \approx \Delta H$）优于 Absolute Mode：高通差分提高了 structural variation 的信噪比。

§4 稀疏滚动
稀疏滚动：线性复杂度的关键

完整计算几何积需要 $O(D^2)$ 的外积展开（所有 channel 两两配对），维度爆炸。CliffordNet 用 Sparse Rolling 机制规避：

定义 channel 循环平移算子 $T_s$，对 offset $s \in S$（例如 $S = \{1, 2, 4, 8, 16\}$ for Lite）：

$$D_s(H, C) = \mathrm{SiLU}(H \odot T_s(C))$$
$$W_s(H, C) = H \odot T_s(C) - T_s(H) \odot C$$

$D_s$ 是内积分支（通过 Hadamard + SiLU 作为非线性 coherence gate），$W_s$ 是外积分支（反对称差分 $h_c c_{c+s} - c_c h_{c+s}$，严格对应 bivector coefficient）。两者 Concat 后线性投影回 $\mathbb{R}^C$。

复杂度：从 $O(ND^2)$ 降到 $O(ND|S|)$，对序列长度 $N$ 和通道 $D$ 线性，$|S|$ 是小常数（Nano 用 2 个 shifts $[1,2]$，Lite 用 5 个 shifts $[1,2,4,8,16]$）。

Circulant Sparsity 的几何意义

完整 channel-wise interaction 形成稠密矩阵 $M \in \mathbb{R}^{D \times D}$。循环平移 $T_s$ 采样该矩阵的特定对角线——这是一种 structured circulant sparsity。作者选择指数型 shifts $\{1, 2, 4, 8, 16\}$ 作为结构性归纳偏置：假设 latent feature channel space 具有 ring-like topology，多尺度依赖可通过固定 offset 的稀疏连接以 logarithmic path length 传播。不学习完整矩阵 $A \in \mathbb{R}^{D \times D}$（那会重新引入 $O(D^2)$ 的 FFN 开销），而是让 projection $P$ 自动吸收 cyclic boundary 导致的 sign flip。

§5 架构
架构详解：Dual-Stream + GGR

CliffordNet Block（No-gFFN-G variant）的 forward pass：

  1. Input Norm: $X_{\text{ln}} = \mathrm{LayerNorm}(X_{l-1})$
  2. Dual-Stream Generation: 确定性流 $Z_{\text{det}} = \mathrm{Linear}(X_{\text{ln}})$ 和 上下文流 $Z_{\text{ctx}} = \mathrm{SiLU}(\mathrm{BN}(\mathrm{DWConv}(X_{\text{ln}})))$
  3. Differential Mode（主要结果使用）: $Z_{\text{ctx}} \leftarrow Z_{\text{ctx}} - Z_{\text{det}}$，把上下文变成离散 Laplacian $C \approx \Delta H$，作为高通结构滤波器
  4. Sparse Rolling Interaction: 对每个 $s \in S$，计算 $W^{(s)}$（wedge）和 $D^{(s)}$（dot），Concat 后 Linear proj → $G_{\text{feat}}$
  5. Gated Geometric Residual (GGR): $M = \mathrm{Cat}([X_{\text{ln}}, G_{\text{feat}}])$，$\alpha = \mathrm{Sigmoid}(\mathrm{Linear}(M))$，$H_{\text{mix}} = \mathrm{SiLU}(X_{\text{ln}}) + \alpha \odot G_{\text{feat}}$
  6. Output Update: $X_l = X_{l-1} + \mathrm{Drop}(\gamma \odot H_{\text{mix}})$，无 FFN

GGR 的三层稳定化动机：(1) SiLU 预滤波抑制负值背景噪声；(2) gate 自适应控制几何力注入强度；(3) LayerScale $\gamma$ 对应 Euler step size $\Delta t$ 的可学习缩放。整体像一个 stabilized numerical solver。

Algorithm 1 完整伪代码（No-gFFN-G Variant） 

Input: Feature map X_{l-1}: (B, H, W, C)
Output: Updated map X_l: (B, H, W, C)

# 1. Input Norm
X_ln = LayerNorm(X_{l-1})

# 2. Dual-Stream Generation
Z_det = Linear_det(X_ln)
Z_ctx = SiLU(BN(DWConv(X_ln)))
if ctx_mode == 'diff':
    Z_ctx = Z_ctx - Z_det          # Differential Mode

# 3. Efficient Interaction (Rolling)
F_list = []
for s in S:                          # e.g., {1, 2, 4, 8, 16}
    Z_d_s = Roll(Z_det, s)
    Z_c_s = Roll(Z_ctx, s)
    W_s = Z_det * Z_c_s - Z_ctx * Z_d_s   # Wedge (bivector)
    D_s = SiLU(Z_det * Z_c_s)              # Dot (scalar coherence)
    if cli_mode == 'full':
        F_list.append(W_s, D_s)

# Concat & Project back to R^C
G_raw = Cat(F_list)       # shape: (..., 2|S|C)
G_feat = Linear_proj(G_raw)  # shape: (..., C)

# 4. Gated Geometric Residual
M = Cat([X_ln, G_feat])
alpha = Sigmoid(Linear_gate(M))
H_mix = SiLU(X_ln) + alpha * G_feat

# 5. Output Update (Note: No FFN applied)
X_l = X_{l-1} + Drop(gamma * H_mix)
return X_l

gFFN 家族（可选增强）：

  • gFFN-L: $C = \Delta H$，local 高频边缘捕捉，驱动 Nano variant
  • gFFN-G: $C = \mathrm{GlobalAvg}(H)$，global 语义相干性，wedge term 突出偏离全局背景的局部特征，inner term 强化与全局场一致的特征
  • gFFN-H: $C = \Delta H + \beta \cdot \mathrm{GlobalAvg}(H)$，通过 field superposition 结合 micro 与 macro context

关键设计选择：CliffordNet 不使用 Clifford-valued weights，只把 geometric product 作为轻量交互算子。这样既保留了 Clifford algebra 的代数结构，又避免了 GATr 等方法中 multivector 权重导致的参数爆炸（4D spacetime algebra 有 16 个分量）。

§6 图解
稀疏滚动 Wedge Product 可视化

循环平移操作将当前状态 $h$ 与通道距离的上下文配对，而非计算完整二次外积。每个 shift 采样 dense correlation matrix 的一条对角线。

图 1：稀疏滚动 Wedge Product 机制。循环平移 $T_s$ 通过采样固定 channel offset 来实现外积，避免显式构建 $D \times D$ 二次张量。几何上，这构造了一个 bivector（绿色平面），表示 $h$ 和 $c$ 张成的 oriented area 和正交性。

  • 完整 loss function（cross-entropy？label smoothing？mixup/cutmix？）
  • Learning rate、weight decay、batch size、AdamW $\beta_1, \beta_2$、DropPath rate
  • 每个 variant 的 depth $L$ 和 channel $D$ 具体值（32/64/96 命名约定未说明）
  • LayerScale $\gamma$ 初始化值
  • gFFN-G 的 broadcasting 细节和 projection 是否与 local 分支共享
  • GPU 型号、PyTorch 版本、CUDA 版本、mixed precision 配置

最小可复现版本：PatchEmbed + 重复 CliffordNet Block（按 Algorithm 1）+ global pooling + classifier，训练用 CIFAR-100/AdamW/cosine/200 epochs/AutoAugment/Random Erasing/patch size 2，其余超参数从开源代码补齐。

官方代码仓库

ParaMind2025/CAN — stars=149, forks=19, 最近更新 2026-03-31。支持 High-performance mode（自定义 CUDA/Triton kernel，RTX 4090 优化）和 Compatibility mode（纯 PyTorch fallback）。实验覆盖 CIFAR-100、Kvasir-Dataset-v2、ISIC2018，暂未在 ImageNet-1K/COCO/ADE20K 上验证。

§12 局限
局限性与开放问题
  • CIFAR-100 验证为主：ImageNet 等大规模数据集结果缺失。No-FFN 范式在数据规模扩大 orders of magnitude 后是否仍然有效，目前是 open question
  • 序列化批评缺乏严格证明：CliffordNet 认为 cross-scan 破坏 2D 流形，但 VMamba 用四方向扫描已部分缓解单一路径偏置。当前 CIFAR-100 结果支持有效性，还不足以完全证明理论解释的因果性
  • 几何阶数有限：当前只使用 Grade 1 × Grade 1 向量乘积（生成 Grade 0 scalar + Grade 2 bivector）。更高阶的 vector-bivector product 未实现
  • Fixed shift topology 是启发式：作者承认当前采用简单固定指数 offsets，inner 与 wedge 共享同一 $S$，消融显示两者之间存在一定程度的信息冗余，可考虑 decoupled sampling
  • 训练细节不透明：完整超参数和模型配置缺失，影响严谨复现
  • arXiv preprint：尚无正式会议接收信息
§13 结论
结论

CliffordNet 提出了一个野心勃勃的主张：用代数完备的 Clifford Geometric Product 同时替代 attention 和 FFN，实现参数效率突破。其核心洞察是「几何积的表示密度足够高，使得 FFN 变得冗余」。

这一工作的价值不在于在 CIFAR-100 上超过 ResNet-18，而在于它提出了一种新的架构设计范式：从数学第一性原理（代数完备性）出发，而非启发式工程模块堆叠。如果 local geometric interaction 足够密集，全局理解可以从中涌现——这暗示了「geometry is all you need」的可能性。

关键开放问题：这种几何驱动的架构能否扩展到 ImageNet 等大规模任务？bivector 结构在更高分辨率、更复杂视觉场景中是否仍然有效？No-FFN 设计在下游任务（检测、分割）中能否维持优势？

「不再问如何让 attention 更几何，而是问：如果几何乘积本身就是 token interaction，还需不需要 attention 和 FFN？」

—— CliffordNet 的核心转向
