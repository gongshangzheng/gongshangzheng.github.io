一国收入的衡量

章节定位

  • 位置：第8篇「宏观经济学的数据」第1章
  • 核心问题：GDP 是什么？如何衡量？它衡量了什么、没衡量什么？
  • 前接：第2章微观/宏观的区分
  • 后接：第24章「生活费用的衡量」
核心概念
GDP 的定义与拆解

国内生产总值（GDP）

在某一既定时期一个国家内生产的所有最终物品与服务的市场价值。

关键词含义排除项
市场价值用市场价格加总不同物品没有市场交易的活动（家务、自种蔬菜）
最终物品只算最终产品中间产品（造纸厂卖给贺卡厂的纸）
物品与服务有形+无形二手交易
一个国家内地理边界本国公民在海外的产出
某一既定时期流量概念存量（如财富）
关键洞察：对整体经济而言，收入必定等于支出。买者的1美元=卖者的1美元收入。
关键公式

GDP 的支出法

$$Y = C + I + G + NX$$

变量含义说明
$Y$GDP（总支出/总收入）
$C$消费（Consumption）家庭物品与服务支出（不含新住房）
$I$投资（Investment）资本品、存货变动、新住房
$G$政府购买不含转移支付
$NX$净出口出口-进口

真实 GDP 与 GDP 平减指数

$$\text{GDP平减指数} = \frac{\text{名义GDP}}{\text{真实GDP}} \times 100$$

$$\text{通货膨胀率} = \frac{\text{GDP平减指数}_{当年} - \text{GDP平减指数}_{上年}}{\text{GDP平减指数}_{上年}} \times 100\%$$

关键区分
指标定义特点
名义 GDP按现期价格评价的生产反映产量和价格变动
真实 GDP按不变基年价格评价的生产只反映产量变动
GDP 的局限：没有衡量闲暇、环境质量、非市场活动、收入分配公平性和地下经济。

要点总结

  • GDP 同时衡量总收入和总支出，两者必然相等
  • Y=C+I+G+NX 是分解 GDP 的基本公式
  • 真实 GDP 是更好的经济福利衡量指标
  • GDP 排除了大量有价值的非市场活动

核心概念
CPI 的计算与构成

消费物价指数（CPI）

普通消费者所购买的物品与服务的总费用的衡量指标。计算五步：固定篮子 → 找价格 → 算篮子费用 → 选基年算指数 → 算通胀率。

$$\text{CPI} = \frac{\text{当年篮子费用}}{\text{基年篮子费用}} \times 100$$

项目美国 CPI 占比
住房42%
交通16%
食物和饮料15%
医疗8%
教育和通信7%
休闲娱乐6%
服装3%
其他3%
关键区分
CPI vs GDP 平减指数
维度CPIGDP 平减指数
覆盖范围消费者购买的所有物品与服务国内生产的所有物品与服务
进口品包含不包含
篮子固定（偶尔调整）随产出组合变动
用途消费者生活费用国内生产物价水平
CPI 的三个问题：替代偏向、新物品引进、无法衡量的质量变动。研究表明 CPI 每年可能高估约 0.5%-1% 的通胀。

通胀校正公式

$$\text{今天的美元数字} = T\text{年的美元数字} \times \frac{\text{今天的物价水平}}{T\text{年的物价水平}}$$

要点总结

  • CPI 用固定篮子法衡量消费者生活费用变动
  • CPI 和 GDP 平减指数的关键差异：进口品和篮子加权方式
  • CPI 存在替代偏向等三个衡量问题
  • 可以用物价指数校正不同时期的美元数字

章节定位

  • 核心问题：什么决定了一国的生活水平？穷国如何赶上富国？
  • 前接：第24章「生活费用的衡量」
  • 后接：第26章「储蓄、投资和金融体系」
核心概念
生产率与增长源泉

生产率

每单位劳动投入所生产的物品与服务的数量。一国的生活水平取决于其生产物品与服务的能力。

决定因素定义举例
物质资本用于生产的设备和建筑物存量车床、推土机、电脑
人力资本工人通过教育获得的知识与技能大学教育、在职培训
自然资源自然界提供的生产投入品土地、矿藏、石油
技术知识生产物品与服务的最佳方法装配线、抗生素
区分：技术知识是"教科书的质量"，人力资本是"阅读教科书的时间量"。
关键公式

生产函数

$$Y = AF(L, K, H, N)$$

规模收益不变：所有投入翻一番 → 产出翻一番。

70 规则

$$\text{收入翻番所需年数} \approx \frac{70}{\text{年增长率}\%}$$

例：年增长2% → 约35年翻一番；年增长7% → 约10年翻一番。

关键区分
概念含义
收益递减资本越多，每新增一单位资本的产出增量越小
追赶效应穷国资本边际回报高，增长更快
马尔萨斯陷阱 vs 新增长理论：马尔萨斯预言人口增长制约生活水平，但过去200年未被证实。技术进步抵消了人口影响。

要点总结

  • 生活水平取决于生产率（每单位劳动的产出）
  • 资本积累受收益递减约束，但穷国可追赶
  • 政府可通过教育、产权、自由贸易、研发促进增长
第26章
储蓄、投资和金融体系

章节定位

  • 位置：第9篇「长期中的真实经济」第2章
  • 核心问题：储蓄如何转化为投资？政府预算如何影响利率？
  • 前接：第25章「生产与增长」
  • 后接：第27章「金融学的基本工具」
核心概念
金融体系与国民储蓄
机构类型定义举例
金融市场直接在借款者与贷款者之间撮合债券市场、股票市场
金融中介机构自身作为贷款者银行、养恤基金、共同基金

国民储蓄恒等式

国民储蓄 S = 私人储蓄 + 公共储蓄

私人储蓄 = Y - T - C

公共储蓄 = T - G

封闭经济中：S = I

关键公式与模型

国民储蓄恒等式

$$S = Y - T - C + (T - G)$$

政策曲线移动利率投资
储蓄激励供给右移↓↑
投资激励需求右移↑↑
预算赤字供给左移↑↓（挤出）
挤出效应：政府借款为预算赤字筹资 → 国民储蓄减少 → 可贷资金供给减少 → 利率上升 → 私人投资被挤出。

要点总结

  • 金融体系将社会储蓄引导至生产性投资
  • 封闭经济中，国民储蓄恒等于投资
  • 可贷资金市场供求决定均衡利率
  • 预算赤字挤出私人投资
