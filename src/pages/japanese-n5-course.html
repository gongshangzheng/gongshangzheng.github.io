这套笔记要解决什么问题

互联网上日语入门资源多到令人眼花缭乱：YouTube 教程、各种 App、博客文章、NHK 节目……但它们大多是碎片化的。你知道五十音图很重要，却不知道学完之后第一步应该做什么；你知道「は」和「が」不一样，却找不到一份系统性的讲解把它们放在同一个框架里说清楚。

这套笔记基于 NHK《簡明日語》48 课教材，按学习路径而非课号组织。每篇笔记聚焦一个核心模块，包含：完整的语法讲解、原始对话例句、练习、易混淆点提示，以及能帮你记忆的插图。目标是让你从"认识五十音"一路走到"能用日语进行简单日常交流"（JLPT N5 / CEFR A1-A2 水平）。

教材背景：NHK《簡明日語》（2019 年 10 月版），通过"春奶奶之家"共享公寓的故事场景教学，48 课覆盖 JFS A1-A2，对应 CEFR A1-A2 / JLPT N5。教材原文 PDF 已嵌入博客，支持在线翻阅。在线课程：nhk.or.jp/lesson/zt

给中文母语者的特别提示

你有一个巨大的不公平优势：汉字。日语中约 2000 个常用汉字你基本都能"猜"出意思。但这同时也是陷阱——你会读汉字，不代表你会读日语的汉字（音读/训读），也不代表你理解它在日语语境中的用法。这套笔记会在关键处标注中文母语者的常见误判点。

📅 学习路径 · 七大模块
Module 1 · 前置

五十音图：平假名与片假名完全入门

在学任何语法之前，你需要先掌握日语的表音文字系统。46 个基本假名、浊音半浊音、拗音、促音、长音——以及为什么同一个发音有两种写法。

  • 平假名 46 音完整图表（清音/浊音/半浊音/拗音）
  • 片假名 46 音完整图表
  • 发音要点：母音、子音、高低重音
  • 中文母语者常见发音陷阱
  • 记忆策略与练习方法
Module 2 · L1-3

基础寒暄与自我介绍

日语日常寒暄语有精确的使用场景。学会最基础的名词句「A は B です」、自我介绍、报出来处，你就能在日本完成第一次完整的对话。

  • 寒暄语体系：早上好/你好/晚上好/谢谢/对不起
  • 名词句 A は B です（判断句）
  • 自我介绍：名字、身份、来处
  • 礼貌体基础：です/ます
  • 原始对话：春奶奶之家初到（L1-3）
Module 3 · L4-8

日常动作与助词系统

助词是日语语法的骨架。掌握六个核心助词（は が を に で の）之后，你就能组装出几乎所有 N5 级别的句子。同时引入动词ます形和テ形请求。

  • 六大助词功能与例句
  • は vs が 的直觉区分
  • 动词ます形：现在/过去/肯定/否定
  • テ形 + ください（请求句）
  • 原始对话：问路/搭电车/介绍朋友（L4-8）
Module 4 · L9-12

购物、指示词与数字

在日本购物和点餐是最高频的场景。这一模块覆盖指示词（これ/それ/あれ）、询问价格、数字系统（含浊音变化的不规则读法），以及存在句「あります/います」。

  • 指示词三档：これ/それ/あれ + この/その/あの
  • 询问价格：いくらですか
  • 数字系统：1-10000 + 浊音变化
  • 存在句：あります（无生命）/ います（有生命）
  • 原始对话：百货地下街/浅草/护身符（L9-12）
Module 5 · L13-22

愿望、邀请与形容词

表达感受和意愿是语言能力的分水岭。い形容词和な形容词的活用、愿望表达（たい/ほしい）、邀请句（ましょう/ませんか），以及否定请求（ないでください）。

  • い形容词 vs な形容词：活用规则
  • 愿望：～たい（想做…）/ ～ほしい（想要…）
  • 邀请：～ましょう / ～ませんか
  • 否定请求：～ないでください
  • 形容词连接：～くて / ～で
Module 6 · L23-34

许可、经验与比较

更复杂的句子结构：请求许可（てもいいですか）、禁止（てはいけません）、经验表达（たことがある）、正在进行（ています），以及比较句（より/ほうが）。

  • 许可：～てもいいですか / 禁止：～てはいけません
  • 经验：～たことがあります
  • 正在进行：～ています
  • 比较：A と B と どちらが…/ A のほうが…
  • 〜すぎる（太…）
Module 7 · L35-48

理由、打算与能力表达

最后冲刺：给出理由（から/ので）、表达意图（つもり/たい）、能力（できる/动词可能形）、传闻（そうです）、样态（そう），以及未来的打算。完成这一模块，你就具备了 N5 级别的完整语法体系。

  • 理由：～から / ～ので
  • 打算：～つもりです
  • 能力：～ができます / 可能形
  • 传闻与样态：～そうです
  • 遗憾：～てしまう
  • 方法：～方
