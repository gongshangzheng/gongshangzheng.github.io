Part 0 · 课程导论
信息论：通信的数学基础

1948 年，Claude Shannon 发表 A Mathematical Theory of Communication，奠定了信息论的数学基础。本课程基于 Cover & Thomas《信息论基础》第2版，围绕一个核心问题展开：在不可靠的信道上，我们能可靠地传输多少信息？

Shannon 将通信抽象为五个组件：

组件符号作用
信源$X$产生消息
编码器$f$映射为传输信号
信道$p(y|x)$传输，可能引入噪声
解码器$g$恢复原始消息
信宿$\hat{X}$接收重建消息

课程的三大支柱定理：

  • 信源编码定理（第5章）：熵 $H(X)$ 是无损压缩的理论极限
  • 信道编码定理（第7章）：当 $R < C$ 时，错误概率可任意小
  • 率失真定理（第10章）：在允许失真 $D$ 下，最少需要 $R(D)$ 比特
教材：Cover & Thomas《信息论基础》第2版，阮吉寿、张华译，机械工业出版社。

前置知识

  • 概率论：联合/条件分布、贝叶斯定理、期望
  • 随机变量：离散/连续分布、概率密度函数
  • 对数与凸函数：$\log_2$、Jensen 不等式
Part 1 · 第2章
熵、相对熵和互信息

1.1 自信息与熵

自信息衡量单个事件的信息量。越不可能发生的事件，信息越多：

$$I(x) = \log \frac{1}{p(x)} = -\log p(x)$$

信息熵是随机变量的平均不确定性（自信息的期望）：

$$H(X) = -\sum_{x \in \mathcal{X}} p(x) \log p(x) = \mathbb{E}[I(X)]$$

二值熵函数 $H_2(p) = -p\log p - (1-p)\log(1-p)$，在 $p=0.5$ 时取最大值 1 bit。

例子：设 $X$ 分布为 $p(0)=0.5, p(1)=0.25, p(2)=0.25$，则 $H(X) = -0.5\log_2 0.5 - 0.25\log_2 0.25 - 0.25\log_2 0.25 = 0.5 + 0.5 + 0.5 = 1.5$ bit。

1.2 联合熵与条件熵

$$H(X,Y) = -\sum_{x,y} p(x,y) \log p(x,y)$$

$$H(X|Y) = -\sum_{x,y} p(x,y) \log p(x|y) = \mathbb{E}[-\log p(X|Y)]$$

链式法则：

$$H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$

推广到 $n$ 个变量：

$$H(X_1, \ldots, X_n) = \sum_{i=1}^{n} H(X_i | X_1, \ldots, X_{i-1})$$

1.3 相对熵（KL 散度）

$$D(p \| q) = \sum_{x} p(x) \log \frac{p(x)}{q(x)} = \mathbb{E}_p\left[\log \frac{p(X)}{q(X)}\right]$$

性质：$D(p\|q) \geq 0$，等号成立当且仅当 $p = q$（信息不等式 / Gibbs 不等式）。

KL 散度不是真正的距离（不满足对称性和三角不等式），但衡量了用 $q$ 编码来自 $p$ 的数据时多花的比特数。

1.4 互信息

$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)$$

$$I(X;Y) = \sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)p(y)} = D(p(x,y) \| p(x)p(y))$$

互信息 = 联合分布与独立假设之间的 KL 散度，即 $X$ 和 $Y$ 偏离独立性的程度。

性质表述
非负性$I(X;Y) \geq 0$
对称性$I(X;Y) = I(Y;X)$
上界$I(X;Y) \leq \min(H(X), H(Y))$
独立性$I(X;Y) = 0 \iff X \perp Y$

1.5 Jensen 不等式与数据处理不等式

Jensen 不等式：若 $f$ 是凸函数，则 $f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]$。对凹函数（如 $\log$），不等号反向。

KL 散度的非负性可由 Jensen 不等式直接证明（以 $\log$ 的凹性）。

数据处理不等式：若 $X \to Y \to Z$ 构成马尔可夫链，则

$$I(X; Z) \leq I(X; Y)$$

直觉：后处理不能创造新信息。这是信息论中最深刻的结论之一。

1.6 Fano 不等式

设 $X \to Y \to \hat{X}$，$P_e = P(X \neq \hat{X})$，则：

$$H(P_e) + P_e \log(|\mathcal{X}|-1) \geq H(X|\hat{X}) \geq H(X|Y)$$

Fano 不等式的意义：如果我们只能观察到 $Y$ 并据此猜测 $X$，那么猜测错误概率 $P_e$ 的下界由条件熵 $H(X|Y)$ 决定。条件熵越大，最小可能的错误率越高。

信息度量关系链：$H(X,Y) = H(X) + H(Y|X)$；$I(X;Y) = H(X) - H(X|Y)$；$D(p\|q) = H(p,q) - H(p)$；$I(X;Y) = D(p(x,y) \| p(x)p(y))$。
Part 2 · 第3章
渐进均分性（AEP）

渐进均分性定理（Asymptotic Equipartition Property）是信息论的核心工具之一。它指出：对于 i.i.d. 随机序列，当长度 $n$ 足够大时，几乎所有序列的"经验熵"都接近理论熵 $H(X)$。

2.1 AEP 定理

设 $X_1, X_2, \ldots, X_n$ 是 i.i.d. $\sim p(x)$，则：

$$-\frac{1}{n} \log p(X_1, X_2, \ldots, X_n) \to H(X) \quad \text{依概率收敛}$$

即：$-\frac{1}{n}\log p(X^n) \xrightarrow{p} H(X)$。长序列的每符号"自信息"趋于熵。

2.2 典型集

定义：序列 $x^n$ 的典型集 $A_\epsilon^{(n)}$ 定义为：

$$A_\epsilon^{(n)} = \left\{ x^n : \left| -\frac{1}{n}\log p(x^n) - H(X) \right| \leq \epsilon \right\}$$

等价表述：$2^{-n(H(X)+\epsilon)} \leq p(x^n) \leq 2^{-n(H(X)-\epsilon)}$

典型集的性质：

  1. 高概率性：$P(A_\epsilon^{(n)}) > 1-\epsilon$（对足够大的 $n$）
  2. 等概率性：典型集中的序列几乎等概率，约 $2^{-nH(X)}$
  3. 大小界：$(1-\epsilon)2^{n(H(X)-\epsilon)} \leq |A_\epsilon^{(n)}| \leq 2^{n(H(X)+\epsilon)}$
直觉：虽然总共有 $|\mathcal{X}|^n$ 个可能序列，但"实际会发生"的序列只有约 $2^{nH(X)}$ 个。若 $H(X) < \log|\mathcal{X}|$，典型集只是全部序列的极小子集。

2.3 AEP 与数据压缩

AEP 给出了信源编码定理的直观证明：只需为典型集中的 $2^{nH(X)}$ 个序列分配索引，用 $nH(X)$ 比特即可表示长度为 $n$ 的序列。

Part 3 · 第4章
随机过程的熵率

第2章的熵 $H(X)$ 只描述单个随机变量。对于随机过程（随机变量序列 $X_1, X_2, \ldots$），我们需要定义熵率来描述信源产生信息的长期平均速率。

3.1 平稳随机过程

随机过程 $\{X_i\}$ 是平稳的，若任意有限子集的联合分布不随时间平移改变：

$$p(X_1, \ldots, X_k) = p(X_{1+l}, \ldots, X_{k+l}), \quad \forall k, l$$

3.2 马尔科夫链

随机过程是马尔科夫链，若：

$$p(X_{n+1} | X_n, X_{n-1}, \ldots, X_1) = p(X_{n+1} | X_n)$$

即"未来只依赖于现在，与过去无关"。用概率转移矩阵 $P$ 描述，$P_{ij} = p(X_{n+1}=j | X_n=i)$。

若马尔可夫链的初始分布 $p(X_1)$ 等于平稳分布 $\mu$（满足 $\mu = \mu P$），则该过程是平稳马尔科夫链。

$$H(\mathcal{X}) = \lim_{n \to \infty} H(X_n | X_{n-1}, \ldots, X_1)$$

Part 4 · 第5章
数据压缩

数据压缩回答一个问题：给定信源分布 $p(x)$，无损编码的最短期望码长是多少？答案由信源编码定理给出：熵 $H(X)$ 是理论下界。

4.1 Kraft 不等式

对任意 $D$ 元前缀码（即时码），码长 $l_1, l_2, \ldots, l_m$ 满足：

$$\sum_{i=1}^{m} D^{-l_i} \leq 1$$

反之，若正整数 $l_i$ 满足 Kraft 不等式，则存在具有这些码长的前缀码。

4.2 信源编码定理（最优码长）

对任意唯一可解码码，期望码长 $L = \sum p_i l_i$ 满足：

$$H_D(X) \leq L < H_D(X) + 1$$

其中 $H_D(X) = H(X)/\log D$ 是以 $D$ 为底的熵。对二进制（$D=2$），$H(X) \leq L < H(X) + 1$。

4.3 Huffman 码

Huffman 编码是最优前缀码构造算法（期望码长最小）。核心思想：概率大的符号分配短码字。

算法步骤：

  1. 将所有符号按概率放入最小堆
  2. 每次取出两个概率最小的节点，合并为新节点（概率 = 两者之和）
  3. 重复直到只剩一个节点（根）
  4. 从根到叶：左分支 = 0，右分支 = 1

Huffman 码的冗余：$H(X) \leq L_{Huffman} < H(X) + 1$。最坏情况下比熵多 1 bit。

4.4 Shannon 码与 Shannon-Fano-Elias 编码

Shannon 码：直接取 $l_i = \lceil \log_D \frac{1}{p_i} \rceil$，满足 Kraft 不等式，且 $H(X) \leq L < H(X) + 1$。

Shannon-Fano-Elias 编码：利用累积分布函数 $F(x) = \sum_{a < x} p(a) + \frac{1}{2}p(x)$，取 $l(x) = \lceil \log \frac{1}{p(x)} \rceil + 1$，用 $F(x)$ 的前 $l(x)$ 位二进制作为码字。这是一种前缀码，期望码长 $L < H(X) + 2$。

Shannon-Fano-Elias 编码是算术编码的离散前身。

数值例子：$p(a)=0.5, p(b)=0.25, p(c)=0.25$。Huffman 码：$a \to 0, b \to 10, c \to 11$，$L = 1.5$ bit = $H(X)$，达到下界。

Part 5 · 第7章
信道容量

信道容量定义了噪声信道的最大可靠传输速率。信道编码定理指出：只要 $R < C$，就存在编码方案使错误概率任意小。

5.1 信道容量定义

离散无记忆信道（DMC）由转移概率 $p(y|x)$ 刻画。信道容量：

$$C = \max_{p(x)} I(X;Y)$$

最大化在输入分布 $p(x)$ 上进行。容量只取决于 $p(y|x)$。

5.2 对称信道

若信道的转移矩阵的行是同一组数的排列，列也是同一组数的排列，则称该信道为对称信道。对称信道的容量在均匀输入分布时达到。

二进制对称信道（BSC）：翻转概率 $p$，容量 $C = 1 - H_2(p)$。

二进制擦除信道（BEC）：擦除概率 $\epsilon$，容量 $C = 1 - \epsilon$。

5.3 联合典型序列与联合 AEP

对 i.i.d. 序列对 $(X^n, Y^n) \sim \prod p(x_i, y_i)$，定义联合典型集：

$$A_\epsilon^{(n)} = \{(x^n, y^n) : |-\frac{1}{n}\log p(x^n) - H(X)| < \epsilon, \; |-\frac{1}{n}\log p(y^n) - H(Y)| < \epsilon, \; |-\frac{1}{n}\log p(x^n, y^n) - H(X,Y)| < \epsilon\}$$

联合典型序列的性质：

  1. $P((X^n, Y^n) \in A_\epsilon^{(n)}) \to 1$
  2. $|A_\epsilon^{(n)}| \approx 2^{nH(X,Y)}$
  3. 若 $\tilde{X}^n, \tilde{Y}^n$ 独立但分别与 $X^n, Y^n$ 同分布，则 $P((\tilde{X}^n, \tilde{Y}^n) \in A_\epsilon^{(n)}) \approx 2^{-nI(X;Y)}$

5.4 信道编码定理

定理：对容量为 $C$ 的 DMC：

  • 若 $R < C$，存在 $(2^{nR}, n)$ 码序列使最大错误概率 $\lambda^{(n)} \to 0$
  • 若 $R > C$，任何 $(2^{nR}, n)$ 码序列的错误概率远离 0

证明思路（随机编码 + 联合典型性解码）：

  1. 随机生成 $2^{nR}$ 个码字，每个码字的 $n$ 个符号 i.i.d. $\sim p(x)$
  2. 发送码字 $x^n(i)$，接收 $y^n$
  3. 解码器找唯一的 $i$ 使 $(x^n(i), y^n)$ 联合典型
  4. 错误概率上界 $\leq \epsilon + 2^{nR} \cdot 2^{-n(I(X;Y)-\epsilon)} \to 0$（当 $R < I(X;Y)$）

5.5 编码方案

重复码：将每个比特重复发送 $n$ 次，可纠正最多 $\lfloor n/2 \rfloor$ 个错误，但码率 $R = 1/n \to 0$。

奇偶校验码：在 $k$ 个信息比特后加 1 个校验比特使总 1 的个数为偶数，可检测奇数个错误。

Hamming 码：$(7,4)$ Hamming 码，4 个信息比特 + 3 个校验比特，可纠正 1 位错误。校验矩阵 $H$ 的列是所有非零的 3 位二进制向量。

5.6 反馈信道与信源信道分离定理

反馈信道：接收端将接收信息反馈给发送端。

定理：反馈不能增加离散无记忆信道的容量。即 $C_{FB} = C$。

但反馈可以简化编码和降低编译码复杂度。

信源信道分离定理：若信源熵率 $H <$ 信道容量 $C$，则存在编码方案使 $P_e \to 0$。反之，若 $H > C$，则不可能。

Part 6 · 第8章
微分熵

离散熵 $H(X)$ 不能直接推广到连续随机变量。微分熵 $h(X)$ 是连续情形的对应物，但其物理含义与离散熵不同。

6.1 定义

$$h(X) = -\int_{-\infty}^{\infty} f(x) \log f(x) \, dx$$

微分熵可为负

与离散熵不同，微分熵可以为负。例如 $X \sim U(0, a)$ 时 $h(X) = \log a$，当 $a < 1$ 时为负值。微分熵不直接表示"比特数"，仅在相对比较中有意义。

6.2 连续随机变量的典型集

类似离散情形，定义体积：

$$A_\epsilon^{(n)} = \{x^n \in \mathbb{R}^n : f(x^n) \in [2^{-n(h+\epsilon)}, 2^{-n(h-\epsilon)}]\}$$

体积满足 $Vol(A_\epsilon^{(n)}) \approx 2^{nh}$。

6.3 联合微分熵、条件微分熵与互信息

$$h(X|Y) = -\int f(x,y) \log f(x|y) \, dx dy$$

$$I(X;Y) = h(X) - h(X|Y) = h(Y) - h(Y|X) = h(X) + h(Y) - h(X,Y)$$

互信息在非负性上保持：$I(X;Y) \geq 0$，尽管 $h(X)$ 本身可能为负。

6.4 多元正态分布的微分熵

对 $X \sim \mathcal{N}(\mu, K)$（$K$ 为协方差矩阵）：

$$h(X) = \frac{1}{2}\log\big((2\pi e)^n |K|\big)$$

特别地，一元正态 $h(X) = \frac{1}{2}\log(2\pi e \sigma^2)$。在所有方差为 $\sigma^2$ 的分布中，高斯分布具有最大的微分熵。

最大熵定理：在给定协方差矩阵 $K$ 约束下，高斯分布使微分熵最大。

Part 7 · 第9章
高斯信道

高斯信道是最重要的一类连续信道：$Y = X + Z$，其中 $Z \sim \mathcal{N}(0, N)$ 为高斯噪声。

7.1 高斯信道的容量

在功率约束 $\mathbb{E}[X^2] \leq P$ 下：

$$C = \frac{1}{2}\log\left(1 + \frac{P}{N}\right) \quad \text{bit/信道使用}$$

达到容量的输入分布是 $X \sim \mathcal{N}(0, P)$。这就是著名的Shannon-Hartley 公式。

7.2 带宽有限信道

若信道带宽为 $W$，噪声功率谱密度为 $N_0/2$，则每秒容量：

$$C = W \log_2\left(1 + \frac{P}{N_0 W}\right) \quad \text{bit/s}$$

例子：电话线 $W = 3000$ Hz，$P/N_0W = 1000$（30 dB），则 $C = 3000 \times \log_2(1001) \approx 29.9$ kbps。

7.3 并联高斯信道

$k$ 个独立并联高斯信道，第 $i$ 个噪声方差 $N_i$。总功率约束 $\sum P_i \leq P$。容量：

$$C = \frac{1}{2}\sum_{i=1}^{k} \log\left(1 + \frac{P_i}{N_i}\right)$$

最优功率分配由注水法（water-filling）给出：$P_i = (\lambda - N_i)^+$，其中 $\lambda$ 使总功率 = $P$。

7.4 高斯彩色噪声信道

噪声不是白噪声（各频率分量相关）。通过 Karhunen-Loève 变换对角化后，转化为并联独立高斯信道，再用注水法分配功率。

7.5 带反馈的高斯信道

与离散情形不同，带反馈的高斯信道容量可以严格增加：

$$C_{n,FB} = \max_{tr(K_X) \leq nP} \frac{1}{2n}\log\frac{|K_{X+Z}|}{|K_Z|}$$

但渐进容量 $C_{FB} = C$（对平稳噪声），即反馈不增加渐近容量。

Part 8 · 第10章
率失真理论

当允许一定失真时，信源可以被压缩到比熵更低的比特率。率失真理论刻画了"率-失真"权衡的精确极限。

8.1 率失真模型

失真度量 $d(x, \hat{x})$：衡量重建质量。常见选择：

  • 汉明失真：$d(x, \hat{x}) = \mathbf{1}_{x \neq \hat{x}}$（误码率）
  • 均方误差（MSE）：$d(x, \hat{x}) = (x - \hat{x})^2$

率失真函数：

$$R(D) = \min_{p(\hat{x}|x): \mathbb{E}[d(X,\hat{X})] \leq D} I(X;\hat{X})$$

$R(D)$ 是在失真不超过 $D$ 的条件下，描述 $X$ 所需的最少比特率。

8.2 率失真函数的性质

  • $R(0) = H(X)$（无损极限）
  • $R(D)$ 是 $D$ 的非增凸函数
  • 当 $D \geq D_{\max}$ 时，$R(D) = 0$

8.3 高斯信源的率失真函数

$X \sim \mathcal{N}(0, \sigma^2)$，MSE 失真：

$$R(D) = \frac{1}{2}\log\frac{\sigma^2}{D}, \quad 0 \leq D \leq \sigma^2$$

达到率失真的重建：$\hat{X} = \frac{\sigma^2 - D}{\sigma^2}X + Z$，其中 $Z \sim \mathcal{N}(0, D(\sigma^2-D)/\sigma^2)$ 且与 $X$ 独立。

8.4 率失真定理

定理：若 $R > R(D)$，则存在率失真码 $(2^{nR}, n)$ 使 $E[d(X^n, \hat{X}^n)] \leq D$。若 $R < R(D)$，则不可能。

失真典型序列：类似 AEP 典型集，定义失真典型序列后，随机编码论证可达性。

8.5 并联高斯信源的率失真

对 $k$ 个独立高斯信源 $X_i \sim \mathcal{N}(0, \sigma_i^2)$，总失真约束 $\sum D_i \leq D$。最优分配由反注水法给出：$D_i = \min(\lambda, \sigma_i^2)$。

与图像压缩的联系：率失真理论是 JPEG、H.264/265 等标准的理论基础。VAE 中的 ELBO = $-E[\log p(x|z)] + D_{KL}(q(z|x)\|p(z))$ 也是率失真优化的变体（重建项 = 失真，KL 项 = 率）。
Part 9 · 末章
信息论与信息安全

信息论不仅解决压缩和传输问题，也为保密通信提供了理论基础。Shannon 1949 年的"保密系统的通信理论"开创了这一方向。

9.1 保密系统模型

保密系统包含：

  • 明文 $M$：原始消息
  • 密钥 $K$：随机选取，与 $M$ 独立
  • 密文 $E = f(M, K)$
  • 解密 $M = g(E, K)$

9.2 完善保密性

定义：若 $I(M; E) = 0$，即密文不泄露任何关于明文的信息，则称该系统具有完善保密性。

等价条件：

  • $I(M; E) = 0$
  • $H(M|E) = H(M)$（已知密文后，明文的不确定性不变）
  • $p(E|M)$ 与 $M$ 无关（所有明文产生任何密文的概率相同）

One-Time Pad：$E = M \oplus K$，$K$ 是与 $M$ 等长的均匀随机串。这是唯一能达到完善保密性的方案，且密钥长度至少等于明文长度。

9.3 冗余度与唯一解距离

自然语言的冗余度：$r = 1 - \frac{H}{\log|\mathcal{X}|}$，其中 $H$ 是每符号实际熵，$\log|\mathcal{X}|$ 是每符号最大熵。

Equivocation：$H(M|E^n)$ 是在观察到 $n$ 个密文符号后，明文剩余的不确定性。

唯一解距离 $n_0$：使 $H(M|E^{n_0}) \approx 0$ 的最小 $n$。直观上，当密文长度超过唯一解距离时，密钥可以被唯一确定。

$$n_0 \approx \frac{H(K)}{r \log|\mathcal{X}|}$$

密钥熵越大、语言冗余度越小，唯一解距离越长，系统越安全。
