PHP：嵌入 HTML 的服务端脚本语言

PHP（PHP: Hypertext Preprocessor）自 1995 年诞生以来，一直是 Web 开发领域最主流的服务端语言之一。它可以直接嵌入 HTML 页面，用 <?php ... ?> 标签包裹服务端逻辑，生成动态页面内容。

截至 2026 年，PHP 8.5 是最新稳定版本，带来了 URI Extension、Pipe Operator、Clone With 等语言革新。本枢纽页按主题组织所有 PHP 相关笔记，形成从入门到实战的完整学习路径。

🚀 PHP 8.5 最新特性

PHP 8.5 — Released Nov 20, 2025

PHP 8.5 是当前最新稳定版本，带来了多项语言层面的革新：

  • Pipe Operator — 新增 |> 操作符，函数链式调用，告别嵌套地狱
  • URI Extension — 内置 URI 解析扩展，替代旧版 parse_url
  • Clone With — 克隆对象时直接修改属性，简化 readonly 类的 "with-er" 模式
  • #[\NoDiscard] — 返回值必须使用，防止 API 调用遗漏
  • Persistent cURL Handles — cURL 句柄跨请求持久化

完整特性列表请参考 PHP 8.x 新特性专题 或 php.net
