What is this?

This is a minimalist blog framework built with pure HTML, CSS, and JavaScript. No frameworks, no build toolchains — just static files served directly.

Quick Start

Writing a new post

Create a Markdown file in src/pages/ with frontmatter, then run node build.js to generate the static site.

Tech Stack

📄
Pure HTML
No template engine runtime
🎨
CSS Variables
Dual theme system
⚡
Vanilla JS
Theme toggle + nav scroll
📝
Markdown
Write in MD, output HTML

Music Demo

You can embed audio players in your posts using the <music-player> custom tag or native HTML5 <audio> elements.
