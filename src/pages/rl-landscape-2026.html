一、LLM + RL：后训练新范式
领域一
从 PPO 到 Self-Distillation
核心 Survey：GFCR (2605.02913) · 47 pages

传统 RL（PPO + Reward Model）在 LLM 后训练中遇到瓶颈：Reward Model 训练成本高、容易过拟合；PPO 样本效率低、训练不稳定；人类偏好数据标注成本极高。

方法演进时间线

阶段方法核心思想局限
1.0PPO + RM训练 Reward Model，PPO 优化RM 成本高、不稳定
2.0DPO直接用偏好数据优化，无需 RM仅适用偏好数据
3.0RLAIFAI 反馈替代人类反馈反馈质量取决于 AI
4.0GRPO组内相对奖励，无需 Critic组内方差大
5.0RLVR可验证奖励（Math/Code）仅适用可验证任务
6.0Self-Distill教师-学生协同进化Teacher 能力触顶

GFCR 四阶段框架

GFCR 将 LLM RL 的 rollout 拆解为 Generate → Filter → Control → Replay 四个模块：

Generate：候选轨迹生成（拓扑结构、脚手架、采样策略）
Filter：中间信号构建（Verifier、PRM、Judge、Learning-Value Signal）
Control：计算分配与决策（预算调度、早停、分支剪枝）
Replay：跨 rollout 复用（缓存、重组合、自演化课程）

近期突破（2026-05）

  • SDAR：Self-Distilled Agentic RL，解决 multi-turn 不稳定，GRPO + OPSD 门控
  • VPD：变分 EM 框架，E-step 自演化 teacher，M-step 学生吸收
  • CM-GRPO：一致性模型 + GRPO，绕过 Euler-Maruyama，用于视频生成
开放问题

Verifier/Judge 校准、计算预算最优分配、Self-evolution 安全性与可追溯性。

二、World Models：从预测到行动
领域二
环境动态的预测表示
核心 Survey：World Model for Robot Learning (2605.00080) · 43 pages

世界模型是环境动态的预测表示，已成为机器人学习的核心组件。支持策略学习、规划、仿真、评估、数据生成。

策略耦合分类法

耦合风格说明代表工作
Decoupled视频生成 + 独立策略UniPi, VidMan, Vidar
Single-backbone统一视觉-动作模型VLA 系列
Latent Dynamics潜空间动态模型Dreamer, SAC, TDM
Unified VLA端到端视觉-语言-动作基础模型驱动

能力层级演进

视频世界模型能力路径：
Imagination-based generation（想象式）→ Controllable（可控）→ Structured（结构化）→ Foundation-scale（基础规模）

作为模拟器的四大应用

  • 策略后训练：低成本 rollout，降低真实环境交互成本
  • 环境评估：在模拟器中验证策略，无需真实机器人
  • 数据生成：合成多样化训练数据
  • 共演化：世界模型与策略联合优化

近期突破

  • AdaWorld：Latent Action 驱动的自适应世界模型
  • PoE-World：Programmatic Experts 组合式世界建模
  • General agents contain world models：理论论证通用 Agent 蕴含世界模型
三、Agentic Reasoning：从 LLM 到自主 Agent
领域三
自主 Agent 的三层维度
核心 Survey：Agentic Reasoning for LLM (2601.12538) · 135 pages

传统 LLM 在封闭世界推理中表现良好，但在开放结尾和动态环境中举步维艰。Agentic Reasoning 范式将 LLM 重构为通过持续交互进行规划、行动、学习的自主 Agent。

三层维度

层次能力核心方法
Foundational单 Agent 规划、工具使用、搜索CoT、ReAct、Plan-then-act
Self-evolving反馈、记忆、适应RL post-training、Memory
Collective多 Agent 协调、知识共享Coordination、Communication

In-context vs Post-training

In-context：通过结构化编排扩展测试时交互（CoT、ReAct）
Post-training：通过 RL/SFT 优化模型行为（GRPO、PPO、DPO）

两者可叠加：SDAR 正是 In-context + Post-training 的融合。
与 GFCR 的关联

Agentic Reasoning 定义了"自主 Agent"的目标形态，而 GFCR 的 rollout 设计服务于 Agentic 能力：Multi-turn 交互 → GFCR Generate 的拓扑结构；Tool use → GFCR Filter 的 verifier 信号；Self-evolution → GFCR Replay 的自演化课程。

五~七、Safe RL · Offline RL · COAML

Safe RL：安全约束与可达性

方法核心思想代表工作
Safety Critics训练安全评估的 CriticCounterexample-Guided Repair
Barrier Certificates构造安全区域的障碍函数Barrier Certified Safety
Constrained MDP将约束融入 MDPCMDP、Lagrangian

Offline RL：从静态数据学习

关键挑战：分布偏移、外推误差、价值高估。核心方法：CQL（保守 Q-learning）、IQL（隐式 Q-learning）、Model-Based Offline。

COAML：决策质量直接优化

传统 PTO 的根本缺陷：预测精度 ≠ 决策质量。COAML 将组合优化求解器作为神经网络的可微分层，实现端到端训练。

范式一致性学习方式损失目标
PTONoDAPrediction Error
Smart-PTOYes†DFRegret / Cost
COAMLYesEnd-to-endDecision Loss
COAML 与 RL 的关系：COAML 中的 RL 分支将决策问题建模为 MDP。与 GFCR 的潜在交叉：COAML 的 "oracle" 类比 GFCR 的 "verifier"；COAML 的 "decision loss" 对应 GFCR 的 "training signal"。

统一趋势

RL 正在从"算法优化"走向"系统设计"：

  • 单一目标优化 → 多组件协同的端到端管道
  • 孤立方法 → 模块化、可组合的框架（GFCR）
  • 纯环境驱动 → LLM 先验 + 世界模型 + RL 的融合
