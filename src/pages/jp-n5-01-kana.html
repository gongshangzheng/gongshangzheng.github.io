为什么必须先学五十音

日语的文字系统由三种文字混合组成：平假名（ひらがな）、片假名（カタカナ）和汉字（漢字）。其中假名是日语的==表音文字==，每一个假名代表一个音节（syllable）。汉字代表语义，而假名承载语法结构和语音信息。

如果你不认识假名，你无法读出任何一个日语句子——即使那个句子里的汉字你都认识。因为助词、动词词尾、形容词变化、所有语法标记，全部用假名书写。

核心事实：平假名和片假名代表完全相同的一组发音（46 个基本音节），只是写法不同。学完平假名后，片假名只是"换一套字形"的问题。推荐先学平假名（使用频率高），再学片假名。
Part 1
五种母音：日语发音的地基

日语只有 5 个母音，而且每个母音的发音在所有音节中保持不变（不像英语那样同一个字母有七八种读法）。掌握这 5 个母音，你就掌握了日语一半的发音。

假名罗马音发音参考口型要点
あ / アa类似汉语"啊"嘴自然张开，舌位低
い / イi类似汉语"衣"嘴角微展，舌位前高
う / ウu类似汉语"乌"但唇不突出嘴唇基本不突出，这是和汉语最大的区别
え / エe类似英语 bed 的 e嘴半开，舌位中
お / オo类似汉语"哦"嘴唇微圆，舌位后
中文母语者发音陷阱

最容易出错的是「う」。汉语的"乌"嘴唇明显前突，但日语的「う」嘴唇几乎不突出。如果你用汉语的"乌"来发日语的 う，日本人一听就知道你是中国人。练习时：发"乌"的音，但嘴唇保持放松、不往前撅。

不规则发音提醒

  • さ行：し 读 shi（不是 si）
  • た行：ち 读 chi（不是 ti），つ 读 tsu（不是 tu）
  • は行：ふ 读 fu（不是 hu），嘴唇几乎不接触，介于 f 和 h 之间
  • ら行：らりるれろ 发音介于英语的 R 和 L 之间，舌尖轻弹上颚（类似美式英语 butter 中的 tt）
  • を：现代日语中发音和 お 几乎相同，但只作为助词使用（标记宾语）
平假名的来源

平假名是从汉字的草书演化而来的。理解这一点有助于记忆字形——很多平假名可以追溯到它对应的汉字原型：

假名来源汉字假名来源汉字
あ → 安い → 以う → 宇え → 衣
か → 加き → 幾く → 久け → 計
さ → 左し → 之す → 寸せ → 世
た → 太ち → 知つ → 川て → 天
な → 奈に → 仁ぬ → 奴ね → 祢
は → 波ひ → 比ふ → 不へ → 部
ま → 末み → 美む → 武め → 女
や → 也ゆ → 由よ → 与
ら → 良り → 利る → 留れ → 礼
わ → 和を → 遠ん → 无

比如「あ」是「安」的草书简写，「か」是「加」的草书。如果你能联想到汉字原形，记忆会快很多。

片假名的用途
  • 外来语：コンビニ（便利店，convenience store）、コーヒー（咖啡，coffee）、テレビ（电视，television）
  • 外国人名/地名：ミーヤー（米亚，Mia）、タム（心心，Tam）
  • 拟声拟态词：ワンワン（汪汪，狗叫）、ドキドキ（心跳声）
  • 强调：类似英语中的斜体（italics）
中文母语者为什么觉得片假名难记

因为平假名在日语中出现频率远高于片假名（助词、动词词尾全是平假名），你自然会对平假名更熟悉。解决方案：刻意练习片假名单词。每学一个片假名，同时记一个用片假名写的外来语单词。如 カ（ka）→ カメラ（照相机，camera）、コ（ko）→ コーヒー（咖啡，coffee）。

Part 4
浊音、半浊音与拗音
浊音（濁音，だくおん）

在清音假名的右上角加两个点（゛，称为濁点/てんてん），就变成了浊音。四行假名有浊音形式：か→が、さ→ざ、た→だ、は→ば。

行浊音罗马音
が行 (from か)が ぎ ぐ げ ごga gi gu ge go
ざ行 (from さ)ざ じ ず ぜ ぞza ji zu ze zo
だ行 (from た)だ ぢ づ で どda ji zu de do
ば行 (from は)ば び ぶ べ ぼba bi bu be bo

注意：じ 和 ぢ 都读 ji，ず 和 づ 都读 zu。现代日语中ぢ和づ几乎不使用。

半浊音（半濁音，はんだくおん）

在は行假名的右上角加一个小圆圈（゜，称为半濁点），变成 p 音：

ぱ行 (from は)ぱ ぴ ぷ ぺ ぽpa pi pu pe po

Part 5
促音（っ）与长音
促音：つ 的缩小版

促音用小写的「っ」（ッ）表示，代表一个音节的停顿。它不发音，但占一个节拍。在罗马音中通常双写后面的辅音。

单词罗马音含义促音位置
がっこうgakkou学校か→が + っ + こう
ずっとzutto一直ず + っ + と
まってmatte等等（て形）ま + っ + て
いっしょにisshoni一起い + っ + しょに
促音的重要性：有无促音会改变词义。如 きた (kita, 来了) vs きった (kitta, 切了)；こて (kote, 鏝) vs こと (koto, 事情)。初学者容易忽略促音的停顿，导致意思不明。
长音：元音的延长

长音是母音延长一个节拍。不同行的假名有不同的长音标记方式：

行长音标记例词含义
あ段后面加 あおかあさん (okaasan)母亲
い段后面加 いせんせい (sensee)老师
う段后面加 うすうがく (suugaku)数学
え段后面加 い 或 えせんせい (sensee)老师
お段后面加 う 或 おおおきい (ookii)大
片假名统一用 长音符号「ー」コーヒー (koohii)咖啡
Part 6
高低重音（ピッチアクセント）

日语的重音是==高低重音==（pitch accent），不是英语那样的强弱重音。每个词的音高模式是固定的——从哪个音节开始下降，或者整体保持高/低。

教材中用 ⎤ 标记音高下降位置：

  • さ⎤むい（寒い，冷）—— 在「さ」之后下降（さ 高、むい 低）
  • あ⎤つい（暑い，热）—— 在「あ」之后下降
  • た⎤だいま（ただいま，我回来了）—— 在「た」之后下降

初学阶段不必过分纠结重音，但要注意：错误的音高可能导致误解。如 は⎤し（桥，hashi⎤）和 はし⎤（筷子，hashi⎤）发音相同但音高模式相反，意思完全不同。

Part 7
记忆策略与练习方法

四种高效记忆方法

  1. 联想记忆法：为每个假名找一个与字形相似的图像或汉字。如「め」像"眼睛"（目），「ね」中间有圆圈（猫 neko 的脸）。
  2. 按行分组法：不要一次背 46 个。先背あ行（a i u e o），再背か行（ka ki ku ke ko），每行 5 个，9 行搞定。や行和わ行只有 3-4 个，很快。
  3. 单词反记法：通过 2-3 个假名组成的简单单词来记忆。如 あめ（雨）、いぬ（狗）、さけ（酒）、ねこ（猫）。
  4. 笔顺书写法：按正确笔顺反复书写。笔顺不是死规矩——正确笔顺写出来的字比例协调、速度快，错误笔顺会导致字形歪斜。基本规则：从左到右、从上到下。

中文记忆法：利用汉字的天然联系

平假名和片假名的字形大多可以从汉字找到对应关系。中文字形是已有的深度记忆，用它来挂靠假名，能事半功倍。核心思路是：找到字形关联 → 联想汉字 → 用汉字的音或义导出假名的音。

平假名：草书化的汉字

平假名本质是汉字草书的简化和流线化，很多字形直接保留了汉字的大致轮廓。记住「它原本是哪个汉字」，假名就不难认了。

易混淆对：片假名专项

有些片假名字形非常接近，是记忆的重灾区。按以下思路区分，一目了然：

混淆对区别要点中文联想
シ vs ツシ 的三点从左到右连下来；ツ 的右下有一点往外伸シ=洗（shi），水往下流；ツ=刺（tsu），往外扎
ソ vs ンソ 起笔在右上；ン 起笔在左下ソ=所（so），右上角开口；ん=嗯（n），左边起笔
ア vs マア 的横线向左收；マ 的横线向右长ア=阿（a），左边短；マ=妈（ma），横线长
ウ vs ゥウ 是完整的片假名；ゥ 是小写的拗音符号ウ=乌（u）；ゥ 永远小于ウ，只出现在复合音节里
ニ vs ソニ 是平直的两横；ソ 有一撇一捺ニ=你（ni），两横很平；ソ=所（so），有撇捺
夕 vs マ夕 的左边是撇；マ 的左边是竖夕=塔（ta），有撇；マ=妈（ma），有竖
记忆优先级建议

对于中文母语者，最容易记住的假名是那些字形几乎等于某个常用汉字的：く（久）、て（天）、た（太）、な（奈）、の（乃）、み（美）。先从这些入手建立信心，再逐一攻破字形较远的。

五十音速查总表

核心数字

  • 基本假名：46 个（平假名 46 + 片假名 46 = 92 个字形，但只有 46 种发音）
  • 浊音：20 个（が/ざ/だ/ば 各 5 个）
  • 半浊音：5 个（ぱ 行）
  • 拗音：33 个常用（きゃ/きゅ/きょ 等 11 组 × 3）
  • 促音：っ（ッ），占一个节拍的停顿
  • 拨音：ん（ン），附在前一个音节后的鼻音

平假名 vs 片假名

  • 平假名：圆滑流畅，用于助词、动词词尾、和语词、语法标记
  • 片假名：方正简洁，用于外来语、外国人名、拟声词、强调
  • 两者发音完全相同，只是字形不同
