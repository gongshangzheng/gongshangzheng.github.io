Part 0
语法和词汇不是两门课

很多学习者把语法和词汇分开练：背单词是一回事，做语法题是另一回事。但母语者的真实加工过程不是这样的——语法模式和词汇搭配在大脑中是作为一个整体（chunk）存储和调用的。

你不会先想到「je」（我）+「avoir」（有）+ 过去时词尾，然后拼出 J'ai eu。你直接提取的是整个语块。这就是为什么本专题的每一个语法点都==嵌在真实例句==里讲解，而不是先列规则再给例句。

核心思路：规则告诉你"为什么"，例句帮你记住"怎么说"，名著原文让你感受"母语者怎么用"。三者缺一不可。

适合谁

本专题面向 B2-C1 水平的学习者。假设你已经系统学过法语基础语法，需要的是查漏补缺、深化理解、激活输出。如果你是 A1-A2，建议先回到 [[how-to-learn-french.html | 系统学习路线]] 打好基础。

怎么用这篇笔记

不要从头读到尾。把它当工具书用：遇到哪个语法点卡壳了，翻到对应章节，把例句读出声来。出声朗读是激活"语感-输出"通道最快的方式。每个例句至少读三遍，第一遍看懂，第二遍感受节奏，第三遍闭上眼睛凭语感复述。

Part 1
名词与限定：法语世界的性别和边界

法语的每一个名词都有阴阳性，每一个名词前面几乎都必须有某种限定词。这不是"规则"，这是法语认知世界的方式——万物皆有名分，皆有归属。

1.1 阴阳性：不只是记忆，更是直觉

初学法语最痛苦的莫过于阴阳性。但大多数名词的性别是有规律可循的：

阳性后缀例词阴性后缀例词
-agele courage, le voyage-tion / -sionla nation, la décision
-mentle développement-téla liberté, la société
-eau / -eule tableau, le feu-eurla couleur, la chaleur
-ismele romantisme-ence / -ancela patience, l'existence
-erle boulanger, le danger-essela tristesse, la sagesse
-oirle soir, le devoir-ièrela lumière, la prière

但规律之外总有例外。与其死记硬背，不如建立一种"法语音感"：在大量阅读中反复接触带冠词的名词，让大脑自动统计出频率。你能感觉到 la table 是阴性、le mur 是阳性——不是因为你记了规则，而是因为你"听惯了"。

「La grammaire est une chanson douce.」

—— Erik Orsenna, La Grammaire est une chanson douce

诺贝尔奖作家 Orsenna 的这本书，把语法比作一首甜美的歌。阴性名词的 la 和阳性名词的 le 就像两个声部，整首歌的旋律就是语法。

1.2 冠词：三套系统的精确切换

法语的冠词系统比中文和英文都复杂得多。它不是"可加可不加"的装饰——冠词承载了名词的"确指程度"和"数量信息"。

类型阳性单数阴性单数复数（阴阳同）核心语义
定冠词le / l'la / l'les双方都知道的特定事物
不定冠词ununedes首次提及、不特定
部分冠词du / de l'de la / de l'des不可数事物的一部分

看三组对比，感受冠词如何精确控制语义：

例句对比：冠词如何改变句意

  • Je mange le pain. — 我在吃那块面包。（说话人和听话人都知道是哪块）
  • Je mange un pain. — 我在吃一个面包。（一整个，初次提及）
  • Je mange du pain. — 我在吃面包。（面包这个种类的一部分，"我在吃些面包"）
关键区分：du pain（部分）≠ un pain（一个完整单位）≠ le pain（特定的那个）。中文翻译可能都是"吃面包"，但法语里这是三种完全不同的认知状态。

在否定句中，不定冠词和部分冠词一律变成 de（或 d'）：

  • J'ai un chat. → Je n'ai pas de chat.（我有一只猫 → 我没有猫）
  • Il boit du café. → Il ne boit pas de café.（他在喝咖啡 → 他不喝咖啡）
  • Il y a des livres. → Il n'y a pas de livres.（有一些书 → 没有书）
定冠词在否定句中不变

Je n'aime pas le café.（我不喜欢咖啡。）这里 le 保持不变——因为你否定的是"咖啡这个整体概念"，而不是"某个不确定的咖啡"。这是中高级学习者最容易忽略的细微差别。

1.3 主有形容词与指示形容词

法语的主有形容词随名词的性数变化：

中文阳性单数阴性单数复数
我的mon livrema tablemes livres
你的ton livreta tabletes livres
他/她的son livresa tableses livres
我们的notre livrenotre tablenos livres
你们的votre livrevotre tablevos livres
他/她们的leur livreleur tableleurs livres

一个容易忽略的规则：当阴性名词以元音或哑音 h 开头时，ma / ta / sa 要变成 mon / ton / son，避免两个元音碰撞。

  • Mon amie（我的女性朋友）——不说 *ma amie
  • Ton école（你的学校）——不说 *ta école
  • Son histoire（他/她的故事）——不说 *sa histoire

「C'est mon ami, c'est mon pays, c'est mon amour.」

—— Édith Piaf, L'Hymne à l'amour

注意 Piaf 在这里用的是 mon 而不是 ma（尽管 amour 在诗歌中可以是阴性），这是元音省略的经典实例。歌曲中的语法往往是最自然的——因为歌手的舌头不会说"不对"的东西。

1.4 名词的复数：看得见的规则

法语名词变复数，最核心的规则是在词尾加 -s，但这个 -s 在口语中几乎不发音！你只能通过冠词来判断单复数：

  • le livre → les livres（口语中 livre 和 livres 发音相同，靠 le vs les 区分）
  • la femme → les femmes（同理，靠冠词）

例外情况要单独记住：

词尾复数变化例词
-al→ -auxun animal → des animaux；un journal → des journaux
-ail→ -ails（大部分）un détail → des détails
-ail（少数）→ -auxun travail → des travaux；un vitrail → des vitraux
-ou（少数）→ -ouxun bijou → des bijoux；un chou → des choux
-eu / -eau / -au→ 加 -xun tableau → des tableaux；un neveu → des neveux
1.5 阴阳性例外：同形不同性的名词

一些名词根据词义不同而有不同的性别。这些词不多，但非常容易出错：

名词阳性含义阴性含义
un livre / une livre一本书一磅（重量单位）
un poste / un poste一个岗位 / 一台设备一个邮局
un mémoire / une mémoire一篇学术论文记忆力
un mode / une mode模式/方式时尚
un tour / une tour一圈/一次塔（la tour Eiffel）
un crêpe / une crêpe一种有褶皱的面料可丽饼
1.6 冠词省略规则

法语中冠词几乎无处不在，但有几个明确的省略场景：

冠词省略的场景

① être 后表示身份/职业（限定性表语）

  • Elle est médecin.（她是医生。）—— 不说 *elle est une médecin
  • Il est professeur.（他是老师。）

② de 后表示性质/材质

  • une robe de soie（一条丝绸裙子）—— 不说 *de la soie
  • une table de bois（一张木头桌子）

③ 固定搭配中的省略

  • avoir faim / soif / froid / chaud（饿/渴/冷/热）
  • à pied / à vélo / en voiture（步行/骑车/开车）
注意：如果表语前有形容词修饰，恢复冠词：Elle est une excellente médecin.（她是一位优秀的医生。）
1.7 指示形容词与 -ci / -là

指示形容词 ce / cet / cette / ces 用来指"这个/那个"：ce livre（这本书）、cette maison（这座房子）、cet homme（这个男人——元音省略）。

加上 -ci（近指）和 -là（远指）可以精确区分远近：

-ci / -là 对比

  • Je prends celui-ci, pas celui-là.（我要这个，不要那个。）
  • Ces jours-ci, il fait froid.（这些天，天气很冷。）
  • Ceux-là sont plus chers.（那些更贵。）

« Les hommes de cette époque ne comprenaient pas les hommes de cette autre époque. Ceux-ci pensaient avoir raison ; ceux-là en étaient certains. »

—— 模仿 Flaubert 叙事风格的自编例句

Part 1 复习速查

  • 阴阳性规律：后缀暗示性别（-tion 阴性，-ment 阳性），但大量阅读培养的"直觉"更可靠
  • 三种冠词：定冠词（已知/整体）、不定冠词（首次/不特定）、部分冠词（不可数的一部分）
  • 否定冠词变换：不定/部分冠词 → de；定冠词保持不变
  • 元音省略：ma/ta/sa 在元音前变成 mon/ton/son
  • 复数标记：书面 -s（口语不发音），特殊词尾 -aux / -oux / -x
