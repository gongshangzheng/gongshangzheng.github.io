Chapter 7
大模型隐写术

本章实现基于 GPT-2 的文本隐写术（Steganography），即在看似正常的文本中隐藏秘密信息。隐写过程对读者完全不可感知，信息可被完整提取。展示了语言模型在信息隐藏领域的独特潜力。

原理
隐写原理
  1. 将秘密信息转换为紧凑的二进制序列（霍夫曼编码或固定长度编码）
  2. 通过编码规则约束 GPT-2 的 token 采样过程，使选中的 token 携带二进制位
  3. 生成表面完全正常的文本，读者无法察觉任何异常
  4. 解码：使用相同的模型和上下文，逆向提取隐藏的二进制序列，还原原始信息
实现
两种编码方案
  • 霍夫曼编码：根据 token 频率自适应分配编码长度，高频 token 用短编码，提高容量效率
  • 固定长度编码：每个 token 携带固定位数的信息，实现简单但效率较低
