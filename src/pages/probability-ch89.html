一、概述
一、概述：统计推断的两大核心方法

统计推断的核心任务：利用来自总体分布 $f_X(x)$ 的随机样本 $\mathbf{X}_n = (X_1, \dots, X_n)$ 的观测数据 $\mathbf{x}_n$，对总体分布进行推断。

在参数方法中，我们假设一族候选分布 $\mathbb{F} = \{f(\cdot, \theta) : \theta \in \Theta\}$，若存在 $\theta_0 \in \Theta$ 使得 $f_X(x) = f(x, \theta_0)$ 对几乎所有 $x$ 成立，则称模型正确设定；否则为误设。

统计推断分为两大支柱：

1. 参数估计（第8章）：用统计量 $\hat\theta(\mathbf{X}_n)$ 估计未知参数 $\theta_0$

2. 假设检验（第9章）：基于数据判断关于 $\theta_0$ 的某个假设是否成立

二、第8章
二、第8章：参数估计
2.1 点估计方法

#### 2.1.1 极大似然估计（MLE）

核心思想（Fisher 提出）：选择使观测数据出现概率最大的参数值。

似然函数：给定数据 $\mathbf{x}_n$，联合 PMF/PDF 作为 $\theta$ 的函数：

$$\hat L(\theta \mid \mathbf{x}_n) = f_{\mathbf{X}_n}(\mathbf{x}_n, \theta)$$

MLE 定义：

$$\hat\theta \equiv \hat\theta_n(\mathbf{X}_n) = \arg\max_{\theta \in \Theta} \hat L(\theta \mid \mathbf{X}_n)$$

关键区别：似然函数 $\hat L(\theta \mid \mathbf{x}_n)$ 与联合 PMF/PDF $f_{\mathbf{X}_n}(\mathbf{x}_n, \theta)$ 数值相等，但概念不同。前者视 $\theta$ 为变量，后者视 $\mathbf{x}_n$ 为变量。

IID 情形下似然函数和对数似然函数分解为：

$$\hat L(\theta \mid \mathbf{X}_n) = \prod_{i=1}^n f(X_i, \theta), \quad \ln \hat L(\theta \mid \mathbf{X}_n) = \sum_{i=1}^n \ln f(X_i, \theta)$$

MLE 存在性的充分条件：若 $\hat L(\theta \mid \mathbf{X}_n)$ 是 $\theta \in \Theta$ 的连续函数，$\Theta$ 为紧集（有界闭集），则 MLE 几乎必然存在。

MLE 的不变性：若 $\hat\theta$ 是 $\theta_0$ 的 MLE，则对任意函数 $g(\cdot)$，$g(\hat\theta)$ 是 $g(\theta_0)$ 的 MLE。

经典例题：

分布MLE
$N(\mu, \sigma^2)$$\hat\mu = \bar X_n$，$\hat\sigma^2 = n^{-1}\sum(X_i - \bar X_n)^2$
$\text{Bernoulli}(\theta)$$\hat\theta = \bar X_n$
$\text{Exponential}(\lambda)$$\hat\lambda = \bar X_n^{-1}$

#### 2.1.2 矩估计法（MME）与广义矩方法（GMM）

MME（最古老的参数估计方法之一）：令样本矩等于总体矩，建立方程组求解参数。

基本步骤：

1. 从模型 $f(x, \theta)$ 计算前 $p$ 阶总体矩 $E_\theta(X^k)$（$k=1,\dots,p$）

2. 用样本矩 $\hat\mu_k = n^{-1}\sum X_i^k$ 替代总体矩

3. 解方程组得到矩估计量

GMM（Hansen, 1982）：将 MME 推广。利用 $q \geq p$ 个矩条件 $E[m(X_i, \theta_0)] = 0$，通过最小化二次型

$$Q(\theta) = \bar m(\theta)' W \bar m(\theta)$$

其中 $\bar m(\theta) = n^{-1}\sum_{i=1}^n m(X_i, \theta)$，$W$ 为权重矩阵。

最优权重：当 $W = V^{-1}$（矩条件的方差-协方差矩阵的逆）时，GMM 达到最高渐近效率。

MLE vs GMM：

  • MLE 利用完整分布信息，一般更有效，但需要正确设定概率模型
  • GMM 只需矩条件，不需要完整分布信息，在模型误设时更稳健
2.2 估计量评估标准

#### 2.2.1 无偏性

$$\text{Bias}_\theta(\hat\theta) = E_\theta(\hat\theta) - \theta = 0$$

无偏估计量在平均意义上给出正确估计，无系统性偏差。

注意：无偏性不总是存在，也不总是最优。例如 $N(\mu, \sigma^2)$ 中 $\mu^2$ 的无偏估计量为 $\bar X_n^2 - S_n^2/n$，比直觉上的 $\bar X_n^2$ 更复杂。

#### 2.2.2 均方误（MSE）准则

$$\text{MSE}_\theta(\hat\theta) = E_\theta(\hat\theta - \theta)^2$$

MSE 分解定理：

$$\text{MSE}_\theta(\hat\theta) = \text{var}_\theta(\hat\theta) + [\text{Bias}_\theta(\hat\theta)]^2$$

这一分解揭示了精度与变异之间的权衡。用射击类比：

  • 低偏差 + 低方差 = 精准且稳定（最优）
  • 低偏差 + 高方差 = 中心正确但分散
  • 高偏差 + 低方差 = 稳定但偏离目标

相对有效性：若 $\text{MSE}_\theta(\hat\theta_1) \leq \text{MSE}_\theta(\hat\theta_2)$，则 $\hat\theta_1$ 在 MSE 准则下更有效。

#### 2.2.3 一致性（相合性）

$$\hat\theta_n \xrightarrow{a.s.} \theta_0 \quad (n \to \infty)$$

即当样本量趋于无穷时，估计量几乎必然收敛到真实参数值。MLE 在正则条件下具有强一致性（定理 8.4）。

2.3 Fisher 信息量与 Cramér-Rao 下界

#### 2.3.1 记分函数与 Fisher 信息

记分函数：$S(X_i, \theta) = \frac{\partial \ln f(X_i, \theta)}{\partial \theta}$

引理 8.3：$E_\theta[S(X_i, \theta)] = 0$。即在正确参数值下，记分函数的期望为零。

Fisher 信息：

$$I(\theta) = E_\theta\left[\left(\frac{\partial \ln f(X_i, \theta)}{\partial \theta}\right)^2\right]$$

直观含义：$I(\theta)$ 测度了总体分布中包含多少关于 $\theta$ 的信息。似然函数关于 $\theta$ 变化越陡峭（尖峰），信息量越大，参数越容易辨识。

信息等式（引理 8.4）：$I(\theta) + H(\theta) = 0$

其中 $H(\theta) = E_\theta\left[\frac{\partial^2 \ln f(X_i, \theta)}{\partial \theta^2}\right]$ 是黑塞函数（测度对数似然函数的曲率）。

这意味着：Fisher 信息 = 记分函数平方的期望 = 对数似然曲率绝对值的期望。

#### 2.3.2 Cramér-Rao 下界

定理 8.11：对 $\tau(\theta)$ 的任意无偏估计量 $\tilde\tau$：

$$\text{var}_\theta(\tilde\tau) \geq B_n(\theta) = \frac{[\tau'(\theta)]^2}{E_\theta\left[\left(\frac{\partial \ln \hat L(\theta \mid \mathbf{X}_n)}{\partial \theta}\right)^2\right]}$$

IID 情形简化为：

$$B_n(\theta) = \frac{[\tau'(\theta)]^2}{n \cdot I(\theta)} = -\frac{[\tau'(\theta)]^2}{n \cdot H(\theta)}$$

C-R 下界法判定最优性的逻辑：

  • 若 $\text{var}_\theta(\tilde\tau) = B_n(\theta)$，则 $\tilde\tau$ 是最优无偏估计量
  • 若 $\text{var}_\theta(\tilde\tau) > B_n(\theta)$，**无法确定** $\tilde\tau$ 是否最优（这是 C-R 下界法的缺陷）
  • 达到 C-R 下界的充要条件：$\tilde\tau - \tau(\theta)$ 与记分函数成线性关系

例：IID Poisson($\lambda$) 下 $\bar X_n$ 达到 C-R 下界 $\lambda/n$，是最优无偏估计量。但 $N(\mu, \sigma^2)$ 下 $S_n^2$ 未达到 C-R 下界 $2\sigma^4/n$，然而 $S_n^2$ 仍然可能是在其他意义下的最优。

2.4 大样本性质：渐近正态性与渐近有效性

定理 8.5（MLE 的渐近正态性）：在正则条件下：

$$\sqrt{n}(\hat\theta - \theta_0) \xrightarrow{d} N(0, -H(\theta_0)^{-1})$$

证明要点：

1. 一阶条件：平均记分函数在 $\hat\theta$ 处为零

2. 中值定理展开

3. CLT → 记分函数均值渐近正态

4. USLLN → 样本黑塞矩阵几乎必然收敛

5. Slutsky 定理完成证明

渐近有效性：MLE 的渐近方差 $-H(\theta_0)^{-1}$ 等于 C-R 下界的逆，故 MLE 渐近地达到了所有无偏估计量方差的下界，是渐近最有效估计量。

2.5 区间估计与置信区间

利用渐近正态性，$\theta_0$ 的渐近 $100(1-\alpha)\%$ 置信区间为：

$$\hat\theta - \frac{z_{\alpha/2}}{\sqrt{-n \hat H(\hat\theta)}} \leq \theta_0 \leq \hat\theta + \frac{z_{\alpha/2}}{\sqrt{-n \hat H(\hat\theta)}}$$

置信区间的正确解读：$P(\hat\theta_L \leq \theta_0 \leq \hat\theta_U) \to 1-\alpha$ 描述的是随机区间覆盖真实参数的概率，不等于 "$\theta_0$ 落在该区间内的概率为 $1-\alpha$"（$\theta_0$ 是固定值，不是随机变量）。

三、第9章：假设检验
3.1 假设检验框架

假设是关于总体分布的表述，分为：

  • **原假设** $H_0: \theta \in \Theta_0$（通常代表"没有效果"或"没有关系"）
  • **备择假设** $H_A: \theta \in \Theta_A$（对原假设的否定）

参数空间被划分为互斥且完备的两个子集：$\Theta_0 \cap \Theta_A = \emptyset$，$\Theta_0 \cup \Theta_A = \Theta$。

简单假设 vs 复合假设：简单假设仅包含一个分布（即一个参数值），复合假设包含多个。

检验统计量 $T(\mathbf{X}_n)$ 将样本空间划分为：

  • **接受域** $\mathbb{A}_n(c) = \{\mathbf{x}_n : |T(\mathbf{x}_n)| \leq c\}$
  • **拒绝域** $\mathbb{C}_n(c) = \{\mathbf{x}_n : |T(\mathbf{x}_n)| > c\}$

$c$ 为临界值。

3.2 两类错误与显著性水平
$H_0$ 为真$H_A$ 为真
**拒绝 $H_0$**第一类错误（弃真），概率 $\alpha(\theta)$正确决策
**不拒绝 $H_0$**正确决策第二类错误（取伪），概率 $\beta(\theta)$

功效函数：

$$\pi(\theta) = P_\theta(\mathbf{X}_n \in \mathbb{C})$$
  • 当 $\theta \in \Theta_0$：$\pi(\theta)$ = 犯第一类错误的概率
  • 当 $\theta \in \Theta_A$：$\pi(\theta) = 1 - \beta(\theta)$ = 正确拒绝的概率

Neyman-Pearson 方法：控制第一类错误概率不超过 $\alpha$（显著性水平），同时使第二类错误概率尽可能小。

两类错误的根本权衡：对固定的 $n$，减小 $\alpha$ 必然增大 $\beta$。唯一的出路是增大样本量。

3.3 p 值

p 值：在 $H_0$ 为真的前提下，观测到比当前检验统计量更极端值的概率。

$$p\text{-value} = P_{H_0}(|T(\mathbf{X}_n)| > |T(\mathbf{x}_n)|)$$

决策规则：若 p 值 $\leq \alpha$，则在显著性水平 $\alpha$ 上拒绝 $H_0$。

p 值的常见误解（Greenland et al., 2016 的经典文献列出了 25 种）：

  • ❌ p 值是 $H_0$ 为真的概率
  • ❌ $p = 0.05$ 意味着结果有 95% 的概率可重复
  • ❌ p 值衡量效应的大小
  • ✅ p 值仅衡量数据与 $H_0$ 的不一致程度，**不是** $H_0$ 的真假概率
3.4 三大经典检验方法

以下三种检验均考虑约束假设 $H_0: g(\theta_0) = \mathbf{0}$，其中 $g: \mathbb{R}^p \to \mathbb{R}^J$ 是约束函数。

#### 3.4.1 Wald 检验

思路：利用无约束估计量 $\hat\theta$，检验 $g(\hat\theta)$ 是否接近零。

$$W = n \cdot g(\hat\theta)' \left[\hat G \hat V \hat G'\right]^{-1} g(\hat\theta) \xrightarrow{d} \chi_J^2$$

其中 $\hat G = G(\hat\theta) = \frac{dg(\hat\theta)}{d\theta}$，$\hat V$ 是 $\sqrt{n}(\hat\theta - \theta_0)$ 的渐近方差的一致估计量。

当 $\hat\theta$ 为 MLE 时，$\hat V = [-\hat H(\hat\theta)]^{-1}$。

特点：只需要无约束 MLE，不需要有约束估计。

#### 3.4.2 Lagrange Multiplier（LM）检验

思路：利用有约束估计量 $\tilde\theta$，检验拉格朗日乘子 $\hat\lambda$ 是否接近零。

$$LM = n\hat\lambda' \hat G(\tilde\theta) \hat H(\tilde\theta)^{-1} \hat G(\tilde\theta)' \hat\lambda \xrightarrow{d} \chi_J^2$$

其中 $\tilde\theta$ 是在约束 $g(\theta) = 0$ 下的 MLE，$\hat\lambda$ 是对应的拉格朗日乘子估计值。

直觉：$\hat\lambda$ 测度了施加约束后似然函数的边际损失。若约束正确，损失应接近零。

特点：只需要有约束 MLE，不需要无约束估计。

#### 3.4.3 似然比（LR）检验

思路：比较有约束与无约束似然值的比值。

$$LR = 2n[\hat l(\hat\theta) - \hat l(\tilde\theta)] \xrightarrow{d} \chi_J^2$$

其中 $\hat l(\theta) = \frac{1}{n}\sum_{i=1}^n \ln f(X_i, \theta)$ 是标准化对数似然函数。

直觉：若约束正确，施加约束不会使似然值显著下降，LR 接近零。

特点：需要同时计算有约束和无约束 MLE。

#### 3.4.4 三大检验的关系

在概率模型正确设定且 $H_0$ 成立时，$W$、$LM$、$LR$ 三者渐近等价（$\xrightarrow{d} \chi_J^2$），差异为 $o_p(1)$。

检验需要的估计量对模型误设的稳健性
Wald无约束 $\hat\theta$可构造稳健版本
LM有约束 $\tilde\theta$可构造稳健版本
LR两者都需要**无法**构造稳健版本

Neyman-Pearson 引理：对于简单假设 $H_0$ vs 简单 $H_A$，似然比检验是一致最大功效（UMP）检验。

3.5 常见检验方法
检验适用场景检验统计量渐近分布
**Z 检验**$\sigma^2$ 已知，检验 $\mu$$Z = \frac{\bar X_n - \mu_0}{\sigma/\sqrt{n}}$$N(0,1)$
**t 检验**$\sigma^2$ 未知，检验 $\mu$$t = \frac{\bar X_n - \mu_0}{S_n/\sqrt{n}}$$t_{n-1}$
**$\chi^2$ 检验**检验方差/拟合优度/独立性$\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$$\chi^2_{df}$
**F 检验**比较两个方差/回归整体显著性$F = S_1^2/S_2^2$$F_{n_1-1, n_2-1}$
3.6 经济学应用示例

教育回报率检验（例 9.1）：检验教育对工资的边际效应 $\theta$ 是否为零，即 $H_0: \theta = 0$。

规模报酬不变检验（例 9.3）：对 Cobb-Douglas 生产函数 $Y = AK^\alpha L^\beta$，规模报酬不变等价于 $H_0: \alpha + \beta = 1$。这是一个 $g(\theta) = \alpha + \beta - 1 = 0$ 的约束检验。

四、关键定理与推导
四、关键定理与推导
4.1 MLE 渐近正态性的证明架构

这是第8-9章最核心的证明链：

1. 引理 8.3：$E_\theta[S(X_i, \theta)] = 0$（由 $\int f(x,\theta)dx = 1$ 对 $\theta$ 求导得）

2. 引理 8.4：$I(\theta) + H(\theta) = 0$（上式再求一次导得）

3. 一致性：$\hat\theta \xrightarrow{a.s.} \theta_0$（极值估计量理论）

4. 中值定理展开：$\frac{1}{n}\sum S(X_i, \hat\theta) = 0 \Rightarrow \frac{1}{n}\sum S(X_i, \theta_0) + \hat H(\bar\theta)(\hat\theta - \theta_0) = 0$

5. CLT → $\frac{1}{\sqrt{n}}\sum S(X_i, \theta_0) \xrightarrow{d} N(0, I(\theta_0))$

6. USLLN → $\hat H(\bar\theta) \xrightarrow{a.s.} H(\theta_0)$

7. Slutsky 定理 → $\sqrt{n}(\hat\theta - \theta_0) \xrightarrow{d} N(0, I(\theta_0)^{-1}) = N(0, -H(\theta_0)^{-1})$

4.2 GMM 渐近正态性
$$\sqrt{n}(\hat\theta - \theta_0) \xrightarrow{d} N(0, \Omega)$$

其中：

  • 最优权重 $W = V^{-1}$ 时：$\Omega = (G_0'V^{-1}G_0)^{-1}$（定理 8.8-8.9）
  • $G_0 = G(\theta_0)$，$V = E[m(X_i, \theta_0)m(X_i, \theta_0)']$

最优 GMM 的渐近方差 $(G_0'V^{-1}G_0)^{-1}$ 不大于任意其他权重下的渐近方差。

五、关键例题
五、关键例题
例 1：伯努利分布下的三大检验

$X_1, \dots, X_n \sim \text{iid Bernoulli}(\theta)$，检验 $H_0: \theta = \theta_0$。

MLE：$\hat\theta = \bar X_n$

  • **Wald**：$W = \frac{n(\bar X_n - \theta_0)^2}{\bar X_n(1 - \bar X_n)} \xrightarrow{d} \chi_1^2$
  • **LM**：$LM = \frac{n(\bar X_n - \theta_0)^2}{\bar X_n(1-\theta_0)^2 + (1-\bar X_n)\theta_0^2} \xrightarrow{d} \chi_1^2$
  • **LR**：$LR = 2n\left[\bar X_n \ln\frac{\bar X_n}{\theta_0} + (1-\bar X_n)\ln\frac{1-\bar X_n}{1-\theta_0}\right] \xrightarrow{d} \chi_1^2$
例 2：正态分布下的均值检验

$X_1, \dots, X_n \sim \text{iid } N(\mu, \sigma^2)$，检验 $H_0: \mu = \mu_0$。

  • **Wald**：$W = \frac{n(\bar X_n - \mu_0)^2}{\hat\sigma^2} \xrightarrow{d} \chi_1^2$
  • **LM**：使用有约束估计 $\tilde\sigma^2 = n^{-1}\sum(X_i - \mu_0)^2$，LM 基于 $\tilde\sigma^2$ 构造
  • **LR**：$LR = n\ln(\tilde\sigma^2 / \hat\sigma^2) \xrightarrow{d} \chi_1^2$
例 3：BLUE 的推导

$X_1, \dots, X_n \sim \text{iid } (\mu, \sigma^2)$。在所有线性无偏估计 $\hat\mu = \sum c_i X_i$（$\sum c_i = 1$）中，最小化方差 $\sigma^2 \sum c_i^2$ 得到 $c_i = 1/n$，即 $\hat\mu = \bar X_n$ 是 BLUE。

例 4：Poisson 分布中 C-R 下界验证

$X_1, \dots, X_n \sim \text{iid Poisson}(\lambda)$：$H(\lambda) = -1/\lambda$，C-R 下界 $B_n(\lambda) = \lambda/n$，而 $\text{var}(\bar X_n) = \lambda/n$，恰好达到下界。
