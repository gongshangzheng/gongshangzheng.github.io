引言：为什么需要 GRN？

视觉生成领域长期存在两条并行的技术路线：扩散模型与自回归（AR）模型。扩散模型以 DiT、Sora、FLUX 为代表，通过连续空间中的渐进去噪实现高质量生成，但存在固有的「固定步数」瓶颈——无论生成内容简单还是复杂，所有样本都消耗相同的计算资源。自回归模型则受 LLM 范式启发，将视觉生成建模为序列预测任务，但面临两大核心缺陷：一是离散 tokenizer 的重建质量显著低于连续 VAE，二是严格因果预测机制导致「一步错、步步错」的误差累积问题。

ByteDance 在 2026 年 4 月提出的 Generative Refinement Networks（GRN），试图在保留 AR 模型自适应计算优势的同时，系统性解决上述两个瓶颈。论文提出了三项核心创新：

  1. HBQ（Hierarchical Binary Quantization）：通过分层二值量化，使量化误差理论上以指数速度衰减（第 j 轮误差 < 1/2^j），在不增加潜在通道数的前提下，首次让离散 tokenizer 在同等 latent 维度下匹敌连续 tokenizer；
  2. 全局细化机制（Global Refinement）：突破传统 AR 模型的严格因果性，允许模型像人类画家一样「回头修改」已生成的 token，从根本上缓解多步误差累积；
  3. 熵引导自适应采样（Entropy-Guided Sampling）：利用 AR 模型固有的可变似然特性，以 token 分布的熵作为复杂度代理，动态分配生成步数——简单区域快速通过，复杂区域精细迭代。
核心主张：GRN 在 ImageNet 256×256 上同时创下重建（rFID 0.56）和生成（gFID 1.81）的新纪录，并成功扩展至 1024×1024 文本到图像生成和 480p 文本到视频生成（2–10 秒），证明该框架在质量和效率上均具备实际部署价值。

相关工作与技术背景

理解 GRN 的创新位置，需要先厘清三个相关技术领域的现状与局限。

视觉 Tokenizer：从 VQVAE 到无查找表方法

视觉 Tokenizer 是生成模型的基础组件，负责将高维连续图像/视频压缩为紧凑的潜在表示。现有方法分为两大类：

  • 向量量化方法（VQVAE、VQGAN）：通过可学习码本将连续特征映射为离散索引。受限于码本规模，重建质量有限，且存在「索引崩溃」问题——部分码本向量从未被使用。
  • 无查找表方法（BSQ、FSQ）：通过分解量化空间避免显式码本，支持更大词汇表，但仍与连续表示存在性能差距。

近期 Infinity、BitDance 等工作通过大幅扩展词汇表规模超越了连续 VAE，但代价是收敛速度变慢和生成模型体积增大。GRN 提出的 HBQ 通过分层二值量化，在不增加通道数的情况下实现指数级误差衰减，首次使离散 tokenizer 在相同潜在维度下与连续 tokenizer 持平。

自回归视觉生成：因果预测的枷锁

自回归视觉生成按顺序逐元素预测，有两种变体：

  • 逐 token 预测（VQGAN、LlamaGen）：按光栅扫描顺序逐个预测空间 token；
  • 逐尺度预测（VAR）：从低分辨率到高分辨率逐步生成，提升质量并实现 10 倍以上推理加速。

然而，AR 模型的严格因果预测机制导致「无法回溯修正先前错误」——这是论文指出的核心缺陷。即使 MaskGIT 等并行解码模型，高置信度 token 一旦生成就不可修改。Infinity 尝试通过随机翻转比特 token 实现自校正，但其假设误差率低于 30%，覆盖范围有限。GRN 的全局细化机制从根本上解决了这一缺陷，允许像人类画家一样逐步完善和修正作品。

扩散模型：固定步数的「一刀切」困境

扩散模型通过学习速度场将简单噪声先验逐步传输到数据分布。但扩散模型通过 MSE 优化，缺乏显式似然估计，被限制为固定步数推理——对所有样本分配相同计算资源，无论复杂度差异。

蒸馏方法（DMD、DMD2）可减少采样步数，但仍依赖预定义的固定步数调度。AdaDiff 使用外部网络确定实例特定步数，但需额外网络和奖励信号，流程复杂。GRN 的差异化在于：通过熵引导采样策略，利用 AR 模型固有的可变似然特性，无需额外网络即可实现复杂度感知的自适应步长生成。

前置知识：理解 GRN 的理论基础

Haar 小波与粗到细分解

Haar 小波是最简单的小波变换，由 Alfréd Haar 于 1910 年提出。它将信号分解为不同频率分量：低频近似（均值）和高频细节（差值）。通过迭代应用，实现从粗到细（coarse-to-fine）的信号表示——每一层将信号减半，同时捕获更精细的细节。

论文明确将 HBQ 的灵感归于 Haar 小波（"Inspired by Harr Wavelet in signal processing"）。HBQ 的每一轮二进制量化对应于从粗到细的频率分解：第一轮决定符号（正/负），误差上界 1/2；第二轮细化，误差上界 1/4；第 j 轮误差上界 1/2^j，呈指数衰减。这种二进制树桶结构与 Haar 小波的二分分解同构，实现「无需增加潜在通道数即可指数衰减重建误差」。

离散 Token 化 vs 连续表示

离散 Token 化的优势在于与 NLP token 统一，可直接使用自回归模型，天然适合分类分布建模。但劣势也很明显：信息损失（"有损"）、梯度传播困难（需 STE 直通估计器）、码本崩溃。连续表示保留实值潜在向量，信息无损、梯度传播顺畅、重建质量高，但需特殊处理（如流匹配/扩散）才能用于生成。

论文核心立场：当前 AR 视觉生成「受限于有损离散 token 化和误差累积」。HBQ 通过「理论上近无损」的量化，使离散 tokenizer 在相同潜在维度下匹敌连续 tokenizer——这是离散表示范式的根本性突破。

信息熵与复杂度感知

信息熵 H(X) = −Σ p(x) log p(x) 度量随机变量的不确定性。熵越高，分布越「平坦」，预测越困难；熵越低，分布越「尖锐」，预测越确定。GRN 的熵引导采样策略利用这一特性：复杂区域（高熵）分配更多细化步骤，简单区域（低熵）快速通过。AR 模型「天生具有复杂度感知能力，表现为可变似然」，而扩散模型缺乏此特性。

核心问题与关键洞察

三大核心问题

GRN 针对视觉生成领域三个相互关联的深层问题：

  1. 扩散模型计算效率低下：对所有样本分配相同的计算资源，无法根据内容复杂度自适应调整步数；
  2. 自回归模型离散化瓶颈：现有离散 tokenizer 重建质量显著低于连续表示，且严格因果预测机制导致误差累积无法修正；
  3. 缺乏全局修正机制：传统 AR 模型一旦生成高置信度 token 便不可更改，无法像人类画家一样 retroactively 修正错误。

GRN 的关键洞察

HBQ 近乎无损离散压缩

通过层级二叉量化，量化误差上限以指数衰减（误差 < 1/2^j），在不增加 latent 通道数的前提下，首次使离散 tokenizer 达到与连续 tokenizer 同等的重建质量（rFID 0.56）。这揭示了「分层粗到细」是连接离散与连续表示的桥梁。

全局细化 = 非因果自校正

GRN 突破了传统 AR 模型的严格因果性约束，允许模型像人类画家一样「回头修改」已生成的 token。与 Infinity 的随机翻转比特（假设错误率<30%）不同，GRN 的全局细化是系统性的、不受错误分布限制的修正能力。

熵作为计算复杂度的代理

利用 token 分布的熵动态决定采样步数，实现「简单样本少步、复杂样本多步」的自适应计算分配，无需额外网络或强化学习训练。这是 AR 模型可变似然特性的显式工程化实现。

HBQ：层次化二值量化

Hierarchical Binary Quantization（HBQ） 是 GRN 的视觉 tokenizer 核心，负责将 VAE 编码后的连续特征映射到离散二进制表示，通过多轮粗到细的量化实现理论上的近无损压缩。

算法流程

  1. 预处理：在 VAE 编码器后添加 tanh(·) 激活函数，将特征表示 F 从无界范围映射到闭区间 (−1, +1)。
  2. 多轮二值量化：每个特征元素进行 M 轮二进制量化，基于二叉树桶结构。第 i 轮的量化中心 c_i 和量化标签 q_i 由以下公式定义： 
    c_i = Σ_{j=1}^{i-1} δ[q_j] / 2^j

q_i = 0  if F ≤ c_i
q_i = 1  if F > c_i
    其中 δ(·) 为 delta 函数：当 q_i = 0 时输出 −1，否则输出 +1。
  3. 误差特性：第 j 轮的量化误差上限为 1/2^j，呈指数衰减。因此随着轮数增加，重建误差迅速收敛，理论上可实现无损量化。
  4. 重建：量化特征 F̂ 通过二进制标签的加权和重建： 
    F̂ = Σ_{j=1}^{M} δ[q_j] · 2^{−j}
  5. 训练：使用直通估计器（STE）将梯度回传至编码器。损失函数为重建损失、LPIPS 感知损失和 PatchGAN 判别器 GAN 损失的加权和。
关键特性：HBQ 不增加 latent 通道数即可实现近无损重建；粗到细表示不同频率信息；在相同 latent 维度下，离散 tokenizer 首次达到与连续 tokenizer 相当的性能。

全局细化机制：突破 AR 因果枷锁

全局细化机制是 GRN 的核心生成创新，允许模型在生成过程中像人类画家一样逐步完善和修正已生成的 token，突破传统 AR 模型的严格因果性限制。

问题背景

传统 AR 模型（包括逐 token 或逐 scale 的模型）严格遵循因果预测，无法回溯修改已生成的 token，导致多步生成中误差累积。即使 MaskGIT 等并行解码模型，高置信度 token 一旦生成就不可修改。论文将其类比为人类画家作画——不是一次性完成，而是通过多轮迭代逐步完善作品，随时修正之前的不当之处。

核心思想

GRN 在 HBQ 的 latent 空间上执行全局细化，模型可以在后续步骤中回溯性地（retroactively）修改之前生成的 token，实现整体性的错误修正。这与扩散模型的迭代去噪有本质区别：扩散模型在固定维度的连续空间中操作，而 GRN 在离散 token 空间中进行结构化修正，保留了 AR 模型的似然建模优势。

训练方式

每轮迭代从均匀分布随机采样随机 token Y_rand 和选择比例 l_t，构造输入 F_t（部分真实 token + 部分随机 token）。损失函数为标准交叉熵（CE）损失，预测 ground-truth token：

L = −E[ (1/N) Σ_{i=0}^{N} log p(y_i | F_t, cond) ]

条件注入方面，Class-to-Image（C2I）采用 in-context class conditioning，Text-to-Image（T2I）采用 in-context self-attention（非 adaln-zero）。

模型变体

  • GRN_ind：将 M 维二进制编码为 INT 标量（K=2^M 类）
  • GRN_bit：直接预测二进制位（K=2 类）

实验表明预测索引（indices）略优于预测比特（bits）：GRNind-B FID 3.56 vs GRNbit-B FID 3.63；GRNbit-L FID 2.47 优于 GRNind-L FID 2.64。

熵引导采样：复杂度感知的自适应步长

熵引导采样（Entropy-Guided Sampling） 是一种复杂度感知的自适应步数生成策略，根据 token 的不确定性（熵）动态分配计算资源，实现「简单样本快速生成，复杂样本精细生成」。

核心思想

不同样本的生成复杂度不同，统一分配计算步骤是低效的。通过监控 token 的熵值，模型可以判断哪些区域需要更多细化步骤：

  • 高熵 token（不确定性高）：分布「平坦」，预测困难，分配更多细化步骤，确保生成质量；
  • 低熵 token（置信度高）：分布「尖锐」，预测确定，快速通过，减少不必要的计算。

熵的计算

推理阶段使用熵引导调度函数动态控制选择比例 l_t，实现自适应步数生成。熵计算如下：

H(Y_t) = (1/N) · (1/log₂ K) Σ_{i=0}^{N} Σ_{j=0}^{K} −p(y_{(i,j)} | F_{t−1}, cond) · log₂ p(y_{(i,j)} | F_{t−1}, cond)

其中 N 为 token 数量，K 为类别数。熵值经过归一化后（除以 log₂ K），取值范围在 0 到 1 之间，便于统一阈值控制。

与扩散模型的本质区别：扩散模型通过 MSE 优化，缺乏显式似然估计，只能使用固定步数。GRN 通过 token 级似然估计实现真正的自适应计算分配——这是 AR 范式相对于扩散范式的结构性优势。

关键实验结论

  1. HBQ 解决了离散 tokenizer 的重建瓶颈，rFID 0.56 超越现有连续和离散方法；
  2. GRN 在 ImageNet C2I、T2I、T2V 三个任务上均达到或超越同等规模 SOTA；
  3. Global Refinement 是核心创新，将无修正基线的 FID 从 185.62 降至 18.13；
  4. Complexity-Aware Sampling 使多数样本 (62.7%) 可用 <50 步完成，提升推理效率；
  5. 预测索引（indices）策略略优于预测原始比特（bits），但差距不大，说明 HBQ 的二进制结构对两种策略都友好。
训练细节与实现

Visual Tokenizer 训练

GRN 采用 Wan 2.1 的 3D causal VAE 作为架构基础，统一处理图像和视频。编码器后接 tanh 激活，将特征映射到 (−1, +1) 区间，再通过 HBQ 量化为离散二进制标签 {q_1, ..., q_M}。梯度回传使用 Straight-Through Estimator (STE)。

损失函数为加权组合：

  • 图像专用 tokenizer：λ_recons=1.0, λ_LPIPS=1.0, λ_GAN=0.3
  • 联合图像-视频 tokenizer：λ_recons=1.0, λ_LPIPS=0.2, λ_GAN=0.005

图像 tokenizer 在 OpenImages 上训练；联合 tokenizer 在公开图像+视频数据集组合上训练。

评估指标说明

  • rFID：重建 FID，衡量 tokenizer 重建质量
  • gFID / FID：生成 FID，衡量生成图像与真实分布的相似度
  • IS：Inception Score，衡量生成图像的多样性和质量
  • LPIPS：感知相似度，基于深度特征的感知距离
  • PSNR：峰值信噪比，像素级重建精度
  • GenEval Overall：T2I 综合评测，覆盖多种生成能力维度
  • VBench Overall：T2V 综合评测，覆盖视频质量、时序一致性等
结论与未来展望

论文主要结论

  1. GRN 通过全局细化机制和复杂度感知生成，在视觉合成任务上实现了 SOTA 性能。ImageNet 256×256 上 rFID 0.56、gFID 1.81，显著优于现有扩散模型和自回归模型；
  2. HBQ 首次使离散视觉 tokenizer 达到与连续 tokenizer 相当的重建质量，量化误差理论上随轮次指数衰减，实现了近无损压缩；
  3. 复杂度感知生成通过熵引导采样动态分配计算资源，简单样本步数少、复杂样本步数多，不牺牲视觉质量；
  4. GRN 具备良好的任务泛化性和可扩展性，成功扩展到 1024×1024 T2I 和 480p T2V 任务。

对未来研究的启示

离散表示的复兴：HBQ 证明离散 tokenizer 不必劣于连续表示，未来可能推动更多基于离散潜空间的生成模型研究，而非一味追求连续扩散。
自适应计算成为标配：复杂度感知生成提示未来视觉模型应像 LLM 一样具备「思考深度」的自适应能力，而非固定步数推理。
全局细化机制的泛化：GRN 的「绘画式」全局修正可扩展到其他序列生成任务（如音频、3D），突破自回归的因果限制。
效率与质量的重新权衡：GRN 在同等规模下优于扩散模型，可能引发对扩散范式主导地位的重新评估，特别是在资源受限场景。
