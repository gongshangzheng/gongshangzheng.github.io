Hub Template
Hub 页结构模板

Hub 页是课程/专题的中枢导航页。以下 frontmatter 和结构可直接复制使用：

---
title: "专题名｜枢纽页"
description: "..."
categories: [分类]
subcategory: "子分类"
tags: [tag1, tag2]
aliases: [子分类]        # 让分类/子分类页自动指向此文章
mathjax: true
hero_title: "专题名"
hero_sub: "枢纽页 · 副标题"
hero_tagline: "一句话描述"
page_style: |
  :root { --hero-grad: linear-gradient(135deg, #0a1628, #0d2240, #0f3060); --accent: #3b82f6; --accent-gold: #f59e0b; }
  @media (prefers-color: dark) { :root { --hero-grad: linear-gradient(135deg, #0a1628, #0d2240, #0f3060); --accent: #60a5fa; --accent-gold: #fbbf24; } }
---

正文结构：

  • .stats — 数字统计条
  • .ch — 概述章节
  • .hub-section + .period-grid — 知识地图卡片网格（样式由 hub.css 提供，无需在 page_style 中重复）
  • {{< details >}} + {{< sortable-list "tags/xxx" >}} — 可折叠的文章列表

关键 shortcut：

  • aliases: [子分类名] — 让 categories/分类/子分类/index.html 直接渲染此文章，而非生成 sortable list
  • hub.css 已提供 .hub-section、.period-grid、.period-card、.topic-list 样式，page_style 只需写配色变量
