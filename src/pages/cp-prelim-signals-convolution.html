通信原理里反复出现的“信号工具箱”

通信原理真正开始讨论调制、滤波、抽样、匹配接收之前，默认你已经熟悉一套信号与系统语言：冲激、阶跃、矩形窗、抽样函数、卷积积分、卷积和。它们不是孤立的数学符号，而是后续所有波形分析的基础积木。

Part 1
一、常用模拟信息及性质

这里的“模拟信息”更准确地说是连续时间常用信号。它们用于描述开关、脉冲、取样、窗口截断和理想化瞬时作用。

1.1 单位冲激信号 $\delta(t)$

单位冲激不是普通函数，而是广义函数。可以把它理解成“宽度无限窄、幅度无限高、面积等于 1”的理想脉冲。

$$\delta(t)=0\quad(t\ne 0),\qquad \int_{-\infty}^{\infty}\delta(t)\,dt=1$$
筛选性质：冲激最重要的作用是从积分中“挑出”某一点的函数值。
$$\int_{-\infty}^{\infty}x(t)\delta(t-t_0)\,dt=x(t_0)$$

移位冲激 $\delta(t-t_0)$ 位于 $t=t_0$；加权冲激 $A\delta(t-t_0)$ 的面积是 $A$。

1.2 单位阶跃信号 $u(t)$

单位阶跃表示“从某一时刻开始打开”。它常用于描述系统输入突然接通、信号从某时刻开始存在。

$$u(t)=\begin{cases}0,&t<0\\1,&t>0\end{cases}$$

$u(0)$ 的取值在不同教材中可能取 $0$、$1$ 或 $\frac12$，一般不影响连续时间积分结果。

$$\frac{d}{dt}u(t)=\delta(t),\qquad \int_{-\infty}^{t}\delta(\tau)d\tau=u(t)$$ {{< jsxgraph title="单位阶跃信号 u(t)" height="300" >}} var b = JXG.JSXGraph.initBoard(el, { boundingbox: [-3, 1.3, 3, -0.3], showCopyright: false, showNavigation: false }); b.create('axis', [[0,0],[1,0]], {}); b.create('functiongraph', [function(x){ return 0; }, -3, 0], { strokeColor:'#2563eb', strokeWidth:2.2 }); b.create('functiongraph', [function(x){ return 1; }, 0, 3], { strokeColor:'#2563eb', strokeWidth:2.2 }); b.create('text', [2.6, -0.15, 't'], { fontSize:13, color:'#888', fontStyle:'italic' }); b.create('text', [0.15, 1, 'u(t)'], { fontSize:14, color:'#2563eb', anchorX:'left' }); {{< /jsxgraph >}}

1.3 符号函数 $\operatorname{sgn}(t)$

符号函数只保留正负号信息：

$$\operatorname{sgn}(t)=\begin{cases}-1,&t<0\\0,&t=0\\1,&t>0\end{cases}$$

它和单位阶跃函数之间有简单关系：

$$\operatorname{sgn}(t)=2u(t)-1$$ {{< jsxgraph title="符号函数 sgn(t)" height="300" >}} var b = JXG.JSXGraph.initBoard(el, { boundingbox: [-3, 1.4, 3, -1.4], showCopyright: false, showNavigation: false }); b.create('axis', [[0,0],[1,0]], {}); b.create('functiongraph', [function(x){ return -1; }, -3, 0], { strokeColor:'#2563eb', strokeWidth:2.2 }); b.create('functiongraph', [function(x){ return 1; }, 0, 3], { strokeColor:'#2563eb', strokeWidth:2.2 }); b.create('text', [2.6, -0.15, 't'], { fontSize:13, color:'#888', fontStyle:'italic' }); b.create('text', [0.15, 1, 'sgn(t)'], { fontSize:14, color:'#2563eb', anchorX:'left' }); {{< /jsxgraph >}}

1.5 矩形窗函数 $\operatorname{rect}(t/T)$

矩形窗表示有限持续时间的开关。常见定义为：

$$\operatorname{rect}\left(\frac{t}{T}\right)=\begin{cases}1,&|t|<\frac{T}{2}\\0,&|t|>\frac{T}{2}\end{cases}$$

边界点 $|t|=T/2$ 的取值常取 $\frac12$，用于保持傅里叶变换对称性。

1.6 三角波函数 $\operatorname{tri}(t/T)$

三角函数可以看作两个等宽矩形窗卷积后的形状。常见定义为：

$$\operatorname{tri}(t)=\begin{cases}1-|t|,&|t|<1\\0,&|t|\ge 1\end{cases}$$

它的峰值在原点，向两侧线性下降。很多“两个矩形脉冲卷积”的结果都会出现三角形。

Part 2
二、连续信号卷积积分

连续时间 LTI 系统的零状态响应由输入 $x(t)$ 和冲激响应 $h(t)$ 的卷积决定：

$$y(t)=x(t)*h(t)=\int_{-\infty}^{\infty}x(\tau)h(t-\tau)d\tau$$

这条公式可以拆成一个图解过程：

  1. 换元：把 $x(t)$、$h(t)$ 改写成 $x(\tau)$、$h(\tau)$。
  2. 翻转：把 $h(\tau)$ 变成 $h(-\tau)$。
  3. 平移：把 $h(-\tau)$ 平移为 $h(t-\tau)$。$t$ 是观察时刻，不是积分变量。
  4. 相乘：找 $x(\tau)$ 与 $h(t-\tau)$ 的重叠区间。
  5. 积分：在重叠区间内计算面积 $\int x(\tau)h(t-\tau)d\tau$。
直觉：卷积积分不是把两个函数简单相乘，而是让一个函数作为“滑动模板”扫过另一个函数，每一个 $t$ 都得到一次重叠面积。

2.1 常用性质

性质公式意义
交换律$x(t)*h(t)=h(t)*x(t)$谁翻转平移都可以，结果一样
结合律$x*(h_1*h_2)=(x*h_1)*h_2$级联系统可合并冲激响应
分配律$x*(h_1+h_2)=x*h_1+x*h_2$并联系统响应可相加
冲激单位元$x(t)*\delta(t)=x(t)$冲激不改变信号
移位冲激$x(t)*\delta(t-t_0)=x(t-t_0)$卷积移位冲激只产生延迟
微分性质$\dfrac{d}{dt}[x*h]=x'*h=x*h'$可把微分转移到任一因子

2.2 例：两个单位宽矩形窗的卷积

设 $x(t)=\operatorname{rect}(t)$，$h(t)=\operatorname{rect}(t)$，两个宽度为 1 的矩形窗卷积。随着一个矩形滑过另一个矩形，重叠长度先线性增加再线性减少，因此结果是三角形：

$$\operatorname{rect}(t)*\operatorname{rect}(t)=\operatorname{tri}(t)$$

这个例子说明：卷积结果的形状，本质上来自两个信号重叠面积如何随 $t$ 改变。

三、常用序列信号及性质

离散时间信号写作 $x[n]$，自变量 $n$ 只取整数。连续时间中的冲激、阶跃、矩形窗都有离散版本。

3.1 单位样值序列 $\delta[n]$

$$\delta[n]=u[n]-u[n-1]$$ $$u[n]=\sum_{k=-\infty}^{n}\delta[k]$$

Part 4
四、离散卷积和

离散时间 LTI 系统的输出由卷积和给出：

$$y[n]=x[n]*h[n]=\sum_{k=-\infty}^{\infty}x[k]h[n-k]$$

它和连续卷积积分完全平行：连续情况把重叠部分“积分成面积”，离散情况把重叠样值“相乘后求和”。

4.1 图解法：翻转、平移、相乘、求和

  1. 换元：写出 $x[k]$ 和 $h[k]$。
  2. 翻转：把 $h[k]$ 变成 $h[-k]$。
  3. 平移：得到 $h[n-k]$。
  4. 找重叠：判断哪些 $k$ 同时让 $x[k]$ 和 $h[n-k]$ 非零。
  5. 相乘求和：把重叠点上的乘积加起来。
记忆：连续卷积是“重叠面积”，离散卷积是“重叠乘积和”。

4.2 表格滑动法

当两个有限长序列较短时，可以把其中一个序列翻转后在另一个序列上滑动，每滑动一个位置就把对齐项相乘求和。这个方法直观，但容易漏项；适合检查支持区间和边界。

4.3 竖式乘法法

如果把有限长序列看成多项式系数，离散卷积就等价于多项式乘法。设：

$$x[n]=\{1,2,1\},\qquad h[n]=\{1,1\}$$

对应多项式：

$$X(z)=1+2z+z^2,\qquad H(z)=1+z$$

竖式乘法：

      1 + 2z + z²
×         1 + z
----------------
      1 + 2z + z²
  z + 2z² + z³
----------------
      1 + 3z + 3z² + z³

所以卷积结果为：

$$y[n]=\{1,3,3,1\}$$

4.4 直接求和核对

按定义：

$$y[n]=\sum_k x[k]h[n-k]$$
$n$非零乘积$y[n]$
0$x[0]h[0]=1\cdot1$1
1$x[0]h[1]+x[1]h[0]=1\cdot1+2\cdot1$3
2$x[1]h[1]+x[2]h[0]=2\cdot1+1\cdot1$3
3$x[2]h[1]=1\cdot1$1

4.5 长度规律

若 $x[n]$ 长度为 $L_x$，$h[n]$ 长度为 $L_h$，且都从 0 开始，则线性卷积长度为：

$$L_y=L_x+L_h-1$$

这个规律可以用来快速检查是否漏算了首尾项。
