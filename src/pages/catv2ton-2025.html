任务背景

虚拟换装（Virtual Try-On, VTON）旨在将服装图像"穿"到目标人物身上，生成穿着指定服装的人物图像或视频。根据输入输出形式可分为两类：

  • 图像换装（Image-based VTON）：输入服装图像 + 人物图像 → 换装后的人物图像
  • 视频换装（Video-based VTON）：输入服装图像 + 人物视频 → 换装后的人物视频，要求时序一致性

传统方法往往需要两套分离的模型——图像换装一套，视频换装一套。CatV2TON 的核心贡献是：用单个扩散 Transformer 同时搞定两种任务，不需要任何额外的 garment encoder、reference net 或 warping 模块。

上图展示了 CatV2TON 在图像换装和视频换装上的效果——服装在人物身上保持高度一致性，同时人物姿态和场景背景得到良好保留。

核心方法

1. 时间维度拼接（Temporal Concatenation）

CatV2TON 的设计哲学是"拼接即一切"——不引入额外的 garment encoder，只将服装和人物的潜空间表示在通道维度上拼接后喂入 DiT。

对于视频换装，服装图像被复制 T 帧，与人物视频的每一帧在通道维度拼接。这样模型在一个 forward pass 中同时看见服装特征和人物姿态，实现"所见即所得"的条件注入。

服装图像 × T帧
+
人物视频 T×H×W
+
DensePose T×H×W
→
通道 concat
→
HunyuanDiT

2. Latent 空间条件注入

Pipeline 的 denoising 方法中，多个 latent 按通道拼接后送入 DiT：

masked_latents_cat  = concat([conditioned_latents, masked_latents], dim=2)
mask_latents_cat   = concat([conditioning_mask_latents, mask_latents], dim=2)
inpaint_latents    = concat([mask_latents_cat, masked_latents_cat], dim=1)
# inpaint_latents shape: (b, 4c, f, h, w)
  • conditioned_latents：目标人物图像（不含服装）的 VAE encoding
  • masked_latents：服装区域被遮盖的人物 VAE encoding
  • pose_latents：DensePose 姿态估计的 VAE encoding

3. 模型架构

骨干网络是来自 EasyAnimateV4-XL-2-InP 的 HunyuanTransformer3DModel，各组件处理方式如下：

组件处理方式
VAEAutoencoderKLMagvit，全部冻结
DiT BackboneHunyuanTransformer3DModel，全部冻结
PoseNet独立编码器，单独训练
可训练参数attn1（自注意力）+ PoseNet，约占 backbone 20%

init_transformer3d_model 中明确禁用了以下模块（这些是 EasyAnimate 原有但 CatV2TON 不需要的功能）：

for block in transformer3d.blocks:
    block.attn_temporal = None   # 时间注意力
    block.attn_norm = None        # CLIP guided attention
    block.attn_clip = None        # CLIP attention
    block.norm_clip = None        # CLIP norm
    block.gate_clip = None        # CLIP gate
    block.norm_clip_out = None    # CLIP norm out
    block.norm2 = None            # FFN 前的第二个 norm
    block.attn2 = None            # 第二个注意力（Cross attention）

4. JointAttnProcessor2_0：SD3 风格的注意力

这是最值得关注的实现细节。JointAttnProcessor2_0 采用 SD3（Stable Diffusion 3）的 MM-DiT 思路，核心是 Q/K/V 三元组的 concat，而非传统 cross-attention：

# Q/K/V 均由两部分 concat 而成
query = concat([to_q(hidden_states), add_q_proj(encoder_hidden_states)], dim=1)
key   = concat([to_k(hidden_states), add_k_proj(encoder_hidden_states)], dim=1)
value = concat([to_v(hidden_states), add_v_proj(encoder_hidden_states)], dim=1)
hidden_states = F.scaled_dot_product_attention(query, key, value)
# SDPA 一次完成自注意力 + 条件注入，再沿 seq 维度 split 回两部分
hidden_states, encoder_hidden_states = split(..., dim=1)

传统 cross-attention 的 Q 来自 hidden_states，K/V 来自 encoder_hidden_states；而 JointAttnProcessor 的 Q,K,V 均由两部分 concat，使得自注意力和条件注入在同一个 SDPA 操作中完成。这种设计的优势是条件注入更直接、无需独立的 cross-attention 路径。

5. PoseNet：姿态编码器

PoseNet 与 DiT backbone 共享相同的 PatchEmbed、TimeEmbedding 架构，但使用独立的 HunyuanDiTBlock 处理。forward 时，pose image（由 DensePose 灰度图经 cv2.COLORMAP_VIRIDIS 转为 RGB）经 VAE encode 后作为 PoseNet 输入，其输出 pose_emb 作为 DiT 的条件传入。

6. Overlapping Clip Inference + AdaCN

长视频推理时采用重叠片段策略：

  • 将长视频切分为长度为 slice_frames（24帧原始分辨率，6帧 latent）的片段
  • 相邻片段重叠 pre_frames（8帧原始分辨率，2帧 latent）
  • 当前片段的前 pre_frames 帧用上一片段的输出填充（sequential frame guidance）
  • 在重叠区域使用 Adaptive Clip Normalization (AdaCN) 对齐相邻片段的分布

AdaCN 的核心公式——通过匹配均值/方差来消除片段间的颜色不匹配和闪烁：

def adacn_params(content_feat, style_feat, eps=1e-5):
    content_mean, content_std = content_feat.mean/var(dim=(2,3,4))
    style_mean, style_std     = style_feat.mean/var(dim=(2,3,4))
    return content_mean, content_std, style_mean, style_std

# 在重叠区域和后续区域应用归一化调整：
result_normed = (content - content_mean) / content_std * style_std + style_mean
# content_feat：当前片段重叠区输出；style_feat：上一片段对应位置输出

7. AutoMasker：服装无关掩码生成

AutoMasker 是整个预处理管线的核心，负责生成"服装无关掩码"（clothing-agnostic mask）。它整合了三路解析器：

解析器类别数用途
DensePose25人体表面分割（躯干、手脚等）
SCHP-ATR18ATR 人体解析
SCHP-LIP20LIP 人体解析

掩码合成逻辑分为三个优先级：

  • STRONG PROTECT：面部、手部、脚部 → 绝对不遮盖
  • WEAK PROTECT：头发、配件、其他衣物 → 尽量保留
  • MASK AREA：根据 part 参数决定替换哪类衣物

smooth_video_mask 对视频掩码进行 3D 平滑——2D 平均池化（空间）+ 1D 平均池化（时序）→ 重新二值化，消除逐帧处理引入的时序抖动。

8. ViViD-S 数据集

论文构建的高质量视频换装数据集，基于 ViViD 原始数据集做了两件事：

  • Back-facing 帧过滤：训练专用朝向识别模型，剔除人物背对照的帧（这类帧与正面服装图像不匹配，引入伪影并拖慢收敛）
  • 3D Mask Smoothing：应用空间+时序平均池化到 clothing-agnostic masks，然后重新二值化

训练策略

  • 混合数据集：图像数据集（VITON-HD, DressCode）+ 视频数据集（ViViD-S, VVT）
  • 图像训练：单帧（f=1）；视频训练：16帧片段
  • 80% backbone 参数冻结 → 仅训练 attention blocks 和 PoseNet
  • 优化器：AdamW，lr=1e-4
  • 推理步数：图像 50 steps，视频 15 steps
  • CFG：图像 1.0（无 CFG），视频 2.5

V2TONPipeline 的核心调用路径：

# 图像换装
image_try_on(source_image, source_mask, conditioned_image, pose_image)
  → prepare_image()              # 图像 → tensor [-1,1]
  → F.max_pool2d(source_mask)   # 掩码空间池化（核=11）
  → _slice_vae()                # VAE encode
  → denoising()                  # 50步 DDPM 去噪
    → step() × N
      → posenet(pose_latents)   # pose embedding
      → transformer3d(... pose_emb=...)  # DiT forward（含 CFG）
  → decode_latents()            # VAE decode
  → output PIL Images

# 视频换装
video_try_on(source_video, mask_video, condition_image, pose_video)
  → _slice_vae()                # VAE encode 全视频
  → while 片段循环（slice=6帧 latent，pre=2帧 latent）:
      → denoising()             # 单片段 15步去噪
      → adacn_params()          # 重叠区域 AdaCN 统计
      → 片段拼接（用 AdaCN 对齐重叠区）
  → decode_latents()            # VAE decode + smooth_output
  → output video tensor (b,t,h,w,c)

技术定位与贡献总结

维度CatV2TON 的做法传统方案（如 OOTDiffusion）
图像/视频统一时间维度拼接 + 混合训练分别训练或视频模型降级处理图像
条件注入latent 通道拼接专用 garment encoder 或 warping
参数效率80% 冻结，仅 attn1 + PoseNet额外编码器全部参与训练
长视频一致性Overlapping clip + AdaCN无或简单拼接
预处理掩码DensePose + SCHP 双解析 + 3D 平滑单一解析器

核心结论：对于 VTON 任务，简单的通道拼接 + 混合训练数据即可实现 SOTA，无需复杂的 garment warping 或额外的 garment encoder。这与 OOTDiffusion、IDM-VTON 等依赖双流网络的方法形成鲜明对比。
