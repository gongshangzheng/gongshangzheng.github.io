# Welcome

This is a **minimalist blog** built with pure HTML, CSS, and JavaScript. No frameworks, no build dependencies — just static files.

## Features

- 🌓 **Dark/Light Mode** — Toggle with the button in the header
- 🎵 **Music Support** — Embed audio players in your posts
- 📱 **Responsive** — Works on all devices
- ⚡ **Fast** — No JavaScript frameworks, pure vanilla

## Quick Start

1. Write your content in Markdown or HTML
2. Run `node build.js` to generate the static site
3. Deploy `public/` to GitHub Pages
