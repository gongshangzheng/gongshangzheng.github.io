第31章
开放经济的基本概念

章节定位

  • 位置：第11篇「开放经济的宏观经济学」第1章
  • 核心问题：贸易余额和汇率如何决定？开放经济中储蓄与投资的关系？
  • 前接：第30章「货币增长与通货膨胀」——从封闭经济的货币分析过渡到开放经济
  • 后接：第32章「开放经济的宏观经济理论」
核心概念
贸易余额与汇率
概念定义
贸易余额（净出口 NX）出口 - 进口
贸易盈余NX > 0（出口 > 进口）
贸易赤字NX < 0（进口 > 出口）

资本净流出（NCO）

本国居民购买外国资产减外国人购买本国资产 = 国民储蓄 - 国内投资：$NCO = S - I$。

汇率类型定义公式
名义汇率一国货币换另一国货币的比率$e$ = 外国货币/本国货币
真实汇率一国物品换另一国物品的比率$q = e \times \frac{P}{P^*}$
真实汇率才是影响国际贸易的真正变量。
关键公式

开放经济国民收入恒等式

$$Y = C + I + G + NX$$

贸易余额与储蓄-投资的恒等关系

$$NX = S - I = NCO$$

S > I → 资本净流出 → NX > 0（贸易盈余）

S < I → 资本净流入 → NX < 0（贸易赤字）

真实汇率

$$\text{真实汇率} = \text{名义汇率} \times \frac{\text{国内物价}}{\text{国外物价}}$$

购买力平价（PPP）

一价定律的延伸：同一种商品在所有国家应以共同货币计价的相同价格出售。

长期推论：$e = P/P^*$，通胀较高的国家货币会贬值。

局限：不可贸易商品、运输成本、贸易壁垒打破 PPP。

关键区分
含义
贸易盈余S > I → 资本净流出 → 为世界其他地区融资
贸易赤字S < I → 资本净流入 → 从世界其他地区借款
贸易赤字未必是坏事：它意味着资本净流入，可能是外国对本国投资前景信心充足的信号。关键在于赤字资金的用途——用于生产性投资还是消费。

要点总结

  • 开放经济中，净出口 = 国民储蓄 - 国内投资
  • 真实汇率是国际贸易的真正价格
  • 购买力平价解释长期汇率趋势，短期汇率受利率、预期影响
第32章
开放经济的宏观经济理论

章节定位

  • 位置：第11篇「开放经济的宏观经济学」第2章
  • 核心问题：可贷资金市场与外汇市场如何同时均衡？政策如何影响贸易余额和汇率？
  • 前接：第31章「开放经济的基本概念」
  • 后接：第33章「总需求与总供给」（第12篇·短期经济波动）
核心模型
开放经济的双市场均衡
市场供求均衡价格
可贷资金市场供给=S，需求=I+NCO(r)真实利率
外汇市场供给=NCO(r)，需求=NX(e)真实汇率
NCO 的双重角色：在可贷资金市场是需求的一部分（购买国外资产需要借贷），在外汇市场是供给的来源（出售本币换取外币）。

均衡条件

$$S = I + NCO(r) \quad \text{且} \quad NCO(r) = NX(e)$$

→ $$S - I = NX$$

NCO 与真实利率负相关：利率↑ → 本国资产更有吸引力 → NCO↓

政策分析
冲击机制结果
政府预算赤字S↓ → 可贷资金供给↓ → 利率↑ → NCO↓ → 外汇供给↓ → 汇率↑NX↓（双赤字效应）
贸易政策（关税）进口↓ → NX↑ → 但汇率升值抵消NX不变，仅改变贸易结构
资本外逃信心危机 → 汇率急贬 → 本币崩溃自我实现的危机
双赤字效应：预算赤字和贸易赤字往往同时出现。政府借款 → 利率↑ → 资本流入 → 汇率升值 → 出口竞争力下降。
贸易政策悖论：关税不能改变贸易余额（NX），只能改变贸易结构。这是因为汇率会通过升值来抵消贸易政策的直接影响。
关键区分
购买力平价本章模型
外汇需求曲线水平线向右下方倾斜
真实汇率固定可变
关系PPP 是本章模型的特例（当净出口对汇率变动极敏感时）

要点总结

  • 开放经济模型同时考察可贷资金和外汇市场，通过 NCO 联系
  • 均衡时：S = I + NCO，NCO = NX
  • 预算赤字 → 利率↑ → 汇率升值 → 贸易赤字恶化（双赤字）
  • 贸易政策不能改变贸易余额
  • 资本外逃导致本币急剧贬值
