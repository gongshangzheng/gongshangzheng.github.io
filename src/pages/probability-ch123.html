概率论与统计学的学习遵循一条清晰的主线：

  • 第 1 章建立"为什么要用概率思维研究经济问题"的动机
  • 第 2 章从具体随机试验出发，给出概率空间的公理化定义（$S, \mathcal{B}, P$ 三元组）
  • 第 3 章将样本空间映射到实数轴，引入随机变量和累积分布函数

课程特色

所有概念引入都以经济学应用为驱动力：恩格尔曲线、菲利普斯曲线、波动聚类、风险中性定价——概率论不是抽象的数学游戏，而是经济实证研究的工具。

Part 1 · 背景问题
为什么经济学需要概率思维

现代经济学研究建立在两个基本公理之上：

  • 经济系统可视为服从某个概率法则的"随机试验"
  • 任何经济现象（常以数据形式测度）都是上述随机试验的一个结果

这两个公理将经验观察与数学概率模型连接起来——统计学因此成为实证研究的核心工具。

统计学的基本流程：数据收集 → 归纳经验典型特征事实 → 建立理论 → 经验验证 → 预测与政策评估。

代表性实例：恩格尔曲线（食品支出比例随收入变化）、菲利普斯曲线（通胀率与失业率负相关）、波动聚类（金融收益率时序特征）——这些都展示了"同类事件在相似条件下重复出现"的统计规律性。
Part 2 · 概念定义
随机试验、样本空间与事件

随机试验（Random Experiment）在相同条件下可重复进行，结果至少两种且事先无法确定哪个结果实现。这是一个将"不确定性"形式化的起点。

课件对应：{{< docpage "pdf/概率论/02.概率论与统计学.pdf" page=3 title="随机试验定义（第2章 第3页）" >}} · {{< docpage "pdf/概率论/02.概率论与统计学.pdf" page=9 title="样本空间举例（第2章 第9页）" >}}

随机试验的两要素

  1. 所有可能结果的集合（即样本空间 $S$）
  2. 每个结果发生的可能性（概率函数 $P$）

样本空间 $S$ 是所有基本结果的集合，可以是有限的（抛硬币 $\{H,T\}$）、可数无限的（$\{1,2,3,\ldots\}$），或不可数的（气温区间 $[t_0, t_1]$）。

事件是样本空间 $S$ 中具有共同特征的基本结果所组成的集合。数学上事件等同于集合——因此"集合"与"事件"两词可以互换。

基本关系：基本结果 $\subseteq$ 事件 $\subseteq$ 样本空间。

集合运算

事件之间的运算与集合运算完全对应：

运算符号含义
交集$A \cap B$$A$ 和 $B$ 同时发生
并集$A \cup B$$A$ 或 $B$（或两者）发生
补集$A^c$不在 $A$ 中的基本结果
差$A - B = A \cap B^c$在 $A$ 中但不在 $B$ 中

互斥（$A \cap B = \varnothing$）且完全穷尽（$\cup_i A_i = S$）的事件集构成样本空间的一个分割（partition）——可类比为正交基的完备集。

德摩根律：$(A \cup B)^c = A^c \cap B^c$，$(A \cap B)^c = A^c \cup B^c$。这是概率计算中经常使用的恒等变换。
Part 3 · 概率空间的公理化
$\sigma$ 代数、概率函数与概率空间

仅用集合运算描述事件还不够——我们需要对事件赋予概率，这要求事件的集合必须构成一个适合概率运算的结构：$\sigma$ 代数。

$\sigma$ 代数（$\sigma$ 域）$\mathcal{B}$

$\mathcal{B}$ 是样本空间 $S$ 中满足以下条件的子集（事件）的集合：

  1. $\varnothing \in \mathcal{B}$（空集封闭）
  2. 若 $A \in \mathcal{B}$，则 $A^c \in \mathcal{B}$（对补集封闭）
  3. 若 $A_1, A_2, \cdots \in \mathcal{B}$，则 $\cup_{i=1}^{\infty} A_i \in \mathcal{B}$（对可数并集封闭）

$(S, \mathcal{B})$ 构成可测空间，$\mathcal{B}$ 是概率函数的定义域。

概率函数 $P: \mathcal{B} \to [0,1]$ 是满足以下条件的映射：

  1. $0 \leq P(A) \leq 1$（非负性）
  2. $P(S) = 1$（规范性）
  3. 若 $A_1, A_2, \cdots$ 互斥，则 $P(\cup_{i=1}^{\infty} A_i) = \sum_{i=1}^{\infty} P(A_i)$（可数可加性）

概率空间

概率空间是一个三元组合 $(S, \mathcal{B}, P)$：

  • $S$：随机试验的样本空间
  • $\mathcal{B}$：$S$ 的子集（事件）构成的 $\sigma$ 域
  • $P: \mathcal{B} \to [0,1]$：概率测度（概率函数）

这个三元组完整描述了随机试验的概率法则。

课件对应：{{< docpage "pdf/概率论/02.概率论与统计学.pdf" page=34 title="概率函数定义（第2章 第34页）" >}}
概率的解释

概率有两种主要的解释方式：

  • 相对频率解释：大量重复试验时结果发生的比例趋于概率（"相同条件"即 IID 假设）。气象局"降水概率 30%"的含义是：在相同天气条件下，长期记录中约 30% 的类似日子会下雨。
  • 主观概率解释：事件发生的主观可能性——用于难以重复的事件（如分析师预测股票、S&P 500 未来走向）。这引出了金融学中风险中性概率的概念。
基本概率法则
法则公式
补公式$P(A) = 1 - P(A^c)$
单调性$A \subseteq B \Rightarrow P(A) \leq P(B)$
包含-排除公式$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
次可加性（布尔不等式）$P(\cup_i A_i) \leq \sum_i P(A_i)$
Part 4 · 条件概率与贝叶斯定理
从条件概率到贝叶斯推断框架

经济事件往往相互关联。条件概率描述了如何利用一个事件的信息来预测另一个事件的概率。

条件概率定义

给定事件 $B$（$P(B) > 0$），事件 $A$ 的条件概率：

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

条件概率 $P(A \mid B)$ 描述的是 $B$ 已发生时对 $A$ 的预测关系，而非因果关系。因果关系需要经济理论来刻画。

条件概率本身也满足概率函数的所有公理——$(S \cap B, \mathcal{B} \cap B, P(\cdot \mid B))$ 构成一个完整的概率空间。

乘法法则

$$P(A \cap B) = P(A \mid B) P(B) = P(B \mid A) P(A)$$

对 $n$ 个事件：$P(\cap_{i=1}^n A_i) = P(A_1) \cdot P(A_2 \mid A_1) \cdot P(A_3 \mid A_1 \cap A_2) \cdots$

全概率公式

若 $A_1, \cdots, A_n$ 互斥且完全穷尽，$P(A_i) > 0$，则对任意事件 $B$：

$$P(B) = \sum_{i=1}^n P(B \mid A_i) P(A_i)$$

意义：将复杂事件 $B$ 的概率分解为若干互斥情形下的条件概率之和。

贝叶斯定理

$$P(A_i \mid B) = \frac{P(B \mid A_i) P(A_i)}{\sum_{j=1}^n P(B \mid A_j) P(A_j)}$$

其中 $P(A_i)$ 为先验概率（prior）——获得 $B$ 信息之前对原因的判断；$P(A_i \mid B)$ 为后验概率（posterior）——获得 $B$ 信息之后的修正判断。

例题：全概率公式与贝叶斯公式
全概率公式与贝叶斯公式典型例题

例 2：红蓝球的次品率

箱子中有 60% 的红球、40% 的蓝球。红球中有一半是次品，蓝球中有 10% 是次品。

（1）随机取一球，求取到次品的概率。

设 $A$ 为「取到红球」，$B$ 为「取到次品」。由全概率公式：

$$P(B) = P(B \mid A)P(A) + P(B \mid \bar{A})P(\bar{A}) = 0.5 \times 0.6 + 0.1 \times 0.4 = 0.34$$

（2）已知取到的是次品，求它是蓝球的概率。

由贝叶斯公式：

$$P(\bar{A} \mid B) = \frac{P(B \mid \bar{A})P(\bar{A})}{P(B)} = \frac{0.1 \times 0.4}{0.34} = \frac{2}{17}$$

例 3：三人考核的通过率

三名员工（猴博士、傻狍子、沙垃鸡）被选中的概率各为 $\frac{1}{3}$，通过率分别为 99%、3%、30%。

（1）随机选一人考核，求通过的概率。

设 $E_1, E_2, E_3$ 分别为选中三人，$T$ 为「通过」。由全概率公式：

$$P(T) = \sum_{i=1}^{3} P(T \mid E_i)P(E_i) = \frac{1}{3}(0.99 + 0.03 + 0.30) = 0.44$$

（2）已知某人通过了考核，求他是傻狍子的概率。

由贝叶斯公式：

$$P(E_2 \mid T) = \frac{P(T \mid E_2)P(E_2)}{P(T)} = \frac{\frac{1}{3} \times 0.03}{0.44} = \frac{1}{44}$$

注意：尽管三人的先验概率相同，但由于傻狍子通过率极低（3%），在已知通过的条件下，他是傻狍子的后验概率仅为 $\frac{1}{44}$——观察数据大幅修正了先验判断。

应用：汽车保费问题

高风险 $P(H)=0.25$，中风险 $P(M)=0.25$，低风险 $P(L)=0.5$。收到超速罚单条件下高风险的概率：

$$P(H \mid B) = \frac{0.25 \times 0.25}{0.25 \times 0.25 + 0.16 \times 0.25 + 0.10 \times 0.5} \approx 0.41$$

从先验 $0.25$ 更新到后验 $0.41$——罚单信息显著提高了高风险判断的概率，保险公司据此可以调整保费定价。

独立性

若 $P(A \cap B) = P(A)P(B)$，则 $A$ 与 $B$ 相互独立。等价条件：$P(A \mid B) = P(A)$——一个事件发生不影响另一个的概率。

注意：独立性与互斥是不同的概念。互斥事件（$A \cap B = \varnothing$）一定不独立，因为通常 $P(A) > 0, P(B) > 0$，而 $P(A)P(B) > 0 = P(A \cap B)$。
Part 5 · 随机变量
从样本空间到实数轴的映射

为什么需要随机变量？原始样本空间因随机试验而异，建立统一的概率理论需要将任意 $S$ 映射到通用空间——实数集 $\mathbb{R}$。

随机变量的定义

随机变量 $X(\cdot)$ 是从样本空间 $S$ 到实数集 $\mathbb{R}$ 的 $\mathcal{B}$可测映射（measurable function）。

对每个基本结果 $s \in S$，存在唯一的实数 $X(s)$ 与之对应。$X$ 可能取的所有实数值的集合构成新的样本空间 $\Omega$，称为 $X$ 的值域。

可测性的关键意义：确保 $\{s \in S: X(s) \leq a\} \in \mathcal{B}$，从而概率 $P(X \leq a)$ 有定义。若 $X(\cdot)$ 不可测，则存在某些实数子集无法定义概率。

从 $P(\cdot)$ 到 $P_X(\cdot)$

原样本空间 $S$ 上的概率函数 $P(\cdot)$ 诱导出新样本空间 $\Omega$ 上的概率函数 $P_X(\cdot)$：

$$P_X(A) = P(\{s \in S: X(s) \in A\}), \quad A \in \mathcal{B}_\Omega$$

可以验证，$P_X(\cdot)$ 也满足概率函数的三条公理——诱导概率函数也是合法的概率函数。

累积分布函数（CDF）

CDF 是刻画随机变量概率分布的核心工具：

$$F_X(x) = P(X \leq x), \quad x \in \mathbb{R}$$

课件对应：{{< docpage "pdf/概率论/03.随机变量和一元概率分布.pdf" page=3 title="随机变量定义（第3章 第3页）" >}} · {{< docpage "pdf/概率论/03.随机变量和一元概率分布.pdf" page=18 title="CDF 定义（第3章 第18页）" >}}
性质内容
规范性$\lim_{x \to -\infty} F_X(x) = 0$，$\lim_{x \to +\infty} F_X(x) = 1$
单调非递减$x_1 < x_2 \Rightarrow F_X(x_1) \leq F_X(x_2)$
右连续$\lim_{\delta \to 0^+} F_X(x+\delta) = F_X(x)$
概率计算$P(a < X \leq b) = F_X(b) - F_X(a)$

注意：$F_X(x)$ 不要求左连续——这是与离散型 CDF 的特性有关的一个细节（跳点左侧概率为零）。

Part 6 · 离散与连续随机变量
PMF 与 PDF：两种描述概率分布的语言
离散随机变量（DRV）

离散随机变量取有限个或可列个值，用概率质量函数（PMF）描述：

$$f_X(x) = P(X = x), \quad x \in \mathbb{R}$$

PMF 性质：

  • $0 \leq f_X(x) \leq 1$（非负性）
  • $\sum_{x \in \Omega} f_X(x) = 1$（归一性）
  • 从 PMF 可求 CDF：$F_X(x) = \sum_{y \leq x} f_X(y)$
连续随机变量（CRV）

连续随机变量的 CDF 是连续函数。其关键性质：

$$P(X = x) = 0, \quad \forall x \in \mathbb{R}$$

课件对应：{{< docpage "pdf/概率论/03.随机变量和一元概率分布.pdf" page=35 title="离散随机变量与 PMF（第3章 第35页）" >}} · {{< docpage "pdf/概率论/03.随机变量和一元概率分布.pdf" page=47 title="连续随机变量与 PDF（第3章 第47页）" >}}
直观理解：类比卫星飞越中国领空——飞越整个中国需 1 小时，飞越福建省需 2 分钟，飞越厦门大学经济楼的时间接近 0 秒。对连续随机变量，任何单点值的概率均为零。

由于 PMF 不再适用（$f_X(x)$ 不再是概率），引入概率密度函数（PDF）：

若 $F_X(x)$ 绝对连续，则存在 $f_X(x)$ 使 $F_X(x) = \int_{-\infty}^x f_X(y)\,dy$。

类型描述工具归一性$P(X=x)$
离散PMF $f_X(x) = P(X=x)$$\sum f_X(x) = 1$$> 0$
连续PDF $f_X(x) = dF_X/dx$$\int f_X(x)dx = 1$$= 0$
同分布与随机占优

$F_X(x) = F_Y(x)$（对所有 $x$）$\Longleftrightarrow$ $X$ 与 $Y$ 同分布。同分布不意味着 $X = Y$——两者可以定义在不同的样本空间上，只要 CDF 一致即可。

一阶随机占优：若 $F(x) \leq G(x)$（对所有 $x$），且存在严格不等式，则分布 $F$ 一阶随机占优 $G$。这意味着：对于任何递增效用函数，最大化预期效用的决策者会偏好 $F$。

Part 7 · 离散型随机变量的数字特征
期望与方差
期望（数学期望）

离散随机变量 $X$ 的数学期望（expected value）定义为：

$$EX = \sum_{i} x_i \, f_X(x_i)$$

期望刻画的是随机变量在大量重复试验中的"平均"取值——它是 PMF 的加权平均，权重即概率。期望未必是 $X$ 最可能取到的值，甚至可能不在 $X$ 的支撑集内。

方差

方差衡量随机变量围绕期望的离散程度：

$$DX = E\left[(X - EX)^2\right]$$

实际计算中更常用的是方差的计算公式：

$$DX = E(X^2) - (EX)^2$$

直觉：$E(X^2)$ 是 $X^2$ 的期望（即 $X$ 取值平方的加权平均），$(EX)^2$ 是均值的平方。两者之差就是方差——偏差平方的期望减去均值平方，恰好等于偏差的方差。这个公式避免了逐项计算 $(x_i - EX)^2$，是实践中最常用的方差计算捷径。
$E(g(X))$：随机变量函数的期望

若 $Y = g(X)$，不需要先求 $Y$ 的分布，可以直接用 $X$ 的分布计算：

$$E[g(X)] = \sum_{i} g(x_i) \, f_X(x_i)$$

例如计算 $E(X^2)$ 时，$g(x) = x^2$，直接代入 $X$ 的取值平方乘以对应概率即可。

例题：离散型随机变量的期望与方差

已知随机变量 $X$ 的分布律如右表所示：

$X$$-2$$0$$1$$2$
$P$$0.3$$0.3$$0.2$$a$
离散型随机变量的方差计算：从分布律到 $EX$、$E(X^2)$、$DX$ 的完整例题

（1）求 $P\{0 \le X < 2\}$。

$$P\{0 \le X < 2\} = P(X=0) + P(X=1) = 0.3 + 0.2 = 0.5$$

（2）求常数 $a$。

由归一性 $\sum f_X(x) = 1$：

$$0.3 + 0.3 + 0.2 + a = 1 \implies a = 0.2$$

（3）求 $X^2$ 的分布律。

$X^2$ 的取值：$(-2)^2 = 4$，$0^2 = 0$，$1^2 = 1$，$2^2 = 4$。注意 $X=-2$ 和 $X=2$ 对应同一值 $4$，概率需合并：

$X^2$$0$$1$$4$
$P$$0.3$$0.2$$0.3+0.2=0.5$

（4）求 $EX$、$E(X^2)$、$DX$。

$$EX = \sum x_i \, f_X(x_i) = (-2)(0.3) + 0(0.3) + 1(0.2) + 2(0.2) = 0$$

$$E(X^2) = \sum x_i^2 \, f_X(x_i) = 4(0.3) + 0(0.3) + 1(0.2) + 4(0.2) = 2.2$$

$$DX = E(X^2) - (EX)^2 = 2.2 - 0^2 = 2.2$$

要点：此题展示了离散型数字特征计算的标准流程——先由归一性确定未知参数，再逐项加权求和算期望，最后用 $DX = E(X^2) - (EX)^2$ 快速得到方差。$EX=0$ 说明 $X$ 正负对称；$DX=2.2$ 说明 $X$ 的取值围绕均值有较大的离散度。

例：硬币抛掷

抛 10 枚均匀硬币恰好 3 次正面的概率：

$$P(X=3) = \binom{10}{3} \left(\frac{1}{2}\right)^3 \left(\frac{1}{2}\right)^7 = \frac{120}{1024} \approx 0.1172$$

其中 $\binom{10}{3} = 120$ 是从 10 次抛掷中选出恰好 3 次正面（不计顺序）的方法数。
