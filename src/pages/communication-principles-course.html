Part 0 · 学习目标
这篇总览页解决什么问题

这不是把 10 章公式重新抄一遍的“目录页”，而是一张课程地图。它解决三个问题：第一，整门通信原理课到底在讲什么；第二，10 章之间是怎样前后勾连的；第三，复习时应该先抓哪几章，再补哪几章。

如果你在课堂上常常出现“这一章公式会写，但不知道它在系统里干嘛”的感觉，那么先看这篇总览页，再回到各章详细笔记，通常会顺很多。

前置知识回顾

通信原理不是孤立课程，它默认你已经会使用几组基础工具。

  • 概率论：概率、期望、方差、条件概率。作用：理解噪声、误码率与随机过程。
  • 信号与系统：傅里叶变换、线性时不变系统、卷积。作用：理解频谱、滤波与匹配滤波器。
  • 复数与相量：欧拉公式、相位、正交分量。作用：理解 PSK、QAM、同步环路。
  • 线性代数与模 2 运算：矩阵、向量、二元域。作用：理解线性分组码、监督矩阵与生成矩阵。
Part 1 · 背景问题
为什么通信原理会显得“碎”

通信原理最容易让人学碎，是因为它同时讨论了三层问题：

  1. 信息层：消息到底有多少不确定性，怎样度量信息。
  2. 传输层：信道会施加什么限制，波形怎样设计，调制方式怎样选。
  3. 接收层：接收端怎样在噪声中恢复消息，怎样降低错误，怎样保持同步。

如果只按章节顺序背结论，很容易把这些层次混在一起。更好的方法，是把课程看成一条完整链路：

主线：信息要传出去 → 信道不完美 → 所以要设计合适的波形与调制 → 接收端还要在噪声中做判决 → 最后再用编码和同步兜底。

Part 3 · 关键路径
复习时最值得优先打通的三条主线

如果时间不够，不要平均发力。通信原理里最适合按“主线”复习。

主线推荐章节解决的问题
信息与信道主线第1章 → 第2章 → 第3章先知道信息怎么度量，再知道噪声怎么建模，最后知道信道会给你什么物理限制。
发送端设计主线第5章 → 第6章 → 第7章先学基带波形，再学载波调制，最后学模拟消息怎样数字化进入系统。
接收端可靠性主线第8章 → 第9章 → 第11章先解决判决问题，再解决纠错问题，最后解决同步问题。

真正复习时，最推荐的顺序是：

推荐顺序：第1章 → 第2章 → 第3章 → 第5章 → 第6章 → 第8章 → 第7章 → 第9章 → 第10章 → 第11章。

Part 5 · 每章一句话速记
考前一天可以只扫这一节
  • 第1章：通信系统先要建模，信息量和熵决定“传的内容有多复杂”，误码率和频带利用率决定“传得好不好”。
  • 第2章：随机过程是噪声的语言，自相关函数和功率谱密度是一对时域/频域镜像。
  • 第3章：信道会失真、会加噪，香农容量给出了可靠传输速率的上限。
  • 第5章：基带传输最怕码间串扰，奈奎斯特准则给出无码间串扰条件。
  • 第6章：载波调制就是让数字基带信号去控制高频正弦波的幅度、频率或相位。
  • 第7章：模拟信号数字化的标准流水线是抽样、量化、编码。
  • 第8章：最佳接收就是在噪声中把错误概率压到最低，匹配滤波器是最优工具。
  • 第9章：差错控制编码通过增加冗余，让错误更容易被发现和纠正。
  • 第10章：伪随机序列表面像噪声，实则有严格代数结构，特别适合扩频与多址。
  • 第11章：同步不是附属问题，而是通信系统能否稳定工作的前提。
Part 6 · 使用建议
怎样配合详细笔记和课件复习
  1. 第一次过课：先看这篇总览页，明确章节关系，再进对应章的详细笔记。
  2. 第二次复习：抓三条主线，不要被零散章节带跑偏。
  3. 做题前：先翻各章的速查表，再回到课件页看细节定义和公式环境。
  4. 考前冲刺：优先记“公式的适用条件”，而不是只记公式外形。
