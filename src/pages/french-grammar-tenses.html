Part 2
动词系统（上）：时态——时间线上的坐标

法语有超过 ==14 种时态==，但日常使用中只需要熟练掌握 6-8 种。关键不是"会变位"，而是在什么语境下选择什么时态。

2.1 直陈式现在时（Présent）：一切的基础

直陈式现在时不仅是"现在发生的事"。它还可以表达：

  • 普遍真理：La Terre tourne autour du Soleil.（地球绕太阳转。）
  • 习惯动作：Je prends le métro tous les jours.（我每天坐地铁。）
  • 近将来：Le train part dans cinq minutes.（火车五分钟后发车。）
  • 历史现在时：En 1789, le peuple de Paris prend la Bastille.（1789 年，巴黎人民攻占巴士底狱。）

第一组动词（-er）的变位最规律：

第一组动词变位示例：parler（说）

  • je parle — 我说
  • tu parles — 你说
  • il/elle parle — 他/她说
  • nous parlons — 我们说
  • vous parlez — 你们说
  • ils/elles parlent — 他们/她们说
发音提示：parle / parles / parlent 发音完全相同（/paʁl/），只有 parlons（/paʁlɔ̃/）和 parlez（/paʁle/）有区别。口语中靠语境和主语区分。

三大不规则动词是地基中的地基，必须脱口而出：

être（是）avoir（有）aller（去）
jesuisaivais
tuesasvas
il/elleestava
noussommesavonsallons
vousêtesavezallez
ils/ellessontontvont

「Je suis un homme qui ne regarde pas les étoiles. Les étoiles, ça ne m'intéresse pas.」

—— Antoine de Saint-Exupéry, Le Petit Prince（商人的话）

这段话中连续使用了直陈式现在时：suis（状态）、regarde（习惯否定）、intéresse（普遍态度）。圣-埃克苏佩里用现在时让小王子的世界永远处于"当下"。

2.2 复合过去时（Passé composé）：已完成的动作

复合过去时是口语中最常用的过去时态。它的结构是 ==助动词（avoir / être 的现在时）+ 过去分词==。

复合过去时的构成

大部分动词用 avoir 作助动词：J'ai mangé. Tu as compris. Ils ont fini.

少数动词用 être 作助动词——主要是表示位置变化和状态变化的不及物动词（Dr. & Mrs. Vandertramp 助记法），以及所有代词式动词：

  • Je suis allé(e) au cinéma.（我去了电影院。）
  • Elle est partie à 8h.（她八点出发了。）
  • Nous nous sommes rencontrés hier.（我们昨天遇见了。）
être 助动词的记忆口诀

Dr. & Mrs. Vandertramp：Devenir, Revenir, Monter, Rester, Sortir, Venir, Aller, Naître, Descendre, Entrer, Retourner, Tomber, Rentrer, Arriver, Mourir, Partir。每个字母对应一个动词，都表示"来了/去了/变了"。

用 être 的动词有一个关键特征：过去分词要随主语的性数变化。

  • Elle est partie.（她走了。）— 阴性加 -e
  • Ils sont partis.（他们走了。）— 复数加 -s
  • Elles sont parties.（她们走了。）— 阴性复数加 -es

「Aujourd'hui, maman est morte. Ou peut-être hier, je ne sais pas.」

—— Albert Camus, L'Étranger

加缪《局外人》的开篇名句。est morte（死了）用的就是 être + 过去分词——mourir（死亡）是状态变化动词，所以用 être。注意默尔索用 est morte（复合过去时）而不是未完成过去时 mourait，暗示他把母亲的死亡看作一个已完成的事件，而非持续的状态——这本身就是人物性格的暗示。

2.3 未完成过去时（Imparfait）：记忆的底色

如果说复合过去时是照片（定格的瞬间），未完成过去时就是视频（持续的背景）。它表达的是：

  • 过去的习惯：Quand j'étais petit, je jouais au foot tous les dimanches.（我小时候每个星期天都踢足球。）
  • 过去的持续状态：Il faisait beau, les oiseaux chantaient.（天气很好，鸟在唱歌。）
  • 背景描写：Le soleil se couchait, la rue était déserte.（太阳正在落下，街道空无一人。）

未完成过去时的变位极其规律——从现在时 nous 形式去掉 -ons，加上词尾 -ais / -ais / -ait / -ions / -iez / -aient：

变位示例：manger（吃）→ nous mangons → mang-

  • je mangeais — 我（过去）在吃
  • tu mangeais — 你（过去）在吃
  • il mangeait — 他（过去）在吃
  • nous mangions — 我们（过去）在吃
  • vous mangiez — 你们（过去）在吃
  • ils mangeaient — 他们（过去）在吃

复合过去时 vs 未完成过去时 是法语学习的经典难关。用一个场景来感受：

经典对比：两个时态如何协作叙事

Il pleuvait（未完成）quand je suis sorti（复合）.

→ 下雨是背景（持续），出门是事件（瞬间）。

Je lisais（未完成）un livre quand le téléphone a sonné（复合）.

→ 读书是持续状态，电话响了是打断事件。

Quand j'habitais（未完成）à Paris, je suis allé（复合）au Louvre trois fois.

→ 住在巴黎是背景/习惯，去了卢浮宫三次是已完成的事实。

「Les jours s'allongeaient, les enfants jouaient dans la rue, et moi, je lisais sur le banc. Soudain, un cri a retenti.」

—— 模仿叙事风格的自编例句

这短短一句展示了两种过去时的完美配合：前三个未完成过去时（s'allongeaient, jouaient, lisais）铺设宁静的背景画面，然后复合过去时（a retenti）突然插入一个瞬间事件，打破宁静。这就是法语叙事的"背景-事件"节奏。

2.4 愈过去时（Plus-que-parfait）：过去的过去

愈过去时表示在另一个过去动作之前已经完成的动作。结构是 ==助动词的未完成过去时 + 过去分词==。

  • J'avais déjà mangé quand il est arrivé.（他到的时候我已经吃过了。）
  • Elle était partie avant que je n'arrive.（她在我到之前就已经走了。）
  • Quand je suis rentré, ils avaient tout fini.（我回家时，他们已经全弄完了。）

« Il avait vingt ans. Ça faisait six mois qu'il n'avait pas vu son père. »

—— 模仿新闻报道风格

愈过去时在叙事中常用于倒叙——在时间线上往回跳一步，补充前置信息。在 DALF C1 的阅读理解中，识别愈过去时是判断叙事顺序的关键。

2.5 简单将来时（Futur simple）与最近将来时（Futur proche）

法语的将来时有两种常用形式：

时态结构语感差异例句
最近将来时aller + 动词原形即将发生、计划中Je vais manger.（我要去吃了。）
简单将来时动词词根 + 词尾较远的未来、承诺、预测Je mangerai.（我会吃的。）

简单将来时的词尾对所有动词都一样：-ai / -as / -a / -ons / -ez / -ont。不规则之处在于词根。

常见不规则简单将来时词根

  • être → je serai（词根 ser-）
  • avoir → j'aurai（词根 aur-）
  • aller → j'irai（词根 ir-）
  • faire → je ferai（词根 fer-）
  • pouvoir → je pourrai（词根 pourr-）
  • venir → je viendrai（词根 viendr-）
  • voir → je verrai（词根 verr-）

「On ne voit bien qu'avec le cœur. L'essentiel est invisible pour les yeux. ... Tu seras responsable de ce que tu as apprivoisé.」

—— Antoine de Saint-Exupéry, Le Petit Prince

小王子中狐狸的秘密。注意时态的切换：voit / est（现在时，普遍真理）→ seras（简单将来时，对"你"未来的承诺）。从永恒真理到个人承诺，时态本身就在叙事。

2.6 条件式现在时（Conditionnel présent）：假设与委婉

条件式的核心语义是"如果……就会……"，但它更常见的功能是委婉表达：

条件式的三种核心用法

① 假设（si + 未完成过去时 → 条件式）

  • Si j'avais le temps, je voyagerais plus.（如果我有时间，我会多旅行。）
  • S'il faisait beau, nous irions au parc.（如果天气好，我们会去公园。）

② 委婉请求/建议

  • Je voudrais un café, s'il vous plaît.（我想要一杯咖啡。——比 je veux 礼貌得多）
  • Vous pourriez répéter, s'il vous plaît ?（您能再说一遍吗？）
  • Il faudrait partir maintenant.（现在该走了。——比 il faut 柔和）

③ 传闻/不确定（新闻报道）

  • Le président serait en voyage en Chine.（据报道总统正在中国访问。——暗示未经证实）

条件式的变位很简单：==简单将来时的词根 + 未完成过去时的词尾（-ais / -ais / -ait / -ions / -iez / -aient）==。

« Je voudrais tant que tu te souviennes
Des jours heureux où nous étions amis.
En ce temps-là, la vie était plus belle
Et le soleil plus brûlant qu'aujourd'hui.」

—— Jacques Prévert, Les Feuilles mortes

普雷维尔的经典歌词。开头的 voudrais（条件式）是一种温柔的愿望，而非强硬的要求。随后切换到未完成过去时 étions / était 描绘回忆的画面。条件式的"温柔"和未完成过去时的"回忆"在这里完美融合。

2.8 叙事练习：两种过去时的协作

掌握复合过去时和未完成过去时的最佳方式是大量练习叙事。以下是三段练习，每段标注了时态选择的原因：

叙事练习一：一个普通的周末

Il faisait（未完成——天气背景）beau ce jour-là. Je lisais（未完成——持续动作）un roman au jardin quand le téléphone a sonné（复合——突发事件）. C'était（未完成——状态描述）ma sœur. Elle m'a annoncé（复合——完成的事件）qu'elle venait（未完成——计划中的动作）me voir. J'ai range（复合——完成的动作）mon livre et je suis allé（复合——完成的位置变化）ouvrir la porte.

叙事练习二：一场意外

Nous marchions（未完成——持续过程）dans la forêt quand soudain nous avons entendu（复合——瞬间事件）un bruit étrange. Le vent soufflait（未完成——背景状态）fort et les arbres se balançaient（未完成——持续画面）. Un éclair a déchiré（复合——瞬间动作）le ciel, puis la pluie a commencé（复合——事件起点）à tomber. Nous courions（未完成——持续动作）vers la maison quand nous avons vu（复合——瞬间发现）un cerf.

2.9 近过去时与近将来时

venir de / aller + 不定式

近过去时（venir de + 不定式）——刚刚发生

  • Je viens de manger.（我刚吃过。）
  • Il venait de partir quand je suis arrivé.（我刚到时他刚走。—— 愈过去时语境）

近将来时（aller + 不定式）——即将发生

  • Je vais manger.（我要去吃了。）
  • Il va pleuvoir.（天要下雨了。）
2.10 简单过去时（Passé simple）：文学的指纹

简单过去时在现代法语口语中已经消失，但在文学作品、历史叙事和正式写作中仍然广泛使用。你的目标不是会用它说话，而是阅读时能认出来：

动词il 形式ils 形式识别特征
parleril parlails parlèrent词尾 -a / -èrent
finiril finitils finirent词尾 -it / -irent
êtreil futils furent完全不规则
avoiril eutils eurent完全不规则
faireil fitils firent词尾 -it / -irent

« Le petit prince fut déçu. Il eût voulu que cette planète lui donnât un spectacle. Mais les grandes personnes furent toutes pareilles. »

—— 模仿 Saint-Exupéry 风格的自编例句（展示简单过去时与虚拟式未完成过去时）
阅读识别：如果你在小说中看到 -a / -it / -ut / -urent / -èrent / -irent 这样的词尾，且语境是过去叙事，那几乎就是简单过去时。它对应复合过去时的书面/文学版本。
2.11 条件式过去时：错过的可能性

==条件式过去时 = avoir / être 的条件式 + 过去分词==。它表达的是在过去没能实现的假设：

  • Si j'avais su, je serais venu.（如果我当时知道，我就会来了。—— 事实上当时不知道，也没来）
  • Si nous avions eu plus de temps, nous aurions visité le musée.（如果我们有更多时间，我们就会参观博物馆了。）
完整公式：si + 愈过去时 → 条件式过去时。这是法语假设句最复杂的组合，也是 C1 考试的必考项。

Part 2 复习速查

  • 现在时：不只"现在"，还有习惯、真理、近将来、历史现在时
  • 复合过去时：avoir / être + 过去分词；être 动词过去分词随主语变化
  • 未完成过去时：nous 形式去 -ons + 词尾；用于背景、习惯、持续状态
  • "背景-事件"叙事：未完成过去时铺背景，复合过去时插事件
  • 愈过去时：过去的过去，用于倒叙和前置信息
  • 将来时选择：最近将来时（aller + 原形，即将/计划）vs 简单将来时（较远/承诺）
  • 条件式三大功能：假设、委婉请求、传闻
