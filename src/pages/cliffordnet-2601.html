核心贡献

CliffordNet 的核心主张很直接：现代视觉架构（CNN、Transformer、ConvNeXt）依赖「空间混合器 + 通道混合器（FFN）」的启发式堆叠，缺乏数学第一性原理。作者用 Clifford Geometric Product 作为唯一的交互算子，同时完成 token mixing 和 channel mixing，实现 No-FFN 设计。

结果：在 CIFAR-100 上，CliffordNet-Nano（1.4M 参数）匹配 ResNet-18（11.2M），压缩比 8×；CliffordNet-Lite（2.6M）达到 79.05%，创下 tiny 模型新 SOTA。

背景：Geometry for Attention → Geometry as Computation

CliffordNet 处于三条研究脉络的交汇处：

  • 几何代数神经网络：Clifford Neural Layers (ICLR 2023)、Clifford Group Equivariant NNs (NeurIPS 2023 oral)、GCAN (ICML 2023) 证明 geometric product 可用于构建具有明确几何性质的神经层
  • 线性复杂度视觉模型：Mamba/SSM 通过状态空间模型替代 quadratic attention；VMamba 用 cross-scan 序列化 2D 图像，核心问题是其扫描破坏了 2D 流形连续性
  • 数学第一性原理叙事：KAN 以 Kolmogorov-Arnold 定理为据；RiemannFormer 用 Riemannian geometry 改造 attention；CliffordNet 则将「几何本身」作为计算

作者 Zhongping Ji（杭州电子科技大学）的研究路线很清晰：2025 年 RiemannFormer 用几何改造 attention → 2026 年 CliffordNet 用几何积直接替换 attention + FFN。两篇均为单作者 arXiv 预印本。

核心转向：从「Geometry for Attention」（几何用来修正 attention）到「Geometry as Computation」（几何本身就是 token 交互机制）。
核心机制：Clifford Geometric Product

CliffordNet 的理论基础是 Clifford Geometric Product：

\[uv = u \cdot v + u \wedge v\]

其中：

  • 内积 $u \cdot v$：捕获特征相干性（标量相似度），类似通道注意力
  • 外积 $u \wedge v$：生成 bivector（双向量），表示 $u$ 和 $v$ 张成的 oriented plane，捕获结构变化和正交性

标准的 dot-product attention 只保留标量相似度 $q \cdot k$，丢弃了方向性和结构信息。CliffordNet 使用完整的 geometric product，同时保留 scalar coherence 和 bivector structure，每次局部交互的信息密度远高于标量点积。

代数完备性：标准神经网络只用对称标量分量 $u \cdot v$（丢弃反对称 bivector 分量 $u \wedge v$）。CliffordNet 显式建模两者，称其为 algebraic completeness。

连续的 Clifford Interaction Ansatz：

\[F(H, C) = P(H \cdot C \oplus H \wedge C)\]

其中 $P$ 是投影回 $\mathbb{R}^D$ 的线性算子。$H$ 是特征状态，$C(H)$ 是几何上下文。

稀疏滚动：线性复杂度的关键

完整计算几何积需要 $O(D^2)$ 的外积展开（所有 channel 两两配对），维度爆炸。CliffordNet 用 Sparse Rolling 机制规避：

定义 channel 循环平移算子 $T_s$，对 offset $s \in S$（例如 $S = \{1, 2, 4, 8, 16\}$ for Lite）：

\[D_s(H, C) = \mathrm{SiLU}(H \odot T_s(C))\]

\[W_s(H, C) = H \odot T_s(C) - T_s(H) \odot C\]

$D_s$ 是内积分支（通过 Hadamard + SiLU），$W_s$ 是外积分支（反对称差分，生成 bivector coefficients）。两者 Concat 后线性投影回 $\mathbb{R}^C$。

复杂度：从 $O(ND^2)$ 降到 $O(ND|S|)$，对序列长度 $N$ 和通道 $D$ 线性，$|S|$ 是小常数（Nano 用 2 个 shifts，Lite 用 5 个 shifts）。

循环平移的几何意义

循环平移 $T_s$ 采样 dense correlation matrix 的特定对角线。这种 structured circulant sparsity 可以理解为在 channel 维度上建立 ring-like topology，logarithmic path length 内传播全局信息，同时保持线性复杂度。

架构详解：Dual-Stream + GGR

CliffordNet Block（No-gFFN-G variant）的 forward pass：

  1. Input Norm: LayerNorm
  2. Dual-Stream Generation: 从 $X$ 生成两条支路 — Deterministic stream $Z_{\text{det}} = \text{Linear}(X)$ 和 Context stream $Z_{\text{ctx}} = \text{SiLU}(\text{BN}(\text{DWConv}(X)))$
  3. Differential Mode（主要结果使用）: $Z_{\text{ctx}} \leftarrow Z_{\text{ctx}} - Z_{\text{det}}$，把上下文变成离散 Laplacian $C \approx \Delta H$，作为高频滤波
  4. Sparse Rolling Interaction: 对每个 $s \in S$，计算 $W^{(s)}$（wedge）和 $D^{(s)}$（dot），Concat 后 Linear proj → $G_{\text{feat}}$
  5. Gated Geometric Residual (GGR): 先 concat $[X, G_{\text{feat}}]$ 过 sigmoid gate 生成 $\alpha$，再 $H_{\text{mix}} = \text{SiLU}(X) + \alpha \odot G_{\text{feat}}$
  6. Output Update: $X_l = X_{l-1} + \gamma \odot H_{\text{mix}}$，无 FFN

GGR 的物理含义：一阶 Euler 离散化，SiLU 预滤波抑制背景噪声，Gated injection 确保几何更新由显著特征驱动。

gFFN 家族（可选增强）：

  • gFFN-L: $C = \Delta H$，local 高频边缘捕捉
  • gFFN-G: $C = \text{GlobalAvg}(H)$，global 语义相干性
  • gFFN-H: $C = \Delta H + \beta \cdot \text{GlobalAvg}(H)$，多层叠加

关键设计选择：CliffordNet 不使用 Clifford-valued weights，只把 geometric product 作为轻量交互算子。这样既保留了 Clifford algebra 的代数结构，又避免了 GATr 等方法中 multivector 权重导致的参数爆炸（4D spacetime algebra 有 16 个分量）。

与 ViT/Attention 的核心差异：dot-product attention 将 token 间关系压缩为标量相似度，丢弃方向性、有向面积、张成平面等几何结构。CliffordNet 用 geometric product 保留 scalar + bivector 两个分量，每次交互的信息密度更高，因此 No-FFN 仍能维持高性能。

与 CNN 的差异：ShuffleNetV2 的 channel shuffle 是手工工程启发式，CliffordNet 的 rolling interaction 可解释为 structured circulant sparsity（采样 correlation matrix 特定对角线），但提供了更明确的代数解释。

与 SSM/VMamba 的差异：VMamba 的 cross-scan 把 2D 图像切成离散 1D 路径，破坏流形连续性。CliffordNet 保持原生 2D feature grid，用 depth-wise convolution 建立局部上下文。

与已有 Clifford 网络的差异：GATr/Clifford Neural Layers 把 weights 作为 multivectors，导致 4D spacetime algebra 有 16 个分量，参数组合爆炸。CliffordNet 仅将 geometric product 作为轻量交互算子，在 real-valued feature maps 上执行，兼顾效率与代数完备性。

复现注意事项

核心算子和 Algorithm 1 伪代码完整，复现可行。但以下细节原文未明确给出：

  • 完整 loss function（cross-entropy？label smoothing？）
  • Learning rate、weight decay、batch size、DropPath rate、LayerScale $\gamma$ 初始化值
  • 每个 variant 的 depth $L$ 和 channel $D$ 具体值（Nano/Lite 的 32/64/96 命名约定未说明）
  • gFFN-G 的 broadcasting 细节和 projection 是否共享
  • 数据处理：mean/std、CIFAR-100 normalization 参数

最小可复现版本应优先实现 No-gFFN-G variant：PatchEmbed + 重复 CliffordNet Block（LayerNorm/Linear det/DWConv+BN+SiLU/diff mode/shift set rolling/Dot+Wedge/Linear projection/sigmoid gate/SiLU state prefilter/LayerScale residual/DropPath）+ global pooling + classifier，训练用 CIFAR-100/AdamW/cosine/200 epochs/AutoAugment/Random Erasing/patch size 2，超参数从开源代码补齐。

官方代码仓库

ParaMind2025/CAN — stars=149, forks=19, 最近更新 2026-03-31。支持两种运行模式：High-performance mode（自定义 CUDA/Triton kernel，RTX 4090 优化）和 Compatibility mode（纯 PyTorch fallback）。实验覆盖 CIFAR-100、Kvasir-Dataset-v2、ISIC2018，暂未在 ImageNet-1K/COCO/ADE20K 上验证。

局限性与未验证问题
  • CIFAR-100 验证为主：ImageNet 等大规模数据集结果缺失，医疗图像（Kvasir/ISIC）分类也属小规模验证
  • 序列化批评缺乏严格证明：CliffordNet 认为 cross-scan 破坏 2D 流形，但 VMamba 用四方向扫描已部分缓解单一路径偏置，当前 CIFAR-100 结果支持其有效性，还不足以完全证明理论解释的因果性
  • 几何严格性边界：当前只是 Grade 1 × Grade 1 向量乘积（生成 Grade 0 scalar + Grade 2 bivector），未来扩展到 vector-bivector product 未实现
  • 训练细节不透明：完整超参数和模型配置缺失，影响严谨复现
  • arXiv preprint 状态：尚无正式会议接收信息
结论

CliffordNet 提出了一个野心勃勃的主张：用代数完备的 Clifford Geometric Product 同时替代 attention 和 FFN，实现参数效率突破。其核心洞察是「几何积的表示密度足够高，使得 FFN 变得冗余」。

这一工作的价值不在于在 CIFAR-100 上超过 ResNet-18，而在于它提出了一种新的架构设计范式：从数学第一性原理（代数完备性）出发，而非启发式工程模块堆叠。如果 local geometric interaction 足够密集，全局理解可以从中涌现——这暗示了「geometry is all you need」的可能性。

关键开放问题：这种几何驱动的架构能否扩展到 ImageNet 等大规模任务？bivector 结构在更高分辨率、更复杂视觉场景中是否仍然有效？No-FFN 设计在下游任务（检测、分割）中能否维持优势？
