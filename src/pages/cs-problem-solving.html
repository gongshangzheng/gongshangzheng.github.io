概述

问题解决和概念研究是认知心理学的核心领域。问题解决探讨人类如何在复杂环境中寻找从初始状态到达目标状态的路径；概念研究则关注人类如何对世界进行分类、建立心理表征系统。本章将系统介绍这两个领域的核心理论和重要发现。

问题解决迷宫示意
一、问题的分类
1.1 良定义问题与劣定义问题

良定义问题（Well-defined Problems）：

  • 具有明确的初始状态、目标状态和操作算子
  • 解决路径清晰可循
  • 例如：河内塔问题、华容道、数字谜题

劣定义问题（Ill-defined Problems）：

  • 没有明确的解决路径
  • 需要发展合适的表征方式
  • 往往需要突然"看清"问题才能解决
  • 例如：创意写作、设计问题、生活中大部分实际问题
1.2 问题空间理论

Newell和Simon（1972）提出的问题空间理论（Problem Space Theory）是现代问题解决研究的理论基础。

核心观点：问题解决是在问题空间中的搜索过程。问题空间包括：

  • 初始状态（Initial State）：问题开始时的状态
  • 目标状态（Goal State）：期望达到的最终状态
  • 中间状态（Intermediate States）：从初始到目标的各个过渡阶段

搜索过程：从初始状态出发，通过一系列操作（算子）逐步接近目标状态。

问题空间理论示意图

以河内塔问题为例，问题的搜索空间可能非常庞大。三个圆盘的河内塔问题有27种可能的状态，而随着圆盘数量增加，状态数量呈指数增长。

二、问题解决策略
2.1 试错法（Trial and Error）

Thorndike（1898-1949）通过"谜箱实验"研究了动物的问题解决行为。

实验设置：猫被关在迷箱中，要通过执行一系列动作（按踏板、拉绳子、推门闩）来逃脱获得食物。

主要发现：

  • 初期行为表现为冲动性和随机性
  • 经过多次尝试后，解决问题的时间逐渐缩短
  • 表现出"效果律"——成功的动作被强化并保留
Thorndike的迷箱实验装置
2.2 爬山法（Hill Climbing）

爬山法是一种简单的启发式策略：

  1. 找到某种衡量当前状态与目标状态距离的方法
  2. 选择能够最大程度减少这个距离的下一步操作

问题：爬山法可能导致次优解，特别是当存在"局部最大值"时——在到达全局最优解之前就陷入了一个局部最优状态。

爬山法示意图
爬山法的局部最大值陷阱

临床证据：前额叶皮层与规划功能密切相关。额叶损伤患者表现出"坚持不懈"（perseverance）症状——他们会重复同样的动作，即使这明显不能解决问题。

2.3 手段目的分析（Means-End Analysis）

手段目的分析是一种更复杂的问题解决策略：

  1. 设定目标
  2. 寻找差距：比较当前状态与目标状态的差异
  3. 找到算子：找到能够减少差距的操作
  4. 应用算子
  5. 重复直到达到最终目标
手段目的分析流程图

刷房子问题示例：

目标层级分析：最终目标（刷好房子）→ 子目标（应用油漆）→ 子目标（需要油漆和刷子）→ 子目标（去五金店）→ 执行 → 回溯执行所有子目标直到完成

目标栈（Goal Stack）：采用"后进先出"的方式管理目标，最后设定的目标最先被处理。

三、功能固着与顿悟
3.1 功能固着（Functional Fixedness）

功能固着是指人们倾向于将物品视为只有其传统功能的样子，难以发现物品的新用途。

Maier的两绳问题（1931）：

  • 房间天花板上有两根绳子，但距离太远无法同时抓住
  • 房间里有钳子
  • 只有39%的被试能在10分钟内想到解决方案

Duncker的蜡烛问题：

  • 给被试一盒图钉、一盒火柴和一支蜡烛
  • 任务：将蜡烛固定在门上
  • 许多被试无法想到可以用装图钉的盒子作为蜡烛的托盘
Duncker蜡烛问题实验材料

特点：

  • 年幼儿童受功能固着的影响较小
  • 这可能是因为他们的经验较少，没有牢固建立物品功能与用途之间的关联
3.2 顿悟（Insight）

顿悟是指对问题的突然理解，通常涉及用完全不同的方式重新表征问题。

经典例子：

六棍问题：六根棍子如何组成四个等边三角形？

  • 错误表征：在平面上尝试
  • 顿悟：在三维空间中考虑，答案是一个金字塔形
六棍问题的常见错误表征
六棍问题的三维空间解答（金字塔结构）

Kohler的黑猩猩实验（1945）：

  • Sultan被关在笼子里，天花板上挂着香蕉
  • 笼子里有两个木箱
  • Sultan能够将箱子叠放在一起，够到香蕉
  • 观察发现，Sultan在解决问题之前似乎经历了"顿悟"
3.3 顿悟的证据

Metcalfe和Weibe（1987）的实验直接比较了顿悟问题与非顿悟问题。

非顿悟问题（代数）：分解因式 16y² - 40yz + 25z²

顿悟问题：

囚犯逃脱问题：一个囚犯试图从塔楼逃脱。他在牢房里发现一根绳子，长度只有安全着陆所需长度的一半。他把绳子分成两半，系在一起，成功逃脱了。他是怎么做到的？

测量方法：每隔15秒让被试评估自己离解决问题有多近（1=完全没头绪，7=几乎解决了）。

结果：

  • 非顿悟问题：接近解决方案时的"温暖感"能够预测最终是否成功
  • 顿悟问题：没有这种渐进式的温暖感变化——突然就解决了
顿悟问题与非顿悟问题的温暖感评级对比
四、概念与分类
4.1 概念的基本功能

概念是人类认知经济性的体现：

  1. 认知经济：通过将世界分成若干类别，减少需要学习、感知、记忆和识别的信息量
  2. 预测能力：允许我们基于类别知识做出准确预测
  3. 交流功能：使人类之间的交流成为可能
4.2 概念化的层次

概念在不同抽象层次上存在，其中基本层次（Basic Level）具有特殊地位：

  • 超ordinate level（上位层次）：如"家具"、"动物"——非常抽象，信息量少
  • 基本层次（Basic Level）：如"椅子"、"狗"——人类最先习得、最自然使用的层次
  • Subordinate level（下位层次）：如"办公椅"、"金毛猎犬"——高度具体

基本层次的特殊性：

  1. 是在形态上相似的最高抽象层次
  2. 发展上：儿童最早学习的词汇层次
  3. 语言上：自然语言中命名最常用的层次

专家与基本层次：研究表明，专家在识别其专业领域的对象时，能够快速识别到下位层次。例如，鸟类专家识别"旅鸫"的速度与普通人识别"鸟"的速度相当。

五、概念的表征理论
5.1 原型理论（Prototype Theory）

核心假设：概念由一个原型——即类别成员的中央倾向——来表征。

多维空间表征：

  • 每个概念可以表示为多维特征空间中的一个点
  • 新颖实例根据其与原型的相似性进行分类
  • 典型成员更接近原型，分类更快、更准确

典型性效应（Typicality Effects）：

  • 典型项目：知更鸟是"鸟"的好例子，分类快
  • 非典型项目：企鹅是"鸟"的差例子，分类慢
原型理论的多维特征空间表征

Posner & Keele的实验：

  • 让被试学习一组打散分布的点模式
  • 测试时，未见过的原型模式比个别实例更容易被识别和记忆
  • 这证明了原型的抽象表征性质

原型理论的问题：

  • 丢失了关于个别样例的所有信息
  • 无法解释类别大小的差异
  • 无法解释特征之间的相关性
5.2 样例理论（Exemplar Theory）

核心假设：类别表征由存储在记忆中的个别样例组成。分类时，将新实例与已知样例进行比较，最相似的样例对分类影响最大。

解释力：

  • 能够解释原型效应（原型与大多数样例相似）
  • 能够解释典型性渐变结构
  • 能够解释变异性效应

变异性效应例子：

  • 大多数披萨直径约12英寸，但变异范围大（2-30英寸）
  • 大多数尺子长度约12英寸，但变异范围小
  • 当判断一个19英寸宽的物体是披萨还是尺子时，大多数人选择披萨
  • 原型理论无法解释这一现象（19英寸距离两个原型同样远）
  • 样例理论可以解释：19英寸与更多披萨样例重叠，因为记忆中有更多接近19英寸的披萨实例
样例理论模型示意

研究结论：Storms等人（2000）的元分析表明，样例模型比原型模型更好地解释了分类实验数据。

5.3 图式理论（Schema Theory）

图式（Schemata）是大型、复杂的知识单元，编码了某一类别成员典型的属性，同时省略了非典型属性。

研究生办公室图式实验：

  • 呈现办公室图片序列
  • 图片包含：椅子、桌子、骷髅头
  • 不包含：书籍
  • 测试：30%的被试错误地回忆起看到了书籍

5.4 多模态概念表征

感知符号理论（Perceptual Symbol Theory, Barsalou, 1999）：

  • 概念由感知符号表征
  • 感知符号是感知过程背后神经状态的记录
  • 表征是经验的模拟，而非符号系统

实验预测：应该能观察到模态切换效应。

模态切换实验：

  • 同一模态条件：banana → YELLOW（都是视觉属性）
  • 不同模态条件：banana → COOL（视觉→触觉）
模态切换效应实验结果

结果：当相邻项目涉及相同感官模态时，反应更快。

神经证据：

  • 当人们观察可操作物体（如锤子）时，控制抓握动作的大脑回路会激活
  • 观察建筑、动物或面孔时，这个回路不会激活

六、专业知识
6.1 专家的认知特点

Chase & Simon的国际象棋研究（1973）：

  • 大师级棋手只考虑约30步
  • 向前搜索的深度只有6步左右
  • 与新手差异不大
  • 关键差异：大师考虑的30步都是"好棋"
  • 依赖约50,000个棋谱模式（Chunk）
国际象棋大师的知识表征

专家的知识组织：

  • 在各自领域内感知有意义的大模式
  • 将信息打包成更大的组块
  • 知识组织方式反映深层理解
  • 专家技能不跨领域迁移
6.2 十年法则

Ericsson等人（1993, 1996）的研究表明：

  • 刻意练习（Deliberate Practice）：高度动机化的、包含仔细自我监控的练习
  • 国际级大师需要10,000-20,000小时的刻意练习
  • 约十年的专注投入才能达到国际水平

优秀音乐家vs.普通音乐家：

  • 两者练习量差异显著
  • 杰出音乐家的练习量远高于普通音乐家
  • 差异与练习量密切相关
十年法则示意图
6.3 天赋vs.练习

关于天赋（talent）作用的争论：

Ericsson的观点：

  • 天赋的概念无法解释任何东西
  • 强调"天才就是90%的汗水加10%的灵感"
  • 这一观点具有争议性

研究共识：

  • 专业知识主要来自经验、练习和刻意练习
  • 遗传因素可能存在但难以量化
  • 实践量是区分优秀与卓越表现的关键因素
七、总结
问题解决的核心要点
  1. 问题空间：问题解决被理解为在问题空间中的搜索过程
  2. 启发式策略：爬山法、试错法、手段目的分析各有优缺点
  3. 功能固着：经验可能导致思维定式，影响问题解决
  4. 顿悟：某些问题需要重新表征才能突然解决
  5. 专业知识：专家依赖大量模式库和有效的知识组织
概念研究的核心要点
  1. 基本层次：概念化存在优先层次，人类最擅长处理基本层次的概念
  2. 原型vs.样例：原型理论强调中央倾向，样例理论强调个别记忆
  3. 图式：大型知识结构组织和编码世界知识
  4. 多模态表征：概念不仅涉及抽象符号，还涉及感知、运动和功能信息
  5. 专业知识：成为专家需要大量刻意练习（约10,000小时）
