Part 1
动词ます形基础：日语的"时态开关"

前两节课我们学的都是名词句（名词 + です），还没有真正接触过动词。从这节课开始，你第一次接触到动词——但别慌，日语的动词变化规律比英语简单得多。

日语动词的礼貌形（丁寧語）都以「ます」结尾，所以叫 ます形。这个形式对不熟悉的人、长辈、上司使用，是日语中最基础的礼貌等级。只要把动词原形（辞書形）的词尾换成「ます」，就完成了从"原形"到"礼貌现在时"的转变。

ます形 时态完整变化表

动词 ます形 可以构成四种时态——现在肯定、现在否定、过去肯定、过去否定。这个表格是日语 N5 语法的四大天王，必须背熟。

时态 构成 功能 例（食べる） 含义
现在肯定 ます 习惯性动作 / 未来计划 たべます 吃（经常/将要）
现在否定 ません 习惯性不动作 / 未来否定 たべません 不吃
过去肯定 ました 已完成的动作 たべました 吃了
过去否定 ませんでした 未完成 / 未曾发生 たべませんでした 没吃
语法细节：ません 和 ませんでした 中的「ま」来自「ません」（否定的郑重形式），这是「ない」的礼貌形。在初学阶段，你只需要记住：ません = 不，ました = 了，把两者组合就是 ませんでした = 没/不...了。

「ある」vs「いる」：日语的存在动词

日语有两个"有/在"：

  • ある（あります）：无意志物体（植物、物品、地点、事件）
  • いる（います）：有意志物体（人、动物、虫子）

记忆技巧：い有一个"人"，所以いる用于有生命的物体。例：「部屋に猫がいます」（房间里有猫）。「部屋にテーブルがあります」（房间里有桌子）。

教材 L4 完整对话：タムさんは日本で何をしますか

这是教材第 4 课的完整场景——米亚向心心展示日本风景照片，心心说出自己的计划。

角色 日语 罗马音 中文
心心 これ、どこですか。 Kore, doko desu ka. 这是哪里？
米亚 沖縄です。 Okinawa desu. 是冲绳。
心心 へえ。 Hee. 哇。
米亚 これは京都です。 Kore wa Kyoto desu. 这是京都。
米亚 京都はとてもきれいですよ。 Kyoto wa totemo kiree desu yo. 京都很漂亮哦。
心心 そうですね。 Sou desu ne. 是啊。
米亚 タムさんは日本で何をしますか。 Tamu-san wa Nihon de nani o shimasu ka. 心心，你打算在日本做什么？
心心 大学 で日本語を勉強します。 Daigaku de Nihongo o benkyou shimasu. 我在大学学日语。
米亚 いいですね。 Ii desu ne. 真好呢。
心心 はい、楽しみです。 Hai, tanoshimi desu. 嗯，我很期待。
关键语法观察：「大学で日本語を勉強します」——这句话同时出现了 で、を、します 三个语法点，正是我们接下来要学的六大助词中的两个。で 表场所，を 表宾语，します 是する的ます形。这句话是助词系统的完美展示。
Part 2
六大助词详解：日语语法的骨架

助词（助詞，じょし）是日语特有的语法成分，没有对应词性的英语翻译。英语有介词（in, on, at, with）、冠词（a, the），但日语的助词功能远比这些丰富——它们标记句子中词语之间的语法关系，让日语即使省略主语也能表达完整语义。

英语靠词序（主语+动词+宾语）来表达关系，日语靠助词——所以英语可以说"The dog bites the man"和"The man bites the dog"意思完全相反（靠词序），日语的「犬が人を噛む」和「人が犬を噛む」靠 が 和 を 区分（词序可互换，助词决定了谁咬谁）。

图：日语六大助词功能示意图——は标记主题，が标记主语，を标记宾语，に标记方向/时间/地点，で标记手段/场所，の连接名词
は（wa）：主题标记——"关于……"

「は」是日语中出现频率最高的助词，用于标记主题（topic）。它的功能不是"说是什么"，而是告诉听话人"我现在要说的是关于这个东西的事"。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
は wa 标记主题（话题） わたしは学生です。 Watashi wa gakusee desu. 我是学生。 L2
标记主题（关于地点） 春さんハウスはどこですか。 Haru-san Hausu wa doko desu ka. 春奶奶之家在哪里？ L1
标记主题（关于动作） タムさんは何
が（ga）：主语标记——"……在发生动作"

「が」标记主语，表示动作或状态的主体。在"狗咬人"这个场景里，狗是主语，人是宾语，日语就说「犬が人を噛む」。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
が ga 标记动作主语 猫がいます。 Neko ga imasu. 有只猫。 L4
标记描述对象 ここが春さんハウスだよ。 Koko ga Haru-san Hausu da yo. 这里就是春奶奶之家啦。 L1
标记能力/状态主体 日本語ができます。 Nihongo ga dekimasu. 我会日语。（能说日语） 衍生
を（wo/o）：宾语标记——"……是动作承受者"

「を」标记宾语（受词），即动作的承受者。「食べる」的承受对象用 を 连接：「パンを食べます」（吃面包）。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
を o / wo 标记宾语（吃） パンをを食べます。 Pan o tabemasu. 吃面包。 L4
标记宾语（喝） 水ををください。 Mizu o kudasai. 请给我水。 L7
标记宾语（做某事） 日本語を勉強します。 Nihongo o benkyou shimasu. 学日语。 L4

注意：「を」在现代日语口语中读作「o」，但在古典日语和某些正式场合读作「wo」。两种发音都是正确的。

に（ni）：多功能的"到达点"助词

「に」是日语中用法最多的助词之一，核心概念是到达点——时间点、方向、存在地点、目的，都用它。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
に ni 时间点 三時に行きます。 Sanji ni ikimasu. 三点去。 L6
方向（去/来） 学校に行きます。 Gakkou ni ikimasu. 去学校。 L5
存在地点（静态） お寺ににあります。 Otera ni arimasu. 在寺院里。 L6
授受/请求对象 友達にあげます。 Tomodachi ni agemasu. 给朋友。 L8 衍生
目的（～に行く） 買い物に行きます。 Kaimono ni ikimasu. 去买东西。 L5
で（de）：场所和手段

「で」有两个核心功能：动作发生的场所和手段/工具。与 に（静态存在地点）区分：に 用于"东西停在哪里"，で 用于"动作在哪里发生"。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
で de 动作场所 大学で日本語を勉強します。 Daigaku de Nihongo o benkyou shimasu. 在大学学日语。 L4
交通工具（手段） 電車で行きます。 Densha de ikimasu. 坐电车去。 L5
工具（笔/语言） 日本語で話します。 Nihongo de hanashimasu. 用日语说。 衍生
原因/场合 会社で働きます。 Kaisha de hatarakiamsu. 在公司工作。 衍生
の（no）：名词之间的连接剂

「の」是日语中使用频率最高的助词，连接两个名词，表示所属、属性、来源。相当于英语的 's、所有格 of，或汉语的"的"。

助词 读音 功能 例句（日语） 例句（罗马音） 例句（中文） 来源
の no 所属（……的） 大家のはるさん Ooya no Haru-san 房东春奶奶 L2
属性（身份/职业） 写真家のミーヤーさん Shashinka no Miiyaa-san 摄影师米亚 L3
关系（人物介绍） 妹のアンナです Imooto no Anna desu. 是我妹妹安娜。 L8
地点+の（……的哪里） ベトナムのから来ました Betonamu kara kimashita. 来自越南。 L3
は vs が：直觉区分指南

这是日语初学者最常踩的坑，也是最难掌握的区分。以下是直觉性的区分原则：

は（主题标记）的核心逻辑：

  • 说话人主动设定话题："关于 X，我要说……"
  • 话题通常是已知信息，双方都谈论过的东西
  • は 可以重复出现——一旦设为话题，后面继续聊这件事

が（主语标记）的核心逻辑：

  • 客观呈现：某个东西正在做某个动作
  • 强调"是 A，不是别人"——区别于其他可能性
  • 首次引入新信息时常用（"有只猫在房间里"——猫是新的）
  • 回答疑问句中疑问词对应部分时用が

は vs が 实境对比：

问：タムさんはにいますか。（心心在哪里？）

答：タムさんは

（这里「大学に」用 に 而不用 で，是因为に表达静态存在的"在……里"）

问：がありますか。（哪里有什么？）

答：ここに猫

は 和 が 的快速判断技巧

  • 当回答"谁是 X"时（X は Y です）：用 は
  • 当回答"有什么/谁在做什么"时（～が ます）：用 が
  • 当你想强调动作执行者而不是话题时：用 が
  • 当话题已经在上下文中明确，只是继续描述时：用 は

例子：「私は学生です」— 我（主题）是一个学生。「私が学生です」— 我就是那个学生（不是别人）。两者意思相近，但前者是叙述，后者是强调/确认。

Part 3
テ形 + ください：请求句的正确打开方式

如果你学会了 ます形，テ形 就是它的下一个台阶。テ形 是日语中使用频率最高的动词变形之一，几乎出现在所有日常会话中——从点餐「水をください」到问路「ここは教えていただけませんか」。

テ形 有三个核心功能：

  • 连接动作：一个动作接着一个动作发生（"先做 A，然后做 B"）
  • 请求：请对方做某事（ください / くださいませんか）
  • 状态描述：ている 表示动作正在进行（N5 范围外，但需要认识）

本节重点学习请求功能——「动词て形 + ください」。

图：三类动词的テ形变形规则总结。一段动词：去掉「る」加「て」；五段动词：查表格找对应词尾；サ变动词：する→して；カ变动词：来る→来て
三类动词 テ形 变形规则

日语动词分为三类：一段动词（五段动词）、一段动词（一段动词）和サ变/カ变动词。不同的类别有不同的テ形 变形规则。

动词类型 识别特征 变形规则 例（原形→て形） 来源
五段动词 词尾在う段（非る结尾） 查表替换词尾：く→いて・ぐ→いで
す→して・つ/る/う→って
ぬ→んで・む→んで		 書く → 書いて（ke→ite） L7
待つ → まって（ち→って） L7
一段动词 词尾是「る」，る前在い/え段 去掉「る」+ 「て」 食べる → 食べて L7
一段动词 词尾是「る」，る前在い/え段 去掉「る」+ 「て」 見る → 見て L7
サ变动词 する（做） する → して 勉強する → 勉強して L5
カ变动词 来る（来） 来る → 来て（特殊） 来る → 来て L7
五段动词的记忆口诀：く ぐ す つ る う ぬ む —— 取每个字的首字母得到「くぐすつるうぬむ」，谐音"哭姑速哭了，无人"——每次变て形时，想想这个口诀，找到词尾对应的替换：

く→いて（买きて，ike）
ぐ→いで（游泳で，ide）
す→して（si→shi te，しして，site）
つ/る/う→って（てい、てり、てう → 都是て）
ぬ→んで（に→んで，nde）
む→んで（も→んで，mde）
※特例：「行く」（去）不属于以上规则，它变「行って」——去い，然后加て，是唯一一个不规则的て形！
请求句：～てください

「动词て形 + ください」构成请求句，语气礼貌程度中等。ください 本身是「くれる」的敬语，表示"请给我（做）……"。

礼貌程度结构日语罗马音中文使用场景
中等礼貌 て形 + ください 水をください。 Mizu o kudasai. 请给我水。 餐厅点餐、商店购物
较有礼貌 て形 + くださいませんか 名前を書いてくださいませんか。 Nombre o kakite kudasaimasen ka. 能否请您写一下名字？ 对不熟悉的人、稍微正式场合
口语请求 て形 + ください（男性/熟稔） ちょっと待って。→ 待ってください。 Chotto matte. → Matte kudasai. 请等一下。 熟悉的朋友之间
教材 L7 完整对话：水をください

这是教材第 7 课的完整场景——心心在便利店第一次尝试用テ形请求句点水。

角色 日语 罗马音 中文
海斗 コンビニはここだよ。 Konbini wa koko da yo. 便利店就是这里啦。
心心 すごい！ Sugoi! 好厉害！
海斗 ほら、自动販売機。 Hora, jidou hanbaiki. 看，自动贩卖机。
心心 あ、水がある！ A, mizu ga aru! 啊，有水！
心心 ええと……水をください。 Eeto... Mizu o kudasai. 呃……请给我水。
海斗 え？自动販売機はボタンを押すんだよ。 E? Jidou hanbaiki wa botan o osu n da yo. 诶？自动贩卖机是按按钮的。
心心 あ、ボタンを押しますね！ A, botan o osu ne! 啊，要按按钮是吧！
心心 水をください……あ、ボタンを押しますね！ Mizu o kudasai... A, botan o osu ne! 请给我水……啊，要按按钮！
海斗 自动販売機は日本語を話さないからね。 Jidou hanbaiki wa Nihongo o hanasanai kara ne. 因为自动贩卖机不会说日语呀。
语言观察：心心一开始对着自动贩卖机说「水をください」，这在日语口语中是很自然的——对着机器说话时用敬语，体现了一种"即使对方是机器，也要礼貌对待"的文化心理。但海斗提醒她自动贩卖机听不懂日语，要按按钮。这里的「话的さない」是「話す」的否定形（ません→ない），意思是"不说"——这已经在为 N4 的语法做铺垫。
常用请求表达列表

除了「～てください」，还有几种常见的テ形 请求表达：

日语罗马音中文语气使用场景
～てください ~te kudasai 请…… 中等礼貌 点餐、购物、日常请求
～てくださいませんか ~te kudasaimasen ka 能否请您……？ 较有礼貌 对长辈、不熟悉的人
～ていただけませんか ~te itadakemasen ka 能否麻烦您……？ 非常礼貌 正式场合、工作请求
～てくださいね ~te kudasai ne 请……哦（叮嘱） 亲近叮嘱 朋友间、叮嘱对方做某事
～てくれる？（男性）/～て？（随便） ~te kureru? / ~te? 你能帮我……吗？ 口语 朋友间随便请求

重要特例：「行く」的て形

「行く」（去）的て形 不是「行いて」，而是「行って」——这不是因为 い→って 的规则，而是因为「い」变成「っ」再加「て」，有一个促音（っ）插入中间。这是唯一一个不规则的五段动词て形！

记忆方式：想象你"去"到一个地方，脚步「顿一下」（促音）再出去。

例：明日京都に行ってください。（明天请去京都。）

Part 4
介绍他人「A の B です」与～さん规则

从教材第 8 课开始，你将学习如何介绍他人——这是建立人际关系的基础语法。核心句型「A の B です」简洁明了，用 の 连接两个名词，表示"A 的 B"。

「A の B です」句型详解

这个句型用于介绍"某人是谁/做什么的"，在教材 L8 中被用来介绍朋友的职业和身份。

句型结构：

A の B です

  • A：关系/身份/职业名词
  • の：助词，连接 A 和 B
  • B：人名
  • です：判断词

字面意思："A 的 B" → 引申为"B 是 A 的人"（朋友/同事/家人）

教材 L8 完整对话：ともだち소개

这是教材第 8 课的完整场景——绫香带着她的朋友安娜来见大家，心心第一次用「A の B です」做介绍。

角色 日语 罗马音 中文
绫香 ともだちを連れてきたよ。 Tomodachi o tsurete kita yo. 我带朋友来了哦。
海斗 へえ、だれ？ Hee, dare? 哇，谁？
绫香 妹のアンナだよ。 Imooto no Anna da yo. 是我妹妹安娜啦。
心心 わあ、きれいな人！ Waa, kireena hito! 哇，好漂亮的人！
安娜 はじめまして。アンナです。 Hajimemashite. Anna desu. 初次见面。我叫安娜。
安娜 よろしくお願いします。 Yoroshiku onegai shimasu. 请多关照。
海斗 綾香の妹のアンナさんだね。 Ayaka no imooto no Anna-san da ne. 是绫香的妹妹安娜吧。
米亚 アンナさん、こちらこそ。 Anna-san, kochira koso. 安娜，才要请您多关照。
绫香 アンナのともだちのヤンを紹介するよ。 Anna no tomodachi no Yan o shakey suru yo. 我来介绍一下安娜的朋友杨。
杨 ヤンのともだちの山田です。 Yan no tomodachi no Yamada desu. 我是杨的朋友山田。
海斗 え、そうなの？ E, sou na no? 诶，是这样吗？
语法观察：这段对话有一个有趣的"套娃"结构——「アンナのともだちのヤンを紹介する」——"介绍（杨）[杨 是]（安娜 的 朋友 的 人]"。三个 の 叠加，构成了"朋友的（朋友）"关系。日语里 の 可以无限叠加来表达复杂的所有关系。
～さん（san）：日语最重要的敬称

「さん」是日语中使用最广泛的敬称，接在姓名、职业、身份后，表示尊敬——相当于汉语的"先生/女士/同学"，但适用范围更广（几乎对任何人都可以使用）。

日语接续方式罗马音中文对应使用场景
姓名 + さん タムさん Tamu-san 心心女士/先生 最常用，适用于几乎所有人
姓 + さん 山田さん Yamada-san 山田先生/女士 职场、正式场合
职业 + さん お店员さん otenin-san 店员先生/女士 服务业，表示对服务的尊敬
关系 + さん お父さん otou-san 爸爸 称呼家人（礼貌形）
昵称/孩子 + さん 花ちゃんさん → 花ちゃん Hanachan-san → Hanachan 小孩子/亲密朋友间可省略 非正式

～さん 的重要规则

  • 不对自己用さん——你是说话者，不能用敬称称呼自己
  • 不对比自己年纪小很多的人用さん——但用了也不会出错，只是显得客气
  • 服务业用职业+さん——店员（てんいんさん）、司机（うんてんしさん）、医生（いしゃさん）
  • 家人关系用礼貌形——爸爸是 お父さん（而不是 父さん），妈妈是 お母さん
  • 不能用姓 + さん 给上级/老师——应该用「先生」（sensee，限于老师）「社長」「部長」等职位敬称
～さん 之外的常见敬称
敬称读音含义使用场景
～くん kun 君（用于男性晚辈/同学） 男性之间、上对下、男性称呼女性朋友
～ちゃん chan 昵称/小孩子 女性、孩子、亲密关系（花ちゃん等）
～せんせい sensee 老师 教师、教练、有教授关系的人
～さま sama 大人（最敬） 服务业（客人、顾客）、极端正式场合
～じん jin ～人 国籍（ベトナムじん = 越南人）
练习
动手练习：写出你的句子

学习助词最好的方法不是背诵，而是自己造句。下面每个练习都要求你用指定的助词造出一个完整的句子。

练习 1：六大助词填空

用は・が・を・に・で・の 填空，每个助词只能用一次：

  1. わたし___日本___来ました。（我从日本来。）
  2. 大学___日本語___べんきょうします。（我在大学学日语。）
  3. ここ___图书馆___です。（这里就是图书馆。）
  4. パンを___食べます。（吃面包。）
  5. 绫香___妹___アンナさん___소개しました。（绫香介绍了她的妹妹安娜。）
答案
  1. わたしは日本から来ました。
  2. 大学で日本語をべんきょうします。
  3. ここが图书馆です。
    ※这里也可以说「ここは图书馆です」，は和が都可以，但语义略有不同
  4. パンを食べます。
  5. 绫香の妹のアンナさんを紹介しました。

练习 2：动词 ます形 时态变化

将以下动词变成四种时态（ます/ません/ました/ませんでした）：

  1. 飲む（のみる）→ ___
  2. 見る（みる）→ ___
  3. 行く（いく）→ ___
  4. する → ___
答案
  1. 飲みます / 飲みません / 飲みました / 飲みませんでした
  2. 見ます / 見ません / 見ました / 見ませんでした
  3. 行きます / 行きません / 行きました / 行きませんでした
  4. します / しません / しました / しませんでした

练习 3：て形 变形

写出以下动词的て形：

  1. 食べる → ___
  2. 書く → ___
  3. 待つ → ___
  4. 行く → ___
  5. 勉強する → ___
答案
  1. 食べて（一段动词：去掉る+て）
  2. 書いて（五段动词：く→いて）
  3. 待って（五段动词：つ→って）
  4. 行って（特例！行く→行って）
  5. 勉強して（サ变动词：する→して）

练习 4：用 ～てください 写请求句

用所给动词的て形，完成请求句：

  1. みず（水をください）——（请给我水）
  2. かす（書く）——（请写这个）
  3. まちます（待つ）——（请等一下 → ___-てください）
答案
  1. 水をください。（喝：请给我水）
  2. これを書いてください。
  3. ちょっと待ってください。（请稍等）
